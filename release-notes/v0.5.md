# RcloneView 0.5 – Release Notes

We're excited to announce the release of RcloneView 0.5, featuring powerful enhancements to Compare and Explorer, expanded Job Schedule capabilities, improved Remote management, refined UI/UX, and key bug fixes. This version is all about making your workflow smoother, smarter, and more stable.

## Key Improvements

### Compare Window Feature Enhancements

- Allows Remote selection directly within the Compare window.
- Saves user-selected display settings in the Compare window.
- Supports mouse drag for file selection.
- Improved UX for remote selection.
- Automatically terminates any running task when the Compare window is closed.

### Explorer UX Enhancement

- Double-clicking a folder in the browser window now automatically highlights it in the left-hand Tree view.

### Job Schedule Feature Additions

- Added new columns for Scheduled Jobs in the Job Manager window.
- Scheduled Job information now also appears in the History window.
- Added icons and tooltips for Scheduled Jobs in Job Manager.
- Added feature to export job settings (based on user request).

### Remote-related Enhancements

- Added ability to register alias remote by selecting a folder.
- Upgraded Rclone to the latest version to support iCloud config.
- Resolved issue preventing iCloud Drive addition due to 2FA limitations.
- Fixed issue where Alibaba OSS buckets did not appear during GUI remote creation.
- Moved `client_id` / `client_secret` fields to Advanced Options.

## UI/UX Improvements

- Improved layout and structure of the main menu bar.
- Changed menu bar color of Sync and Compare for better visibility.
- Improved window resizing behavior for the Windows version.
- Reviewed and enhanced dark mode across the application.
- Improved user guidance message when creating a directory at the root level.

## Bug Fixes

- Fixed issue where identical source and destination caused incorrect info in the Completed logs.

## Other Feature Enhancements

- Added target file names to the Details view in dry-run results.
