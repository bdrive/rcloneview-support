# RcloneView 0.4 – Release Notes

We're excited to announce the release of RcloneView v0.4, which includes Sync Scheduling, an improved Folder Comparison feature, enhanced Linux support, and various UI and performance optimizations. This update also resolves multiple bugs to enhance stability and usability.

## New Features & Improvements

- **Sync Scheduling:** You can now schedule jobs to run periodically. Note that RcloneView must be running for scheduling to function. The application also supports background execution with system tray integration.
- **Folder Comparison:** We have significantly improved folder comparison for better speed and stability. You can now visually compare folders and recursively analyze differences within subfolders, making file management more intuitive and efficient.
- **Remote Management Enhancements:** The `client_id` and `client_secret` fields have been moved to the advanced options.
- **Enhanced Linux Support:** Fixed compatibility issues with recent Linux distributions.

## Bug Fixes

- Fixed an issue with the log file text field in Global Settings.
- Enhanced error handling throughout the application.
