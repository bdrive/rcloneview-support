# RcloneView 1.3 – Release Notes

**Release Date:** February 2026<br/>
**Platforms:** Windows, macOS, Linux

RcloneView 1.3 adds new job management features and expands integration with popular communication tools like Slack and Discord. It also delivers refinements to email notifications, UI behavior, and job execution tracking.

## New Features

- **Batch Job Support (Beta):**
  You can now group multiple jobs into a batch and run them together.
  This beta feature allows you to define a sequence of jobs to execute in order, making it easier to manage related tasks.
  Additional controls and error handling are planned in future updates.

- **Move Job Type Added:**
  New "Move" job type is now available, allowing you to transfer files by moving them from source to destination.

- **Retry Failed Jobs:**
  You can now retry failed jobs without needing to recreate them manually.

- **Slack Integration:**
  RcloneView now supports Slack notifications and job control from Slack channels.

- **Discord Integration:**
  You can now receive job notifications through Discord.

- **Remote Control Setup Guide Placement:**
  Links for Telegram and Slack remote-control setup guides are now easier to find at the top of the integration section.

- **Indonesian Language Support:**
  The interface is now available in Bahasa Indonesia.

## Improvements

- **Email Notification Enhancements:**
  - Improved layout and clarity of the email settings panel
  - Detailed information is now included in job status emails
  - Fixed rare UI crashes in the email configuration screen
  - Email notifications now remain accessible after a free trial ends

- **Remote Icons Updated:** Added new icons and refreshed old ones for easier remote identification.

- **File List Loading Stability:** Prevented cases where navigating away during slow file loads could result in outdated content being shown.

- **"No Changes" Result Handling:** Ensures the "No Changes" status is retained in detail views regardless of job completion state.

## Bug Fixes

- **OneDrive File Deletion:** Resolved a problem that prevented file deletions on OneDrive remotes.
- **Transfer Status Display Bug:** Fixed an issue where files appeared to be still transferring even after job completion.
- **Compare with Hash on Local Disks:** Fixed a hang condition when comparing files using hash checks on local storage.
- **Job History Inaccuracy:** Force-terminated jobs are now correctly removed from the "running" state in History.
- **Transfer Cancel Tooltip Misalignment:** Corrected tooltip positioning in the Transfer view.
- **Re-registering Same-Named Remotes:** Prevented reuse of outdated file lists when remotes are added with the same name as before.
- **Dry Run Column Truncation:** The final column in the Dry Run details panel now auto-expands to fit its contents.
- **External rclone + Google Drive Token Error:** Fixed a connection issue when using external rclone with headless-mode Google Drive tokens.
