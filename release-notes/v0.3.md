# RcloneView 0.3 – Release Notes

We're excited to release RcloneView v0.3, featuring Folder Comparison, enhanced Linux support, and several UI and performance enhancements. This update also includes various bug fixes to improve stability and usability.

## New Features & Improvements

- **Folder Comparison:** We're proud to introduce folder comparison support! Now, you can visually compare folders and recursively analyze differences within subfolders, making file management more intuitive and efficient.
- **Enhanced Linux Support:** Fixed issues with recent Linux distributions.
- **UI & UX Improvements:** Updated main bar UX design, adjusted icon colors and file name text for visibility, improved remote list icons for free license users, refined menu bar layout and tooltips, fixed window closing shortcuts, and adjusted macOS window resizing visibility.
- **Sync & Remote Management Enhancements:** Added step completion indicators and quick navigation in Sync dialog, fixed key file pass auto-generation in SFTP remote registration, improved folder sync comparison, and resolved missing provider icons in remote registration.
- **Miscellaneous Enhancements:** Fixed Lifetime License and About display issues, improved RcloneView advertisement update functionality, displayed Rclone-related settings in the locations panel, ensured taskbar icon behavior properly displays windows, fixed tray execution notifications, ensured windows close to the tray and support tray-based exit.

## Bug Fixes

- Resolved freezing issues in job and mount dialogs when adding jobs without a remote.
- Improved error handling throughout the application.
- Fixed Windows advertisement loading issue.
