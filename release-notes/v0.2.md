# RcloneView 0.2 – Release Notes

We're excited to release RcloneView v0.2, featuring multi-window support, enhanced synchronization, Linux package updates, and a refined UI for better performance and usability. This update also includes critical bug fixes to ensure a smoother experience.

## Download

- **Windows:** [x86_64](https://downloads.bdrive.com/rclone_view/builds/setup_rclone_view-0.2.158.exe)
- **macOS:** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-0.2.158.dmg)
- **Linux:** [AppImage x86_64](https://downloads.bdrive.com/rclone_view/linux/RcloneView-x86_64.AppImage) · [AppImage arm64](https://downloads.bdrive.com/rclone_view/linux/RcloneView-aarch64.AppImage)

## New Features & Improvements

- **Linux Distribution Support:** Introduced Linux AppImage package support, along with better window management and multi-window operations on Linux. Please note that FUSE installation is required to use the AppImage on Linux. RPM and DEB packages will follow soon.
- **Multi-Window Support:** Introduced multiple-window management across platforms, including Linux, macOS, and Windows.
- **1:N Synchronization:** Added multi-job (1:N) sync functionality and refined its UX.
- **Provider Icons:** Added provider icons in the new remote dialog and remote lists for better visual clarity.
- **Embedded Rclone Updates:** Implemented internal/embedded Rclone update capabilities on Windows and macOS, with proper shutdown on app exit.
- **Improved UI & Accessibility:** Including improved tooltips and movable settings and configuration popup dialogs.
- **Enhanced job and mount dialogs:** Preventing freezing when adding jobs without a remote.
- **Improved transmission speed and remaining time calculation displays.**
- **Upgraded to Rclone 1.68.1:** Ensured proper handling of new remotes.

## Bug Fixes

- Corrected discrepancies in job history time calculations.
- Resolved issues with text truncation in the menu bar.
- Fixed font size problems in message dialogs.
- Eliminated freezes and improved error handling throughout the app.
