# RcloneView 0.6 – Release Notes

**Release Date:** April 2025<br/>
**Platforms:** Windows, macOS, Linux

We are excited to announce the release of RcloneView 0.6, bringing significant improvements to user experience, tray functionality, filtering capabilities, and overall stability. This version reflects numerous enhancements, bug fixes, and new features based on user feedback and ongoing development efforts.

## New Features

- **Auto-Mount on Startup:** RcloneView now supports automatic remote mounting at application launch. Mount settings are persistently saved and managed in the internal database.
- **Quick Mount from Tray:** You can now mount your remote directly from the tray menu, streamlining the process for frequently accessed remotes.
- **Open Remote from Tray Menu:** Added functionality to quickly open a remote directory from the tray, improving accessibility and convenience.
- **Regex Support for Filtering:** Advanced users can now utilize regular expressions in the filter view, fully leveraging rclone's native filter capabilities.
- **Compare View Enhancements - Delete Support:** Files can now be deleted directly from the Compare view, expanding management capabilities within the app.

## UX & Functional Improvements

- **Tray Icon Adaptation for macOS:** The tray icon dynamically adjusts to light and dark macOS backgrounds for better visibility.
- **Explorer Enhancements:** Users can now manually enter a file path in the explorer to navigate directly to the location.
- **Localized Time Display:** File timestamps are now displayed according to the local system time zone.
- **History View UI Fixes:** Improved visibility of the Start Time field and corrected issues with local icon rendering.

## Miscellaneous

- **Removed Advertisements:** The application is now completely ad-free, offering a cleaner and more professional experience.
