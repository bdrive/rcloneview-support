# RcloneView 0.8 – Release Notes

**Release Date:** July 2025<br/>
**Platforms:** Windows, macOS, Linux

RcloneView 0.8 is a major update focusing on performance stability, sync job usability, error handling, and extended cloud provider support. This release also includes key improvements to user interface behavior and compatibility.

## New Features & Enhancements

- **Unified Transfer & Completed Panels:** Merged into a single view for streamlined job tracking.
- **Mount Dialog Enhancements:** Added a `Reload` button and adjusted height for usability.
- **Remote Tab Dragging:** Drag remote tabs between panels for flexible UI management.
- **License Handling Improvements:** Better error reporting and license transfer options when registration fails or device access is lost.
- **Dropbox Business Support:** Fixed alias visibility issues and improved upload error handling.
- **Sync-to-Job Flow:** Automatically opens Job Manager after saving a sync task.
- **Scheduling History Logging:** Error logs for failed scheduled jobs now appear in History.
- **Disable Sync Completion Popups:** New option to turn off sync completion notifications.

## UI & UX Improvements

- **Explorer Panel Double-Click Fix:** Fixed remote not opening on job double-click.
- **Left-Aligned Job Icons & Names:** Improved layout consistency.
- **About Dialog Dark Mode Fix:** Resolved visibility issues in dark mode.
- **Tap Behavior Update:** Prevents entire text selection when tapping fields.
- **Pre-Filled Rename Field:** Displays current filename during rename.
- **Main Menu Styling:** Enhanced contrast for icons and text.
- **Custom Uninstall Icon:** Added a branded uninstall icon.
- **Log File Cleanup on Startup:** The application now deletes the previous log file when starting, ensuring cleaner logs for each session.

## Bug Fixes

- **DB Upgrade Blocker:** Fixed a startup crash related to database upgrade failure.
- **Compare View Fixes:** Corrected file count display and source/destination selection bugs.
- **Job Import/Export Issues:** Improved UX and prevented duplicate job names on import.
- **Mount Drive Conflict:** Prevented drive letter duplication on mount.
- **Rclone Execution Path:** Now always runs from the system root directory.
- **Trailing Slash on Folder Delete:** Ensures accurate deletion of directories.
- **Network Loss Handling:** Popup message now shown when network disconnects occur.
- **Windows Remote & Folder Refresh:** Fixed remotes not appearing after CLI add and folders not refreshing after file deletion.
- **Multi-File Delete Logging:** All deleted files now logged in the completed list.
- **Dry Run Freeze Fix:** Resolved app unresponsiveness when viewing many file results.
- **KDE Plasma Tray Icon Support:** Fixed tray icon display issues when running on KDE Plasma desktop environment.

## Localization & Compatibility

- **Japanese Localization:** Added missing Japanese localization files.
- **German Translation Fix:** Resolved compilation error due to incorrect translation strings.
