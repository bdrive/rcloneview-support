# RcloneView 1.2 – Release Notes

**Release Date:** December 2025<br/>
**Platforms:** Windows, macOS, Linux

RcloneView 1.2 introduces focused improvements around job execution, notifications, cloud compatibility, and UI behavior. This release emphasizes better visibility and control of long-running jobs, along with fixes for common workflow issues.

## New Features

- **Email Notifications for Large File Transfers:**
  Email notifications can now be enabled specifically for large file transfers. This helps track long-running or high-impact jobs more effectively. Email relay configuration is available for PLUS users.

- **Telegram Notifications & Job Control:**
  Telegram integration has been expanded to support both job notifications *and* remote job control. You can receive job status updates and start, stop, or manage running jobs directly from Telegram without accessing the app UI.

- **New S3 Provider Support:**
  Added support for an additional S3-compatible storage provider.

- **Custom Flags for Sync/Copy Jobs:**
  Users can now specify custom rclone flags when creating sync or copy jobs.

- **Terminal Selection Copy:**
  Selected text in the built-in Terminal tab can now be copied to the clipboard.

## Improvements

- **Email Settings UX Improvements:** Improved layout and terminology in email notification settings for clarity.
- **Job Name Display in Transfer View:** The Transfer tab now shows the job name for active transfers.
- **Duplicate Job & Scheduler Execution Prevention:** Prevents the same job from running multiple times, including scheduled runs.
- **Remote Folder Name Visibility:** Fixed truncation issues in Add Job and Job Manager views.
- **Dry Run "See Details" Column Auto-Resize:** Automatically expands the last column to avoid truncated text.
- **New Remote Dialog Alignment:** Improved icon and text alignment for a cleaner layout.
- **Upcoming Features Panel Refinement:** Topics with accepted answers are now hidden.
- **macOS /Volumes Folder Detection:** Automatically detects additions and removals under `/Volumes`.

## Bug Fixes

- **OneDrive File Deletion:** Fixed an issue where files could not be deleted from OneDrive remotes.
- **Google Drive Token Error (External Rclone):** Resolved connection errors when using headless Google Drive tokens.
- **Transfer Status Mismatch:** Fixed cases where transfers completed but were shown as "in progress".
- **Job History State Fix:** Jobs force-terminated during execution no longer remain marked as running.
- **Local Disk Hash Compare Stall:** Fixed an issue where hash-based compare could stall on local disks.
- **"No Changes" Status Consistency:** "No Changes" is now preserved regardless of overall job completion.
- **Transfer Cancel Tooltip Position:** Corrected tooltip placement for the cancel button.
- **Duplicate Remote Registration:** Prevented reuse of stale file lists when re-adding a remote with the same name.
- **rclone Config Compatibility Updates:** Updated handling of configuration values to align with rclone version changes.
