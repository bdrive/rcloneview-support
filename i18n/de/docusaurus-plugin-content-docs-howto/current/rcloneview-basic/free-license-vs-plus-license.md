# RcloneView Lizenztypen

RcloneView bietet zwei Lizenztypen an, um unterschiedliche Nutzungsebenen zu unterstützen: **Free** und **Plus**.  
Die folgende Tabelle vergleicht die verfügbaren Funktionen und Einschränkungen der einzelnen Pläne:

## 🔍 Funktionsvergleich

| Funktion                           | Free-Lizenz | Plus-Lizenz |
| ---------------------------------- | ------------ | ------------ |
| Remotes einbinden/aushängen        | ✅            | ✅            |
| Datei-Explorer (Durchsuchen/Kopieren/Löschen) | ✅            | ✅            |
| Ordnervergleich & Dateiverwaltung  | ✅            | ✅            |
| Ordnervergleich mit Filter         | ❌            | ✅            |
| Datei-Download/-Upload             | ✅            | ✅            |
| Synchronisations- & Auftragsverwaltung | ✅            | ✅            |
| 1:N-Synchronisation                | ✅            | ✅            |
| Zeitplanbasierter Synchronisationsauftrag | ❌            | ✅            |
| Automatischer Start des Zeitplans beim Hochfahren | ❌            | ✅            |
| Cloud-Speicher einbinden           | ✅            | ✅            |
| Automatisches Einbinden beim Hochfahren | ❌            | ✅            |
| Auftragsverlauf & Protokolle       | ✅            | ✅            |
| Systemtray-Integration             | ✅            | ✅            |
| Mehrere RcloneView-Fenster         | ❌            | ✅            |
| Lizenzschlüssel erforderlich       | ❌            | ✅            |

## So aktivieren Sie die Plus-Lizenz

1. Besuchen Sie die [Preisseite](https://rcloneview.com/src/pricing.html) und erwerben Sie eine Lizenz.
2. Gehen Sie in RcloneView zu **Help > Activate License**.
3. Geben Sie Ihre **E-Mail-Adresse** und den bereitgestellten **Lizenzschlüssel** ein.

Nach der Aktivierung werden die Premium-Funktionen automatisch freigeschaltet.
