# So verwenden Sie einen Rabattgutschein

:::info Richtlinie für Rabattgutscheine
Ein Rabattgutschein wird pro Benutzer (E-Mail-Adresse) ausgestellt und kann unabhängig von der Kaufmenge angewendet werden. Er kann nur einmal verwendet werden.
:::

Führen Sie die folgenden Schritte aus, um beim Kauf einer Plus- oder Lifetime-Lizenz einen Rabattgutschein anzuwenden.

1. Speichern oder kopieren Sie den Rabattgutscheincode aus der E-Mail, die Sie erhalten haben.
2. Gehen Sie zu rcloneview.com und öffnen Sie [die Plus-Kaufseite](https://rcloneview.com/src/pricing), klicken Sie dann für die Plus- oder Lifetime-Lizenz auf **Buy Now**.

<img src="/support/images/en/howto/FAQ/click-buy-now-buttons.png" alt="click buy now buttons" class="img-medium img-center" />

3. Geben Sie die E-Mail-Adresse ein, an die Sie Ihren Lizenzcode erhalten möchten, und klicken Sie dann auf **Continue**.
<img src="/support/images/en/howto/FAQ/enter-email-address.png" alt="enter email address" class="img-medium img-center" />

4. Klicken Sie auf `**Enter Coupon Code**`.

<img src="/support/images/en/howto/FAQ/click-enter-coupon-code.png" alt="click enter coupon code" class="img-medium img-center" />

5. Fügen Sie Ihren gespeicherten Rabattgutscheincode ein, klicken Sie auf **Apply**, bestätigen Sie, dass der reduzierte Gesamtbetrag angezeigt wird, und schließen Sie die Zahlung ab.

<img src="/support/images/en/howto/FAQ/enter-your-coupon-code.png" alt="enter your coupon code" class="img-medium img-center" />
