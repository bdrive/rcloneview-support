# Proton Drive

## So fügen Sie Proton Drive mit RcloneView hinzu (Windows)

### Schritt 1: Öffnen Sie den Remote Manager

- Klicken Sie oben rechts im **Remote Manager** auf **`+ New Remote`**.
- Alternativ können Sie im Explorer-Bereich auf die Schaltfläche **`+`** klicken und **`New Remote`** auswählen, um die Remote-Konfiguration zu starten.

<div class="img-grid-2">
<img src="/support/images/en/howto/remote-storage-connection-settings/add-new-remote.png" alt="add new remote" class="img-medium img-center" />
<img src="/support/images/en/howto/remote-storage-connection-settings/add-new-remote-explorer.png" alt="add new remote explorer" class="img-medium img-center" />
</div>
---

### Schritt 2: Wählen Sie Proton Drive als Speicheranbieter

1. Geben Sie in der Leiste **Search Provider** `proton` ein.
2. Wählen Sie **Proton Drive** aus der Liste aus.

Sie gelangen anschließend zum Konfigurationsbildschirm für Proton Drive.

<img src="/support/images/en/howto/remote-storage-connection-settings/select-proton-drive-remote.png" alt="select proton drive remote" class="img-medium img-center" />

### Schritt 3: Konfigurieren Sie Ihren Proton Drive-Remote

Füllen Sie die erforderlichen Felder im Konfigurationsformular aus:

- **Remote name**: Ein aussagekräftiger Name für Ihren Remote (z. B. `MyProtonDrive`)
- **username**: Ihre Proton-E-Mail-Adresse
- **password**: Ihr Proton-Kontopasswort
- **2fa** (optional): Ihr aktueller 2FA-Code (nur wenn 2FA aktiviert ist)

Nachdem Sie alle erforderlichen Informationen eingegeben haben, klicken Sie auf **`Add Remote`**, um die Einrichtung abzuschließen.

<img src="/support/images/en/howto/remote-storage-connection-settings/remote-configure-proton.png" alt="remote configure proton" class="img-medium img-center" />

### Schritt 4: Bestätigen Sie den hinzugefügten Remote

Nach dem Hinzufügen erscheint Ihr Proton Drive-Remote (z. B. `MyProtonDrive`) in der Liste des **Remote Manager**.

Sie können nun:
- Auf **`Open`** klicken, um die Inhalte Ihres Proton Drive zu durchsuchen.
- Ihn für Übertragungen, Mounts oder geplante Aufgaben verwenden.
- Die Remote-Konfiguration jederzeit bearbeiten oder löschen.

<img src="/support/images/en/howto/remote-storage-connection-settings/remote-manager-proton-view.png" alt="remote manager proton view" class="img-medium img-center" />


✅ **Fertig!** Ihr **Proton Drive** ist jetzt erfolgreich mit **RcloneView** verbunden.
