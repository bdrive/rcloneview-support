# Mega

## So fügen Sie Mega mit RcloneView hinzu (Windows)

### Schritt 1: Öffnen Sie den Remote Manager

- Klicken Sie oben rechts im **Remote Manager** auf **`+ New Remote`**.
- Alternativ können Sie im Explorer-Bereich auf die Schaltfläche **`+`** klicken und **`New Remote`** auswählen, um die Remote-Konfiguration zu starten.

<div class="img-grid-2">
<img src="/support/images/en/howto/remote-storage-connection-settings/add-new-remote.png" alt="add new remote" class="img-medium img-center" />
<img src="/support/images/en/howto/remote-storage-connection-settings/add-new-remote-explorer.png" alt="add new remote explorer" class="img-medium img-center" />
</div>

### Schritt 2: Wählen Sie Mega als Speicheranbieter

1. Geben Sie in der **Search Provider**-Leiste `mega` ein.
2. Klicken Sie in der Liste auf die Option **Mega**.

Sie gelangen anschließend zum Konfigurationsbildschirm für Mega.

<img src="/support/images/en/howto/remote-storage-connection-settings/select-mega-to-add-remote.png" alt="select mega to add remote" class="img-medium img-center" />

### Schritt 3: Konfigurieren Sie Ihren Mega-Remote

Füllen Sie die erforderlichen Informationen im Formular aus:

- **Remote name**: Ein aussagekräftiger Name für Ihren Remote (z. B. `MyMega`)
- **user**: Ihre **Mega-E-Mail-Adresse** (z. B. `example@gmail.com`)
- **pass**: Ihr **Mega-Passwort**

Nachdem Sie alle Werte eingegeben haben, klicken Sie auf **`Add Remote`**, um die Einrichtung abzuschließen.

<img src="/support/images/en/howto/remote-storage-connection-settings/remote-configure-mega.png" alt="remote configure mega" class="img-medium img-center" />

### Schritt 4: Bestätigen Sie den hinzugefügten Remote

Nach dem Hinzufügen erscheint Ihr neuer Mega-Remote (z. B. `MyMega`) in der Liste des **Remote Manager**.

Sie können nun:
- Auf **`Open`** klicken, um Ihre Dateien zu durchsuchen.
- Ihn in Synchronisations-, Kopier- oder Mount-Aufträgen verwenden.
- Den Remote jederzeit verwalten oder löschen.

<img src="/support/images/en/howto/remote-storage-connection-settings/remote-manager-mega-view.png" alt="remote manager mega view" class="img-medium img-center" />


✅ **Fertig!** Sie haben Ihren **Mega**-Speicher erfolgreich mit **RcloneView** verbunden.
