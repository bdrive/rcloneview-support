# RcloneView 0.5 – 릴리즈 노트

Compare와 Explorer의 강력한 기능 개선, 확장된 작업 예약(Job Schedule) 기능, 향상된 리모트 관리, 개선된 UI/UX, 주요 버그 수정을 담은 RcloneView 0.5 출시를 알려드립니다. 이번 버전은 여러분의 워크플로우를 더 매끄럽고, 스마트하고, 안정적으로 만드는 데 초점을 맞췄습니다.

## 주요 개선 사항

### Compare 창 기능 강화

- Compare 창 내에서 직접 리모트를 선택할 수 있습니다.
- Compare 창에서 사용자가 선택한 표시 설정을 저장합니다.
- 마우스 드래그를 이용한 파일 선택을 지원합니다.
- 리모트 선택 UX를 개선했습니다.
- Compare 창을 닫으면 실행 중인 작업을 자동으로 종료합니다.

### Explorer UX 개선

- 브라우저 창에서 폴더를 더블클릭하면 이제 왼쪽 트리 뷰에서 해당 폴더가 자동으로 강조 표시됩니다.

### 작업 예약(Job Schedule) 기능 추가

- Job Manager 창에 예약된 작업(Scheduled Jobs)을 위한 새 열을 추가했습니다.
- 예약된 작업 정보가 이제 History 창에도 표시됩니다.
- Job Manager의 예약된 작업에 아이콘과 툴팁을 추가했습니다.
- 작업 설정을 내보내는 기능을 추가했습니다(사용자 요청 기반).

### 리모트 관련 개선 사항

- 폴더를 선택하여 별칭(alias) 리모트를 등록하는 기능을 추가했습니다.
- iCloud 설정을 지원하도록 Rclone을 최신 버전으로 업그레이드했습니다.
- 2FA 제한으로 인해 iCloud Drive를 추가할 수 없던 문제를 해결했습니다.
- GUI 리모트 생성 시 Alibaba OSS 버킷이 표시되지 않던 문제를 수정했습니다.
- `client_id` / `client_secret` 필드를 고급 옵션(Advanced Options)으로 이동했습니다.

## UI/UX 개선

- 메인 메뉴 바의 레이아웃과 구조를 개선했습니다.
- 가시성을 높이기 위해 Sync와 Compare의 메뉴 바 색상을 변경했습니다.
- Windows 버전의 창 크기 조절 동작을 개선했습니다.
- 애플리케이션 전반의 다크 모드를 검토하고 개선했습니다.
- 루트 레벨에 디렉터리를 생성할 때의 사용자 안내 메시지를 개선했습니다.

## 버그 수정

- 소스와 대상이 동일할 때 완료(Completed) 로그에 잘못된 정보가 표시되던 문제를 수정했습니다.

## 기타 기능 개선

- 드라이런(dry-run) 결과의 상세(Details) 보기에 대상 파일명을 추가했습니다.
