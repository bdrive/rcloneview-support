# RcloneView 1.2 – 릴리스 노트

**출시일:** 2025년 12월<br/>
**지원 플랫폼:** Windows, macOS, Linux

RcloneView 1.2는 작업 실행, 알림, 클라우드 호환성, UI 동작 전반에 걸쳐 세부적인 개선 사항을 담고 있습니다. 이번 릴리스는 장시간 실행되는 작업에 대한 가시성과 제어를 강화하는 동시에, 일반적인 워크플로 문제들을 수정하는 데 중점을 두었습니다.

이메일 및 Telegram 연동이 확장되어 더 명확한 작업 상태 알림을 제공합니다. 이제 Telegram으로 알림을 받을 뿐만 아니라 작업 실행을 원격으로 제어할 수도 있습니다.
:::

## 새로운 기능

- **대용량 파일 전송용 이메일 알림:**
  대용량 파일 전송에 한해 이메일 알림을 활성화할 수 있게 되었습니다. 장시간 실행되거나 영향이 큰 작업을 더 효과적으로 추적하는 데 도움이 됩니다. 이메일 릴레이 설정은 PLUS 사용자에게 제공됩니다.

- **Telegram 알림 및 작업 제어:**
  Telegram 연동이 확장되어 작업 알림과 원격 작업 제어를 모두 지원합니다. 앱 UI에 접속하지 않고도 Telegram에서 직접 작업 상태 업데이트를 받고, 실행 중인 작업을 시작·중지·관리할 수 있습니다.

- **신규 S3 프로바이더 지원:**
  추가적인 S3 호환 스토리지 프로바이더에 대한 지원이 추가되었습니다.

- **동기화/복사 작업용 사용자 지정 플래그:**
  사용자가 동기화 또는 복사 작업을 생성할 때 사용자 지정 rclone 플래그를 지정할 수 있게 되었습니다.

- **터미널 선택 영역 복사:**
  내장 Terminal 탭에서 선택한 텍스트를 클립보드로 복사할 수 있게 되었습니다.

## 개선 사항

- **이메일 설정 UX 개선:** 이메일 알림 설정의 레이아웃과 용어를 개선하여 명확성을 높였습니다.
- **전송 화면 내 작업 이름 표시:** Transfer 탭에서 현재 진행 중인 전송의 작업 이름을 표시합니다.
- **중복 작업 및 스케줄러 실행 방지:** 예약 실행을 포함하여 동일한 작업이 여러 번 실행되는 것을 방지합니다.
- **리모트 폴더 이름 가시성:** Add Job 및 Job Manager 화면에서 발생하던 텍스트 잘림 문제를 수정했습니다.
- **Dry Run "See Details" 열 자동 크기 조정:** 텍스트 잘림을 방지하기 위해 마지막 열이 자동으로 확장됩니다.
- **New Remote 대화상자 정렬 개선:** 아이콘과 텍스트 정렬을 개선하여 더 깔끔한 레이아웃을 제공합니다.
- **Upcoming Features 패널 개선:** 채택된 답변이 있는 항목은 이제 숨겨집니다.
- **macOS /Volumes 폴더 감지:** `/Volumes` 하위의 추가 및 제거를 자동으로 감지합니다.

## 버그 수정

- **OneDrive 파일 삭제:** OneDrive 리모트에서 파일을 삭제할 수 없던 문제를 수정했습니다.
- **Google Drive 토큰 오류 (외부 rclone):** 헤드리스 Google Drive 토큰 사용 시 발생하던 연결 오류를 해결했습니다.
- **전송 상태 불일치:** 전송이 완료되었음에도 "진행 중"으로 표시되던 문제를 수정했습니다.
- **작업 이력 상태 수정:** 실행 중 강제 종료된 작업이 더 이상 실행 중 상태로 남지 않도록 수정했습니다.
- **로컬 디스크 해시 비교 멈춤 현상:** 로컬 디스크에서 해시 기반 비교가 멈추던 문제를 수정했습니다.
- **"변경 사항 없음" 상태 일관성:** 전체 작업 완료 여부와 관계없이 "변경 사항 없음" 상태가 유지됩니다.
- **전송 취소 툴팁 위치:** 취소 버튼의 툴팁 위치를 수정했습니다.
- **중복 리모트 등록:** 동일한 이름으로 리모트를 다시 추가할 때 이전 파일 목록이 재사용되던 문제를 방지했습니다.
- **rclone 설정 호환성 업데이트:** rclone 버전 변경에 맞춰 설정 값 처리 방식을 업데이트했습니다.
