# RcloneView 0.6 – 릴리스 노트

**출시일:** 2025년 4월<br/>
**지원 플랫폼:** Windows, macOS, Linux

사용자 경험, 트레이 기능, 필터링 기능 및 전반적인 안정성이 크게 개선된 RcloneView 0.6의 출시를 알려드립니다. 이번 버전은 사용자 피드백과 지속적인 개발 노력을 바탕으로 한 다양한 개선 사항, 버그 수정 및 새로운 기능을 반영합니다.

## 새로운 기능

- **시작 시 자동 마운트:** RcloneView가 이제 애플리케이션 실행 시 리모트를 자동으로 마운트하는 기능을 지원합니다. 마운트 설정은 내부 데이터베이스에 영구적으로 저장 및 관리됩니다.
- **트레이에서 빠른 마운트:** 이제 트레이 메뉴에서 직접 리모트를 마운트할 수 있어, 자주 사용하는 리모트에 대한 접근 과정이 간소화되었습니다.
- **트레이 메뉴에서 리모트 열기:** 트레이에서 리모트 디렉터리를 빠르게 열 수 있는 기능이 추가되어 접근성과 편의성이 향상되었습니다.
- **필터링을 위한 정규식 지원:** 고급 사용자는 이제 필터 뷰에서 정규식을 활용하여 rclone의 네이티브 필터 기능을 온전히 활용할 수 있습니다.
- **비교 뷰 개선 - 삭제 지원:** 이제 비교(Compare) 뷰에서 직접 파일을 삭제할 수 있어, 앱 내 관리 기능이 확장되었습니다.

## UX 및 기능 개선

- **macOS용 트레이 아이콘 적응:** 트레이 아이콘이 macOS의 밝은/어두운 배경에 따라 동적으로 조정되어 가시성이 향상되었습니다.
- **탐색기 개선:** 이제 탐색기에서 파일 경로를 직접 입력하여 해당 위치로 바로 이동할 수 있습니다.
- **현지화된 시간 표시:** 파일 타임스탬프가 이제 로컬 시스템 시간대에 맞춰 표시됩니다.
- **히스토리 뷰 UI 수정:** 시작 시간(Start Time) 필드의 가시성이 개선되었으며, 로컬 아이콘 렌더링 관련 문제가 수정되었습니다.

## 기타

- **광고 제거:** 애플리케이션이 완전히 광고 없이 제공되어, 더 깔끔하고 전문적인 경험을 제공합니다.
