# RcloneView 1.4 – 릴리스 노트

**출시일:** 2026년 3월<br/>
**지원 플랫폼:** Windows, macOS, Linux

RcloneView 1.4에서는 iCloud 사진 연동, 원격 접속을 위한 웹 기반 프런트엔드, 강화된 배치 작업(Batch Job) 기능과 함께 모든 플랫폼에서의 다양한 안정성 및 사용성 개선 사항이 도입되었습니다.

## 주요 변경 사항

- **iCloud 사진 지원 (베타, 읽기 전용):**
  RcloneView 안에서 바로 iCloud 사진 라이브러리를 탐색하고 다운로드할 수 있습니다.
  앱을 벗어나지 않고도 앨범을 확인하고, 날짜별로 검색하고, 원본을 다운로드할 수 있습니다.
  이 베타 기능은 읽기 전용 접근만 지원하며, 업로드·삭제·이름 변경은 현재 지원되지 않습니다.

- **웹 프런트엔드 (베타):**
  브라우저를 통해 어디서든 RcloneView에 접속할 수 있습니다. 내장 웹 서버는 완전한 기능을 갖춘
  듀얼 패널 파일 탐색기와 실시간 전송 모니터링을 제공합니다.
  비밀번호로 보호된 접근과 원격 연결 허용 옵션을 포함합니다.

- **강화된 배치 작업:**
  배치 작업이 단계별 실행, 단계별 드라이런(dry run), 단계 간 변수 종속성, 위험도 표시,
  템플릿 검색, 전체 실행 기록 추적 기능으로 크게 업그레이드되었습니다.
  드라이런 미리보기와 명확한 위험도 표시를 통해 복잡한 다단계 워크플로우를 자신 있게 구성할 수 있습니다.

## 신규 기능

- **필터 프리셋 저장 및 불러오기:**
  다양한 작업과 세션에서 재사용할 수 있도록 필터 프리셋을 저장하고 불러올 수 있습니다.

- **Windows 마운트 옵션:**
  Windows에서도 마운트 작업에 대한 파일 및 디렉터리 권한 설정을 사용할 수 있습니다.

## 개선 사항

- **macOS 12 지원:**
  macOS 12(Monterey) 호환성이 복원되었습니다.

- **Dropbox 점 파일(dot-file) 제외:**
  Dropbox로 동기화할 때 점 파일(예: .DS_Store)을 자동으로 제외하는 옵션이 추가되었습니다.

- **Linux 트레이 아이콘 호환성:**
  네이티브 트레이를 지원하는 Linux Cinnamon 등의 데스크톱 환경에서 시스템 트레이 처리가 개선되었습니다.

- **Windows 트레이 아이콘 색상:**
  Windows에서 트레이 아이콘 색상을 조정할 수 있게 되었습니다.

- **데이터베이스 경로 처리:**
  더 원활한 업그레이드를 위해 레거시 경로를 지원하도록 데이터베이스 경로 설정이 개선되었습니다.

- **Proton Drive 제거:**
  업스트림 변경 사항으로 인해 Proton Drive가 리모트 공급자 목록에서 제거되었습니다.

## 버그 수정

- **사용자 지정 플래그가 적용되지 않는 문제:**
  작업 실행 중 사용자 지정 플래그가 무시되던 문제를 수정했습니다.

- **이동(Move) 작업 드라이런:**
  이동 작업 유형에서 드라이런 버튼이 활성화되지 않던 문제를 수정했습니다.

- **터미널 최대화 상태:**
  터미널 패널을 최대화했을 때 발생하던 두 가지 문제(UI 요소 상태 오류 포함)를 수정했습니다.

- **데이터베이스 초기화:**
  간헐적으로 "database closed" 오류를 유발할 수 있었던 중복 sqfliteFfiInit() 호출을 제거했습니다.

- **보안 저장소 기기 ID:**
  보안 저장소의 기기 ID 처리 문제를 수정하고, Synology 자동 감지 비활성화 상태가 저장되지 않던 버그를 해결했습니다.

- **스케줄러 입력 지연:**
  스케줄링 매개변수 입력 시 발생하던 UI 지연 문제를 수정했습니다.
