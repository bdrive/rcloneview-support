# RcloneView 0.3 – 릴리스 노트

폴더 비교 기능과 향상된 Linux 지원, 그리고 다양한 UI 및 성능 개선 사항을 담은 RcloneView v0.3을 출시하게 되어 기쁩니다. 이번 업데이트에는 안정성과 사용성을 높이기 위한 다양한 버그 수정도 포함되어 있습니다.

- **macOS:** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-0.3.190.dmg)

## 새로운 기능 및 개선 사항

- **폴더 비교:** 폴더 비교 지원 기능을 자랑스럽게 소개합니다! 이제 폴더를 시각적으로 비교하고 하위 폴더 내의 차이점을 재귀적으로 분석하여 파일 관리를 더욱 직관적이고 효율적으로 수행할 수 있습니다.
- **향상된 Linux 지원:** 최신 Linux 배포판에서 발생하던 문제를 수정했습니다.
- **UI 및 UX 개선:** 메인 바 UX 디자인을 업데이트하고, 아이콘 색상과 파일 이름 텍스트의 가독성을 조정했으며, 무료 라이선스 사용자를 위한 리모트 목록 아이콘을 개선하고, 메뉴 바 레이아웃과 툴팁을 정비했으며, 창 닫기 단축키를 수정하고, macOS 창 크기 조절 가시성을 조정했습니다.
- **동기화 및 리모트 관리 개선:** 동기화 대화 상자에 단계 완료 표시기와 빠른 탐색 기능을 추가하고, SFTP 리모트 등록 시 키 파일 패스 자동 생성 문제를 수정했으며, 폴더 동기화 비교 기능을 개선하고, 리모트 등록 시 누락되던 프로바이더 아이콘 문제를 해결했습니다.
- **기타 개선 사항:** Lifetime 라이선스 및 정보(About) 표시 문제를 수정하고, RcloneView 광고 업데이트 기능을 개선했으며, 위치 패널에 Rclone 관련 설정을 표시하고, 작업 표시줄 아이콘 동작이 창을 올바르게 표시하도록 보장하고, 트레이 실행 알림 문제를 수정했으며, 창이 트레이로 정상적으로 닫히고 트레이 기반 종료를 지원하도록 했습니다.

## 버그 수정

- 리모트 없이 작업을 추가할 때 작업 및 마운트 대화 상자에서 발생하던 멈춤 문제를 해결했습니다.
- 애플리케이션 전반의 오류 처리를 개선했습니다.
- Windows 광고 로딩 문제를 수정했습니다.
