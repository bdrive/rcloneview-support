# RcloneView 0.4 – 릴리즈 노트

동기화 스케줄링, 개선된 폴더 비교 기능, 향상된 Linux 지원, 다양한 UI 및 성능 최적화를 포함한 RcloneView v0.4 출시를 발표하게 되어 기쁩니다. 이번 업데이트는 안정성과 사용성을 높이기 위해 여러 버그도 수정했습니다.

## 다운로드

- **Windows:** [x86_64](https://downloads.bdrive.com/rclone_view/builds/setup_rclone_view-0.4.215.exe)
- **macOS:** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-0.4.219.dmg)
- **Linux:** [AppImage x86_64](https://downloads.bdrive.com/rclone_view/linux/0.4/RcloneView-0.4.190-x86_64.AppImage) · [AppImage arm64](https://downloads.bdrive.com/rclone_view/linux/0.4/RcloneView-0.4.190-aarch64.AppImage) · [Deb x86_64](https://downloads.bdrive.com/rclone_view/linux/0.4/rclone_view-0.4.190+1-linux-x86_64.deb) · [Deb arm64](https://downloads.bdrive.com/rclone_view/linux/0.4/rclone_view-0.4.190+1-linux-aarch64.deb) · [RPM x86_64](https://downloads.bdrive.com/rclone_view/linux/0.4/rclone_view-0.4.190+1-linux-x86_64.rpm) · [RPM arm64](https://downloads.bdrive.com/rclone_view/linux/0.4/rclone_view-0.4.190+1-linux-aarch64.rpm)

## 새로운 기능 및 개선 사항

- **동기화 스케줄링:** 이제 작업을 주기적으로 실행되도록 예약할 수 있습니다. 스케줄링이 작동하려면 RcloneView가 실행 중이어야 합니다. 또한 시스템 트레이 통합을 통한 백그라운드 실행도 지원합니다.
- **폴더 비교:** 속도와 안정성을 높이기 위해 폴더 비교 기능을 대폭 개선했습니다. 이제 폴더를 시각적으로 비교하고 하위 폴더 내의 차이점을 재귀적으로 분석할 수 있어 파일 관리가 더욱 직관적이고 효율적입니다.
- **리모트 관리 개선:** `client_id` 및 `client_secret` 필드가 고급 옵션으로 이동되었습니다.
- **향상된 Linux 지원:** 최신 Linux 배포판과의 호환성 문제를 수정했습니다.

## 버그 수정

- 전역 설정의 로그 파일 텍스트 필드 관련 문제를 수정했습니다.
- 애플리케이션 전반의 오류 처리를 개선했습니다.
