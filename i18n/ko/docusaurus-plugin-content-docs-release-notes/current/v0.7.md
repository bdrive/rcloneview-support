# RcloneView 0.7 – 릴리스 노트

**출시일:** 2025년 6월<br/>
**지원 플랫폼:** Windows, macOS, Linux

RcloneView 0.7은 신뢰성, 사용성, 클라우드 호환성을 개선하기 위한 다양한 향상 사항, 중요한 버그 수정, 새로운 기능을 제공합니다. 이번 버전에는 성능 향상과 향후 호환성을 위한 핵심 종속성 업그레이드도 포함되어 있습니다.

## 새로운 기능

- **공개 공유 링크 생성:** 지원되는 리모트에 대해 인터페이스에서 직접 공개 공유 링크를 생성하고 복사할 수 있습니다.
- **Google Drive 중복 제거(Dedupe):** rclone의 `dedupe` 기능을 사용하여 Google Drive의 중복 파일명을 해결합니다.
- **SSL 인증서 무시 옵션:** 새 리모트를 등록할 때 SSL 검증을 무시하는 옵션을 추가했습니다.
- **Rclone 설정 비밀번호 보안 저장:** 비밀번호가 이제 플랫폼 네이티브 API를 사용하여 안전하게 저장됩니다.
- **동기화 완료 알림:** 동기화가 완료되면 "자세히 보기" 옵션이 포함된 알림이 표시됩니다.
- **잘못된 리모트 이름 경고:** 리모트 이름에 유효하지 않은 문자가 포함된 경우 사용자에게 알림을 표시합니다.
- **삭제된 리모트 처리:** 삭제된 리모트에 연결된 탭이 적절한 메시지와 함께 정상적으로 처리됩니다.
- **비교 보기 – 표시 기준 폴더 검색:** 폴더 검색이 현재 표시 필터 설정을 반영하도록 개선되었습니다.

## 개선 사항

- **다크 모드 UI 수정:** 다크 모드 사용 시 마운트 대화상자에서 검은색 텍스트가 잘 보이지 않던 문제를 수정했습니다.
- **Flutter 3.32.0용 대화상자 테마 업데이트:** 대화상자 테마 구현을 `DialogThemeData`를 사용하도록 업데이트했습니다.

## 버그 수정

- **전송 창의 속도 표시 버그:** 전송 중 속도가 부정확하게 표시되던 문제를 수정했습니다.
- **설정 찾아보기 버튼 작동 안 함:** 설정 패널의 찾아보기 버튼이 반응하지 않던 문제를 수정했습니다.
- **새 리모트 공급자 목록 누락:** 새 리모트 추가 시 공급자 옵션이 누락되던 문제를 해결했습니다.
- **리모트 목록 새로고침 문제:** 리모트 목록이 제대로 새로고침되지 않던 문제를 수정했습니다.
- **S3 폴더 생성 오류 미표시:** S3에서 폴더 생성이 실패할 때 오류 메시지가 표시되도록 수정했습니다.
- **AWS S3 폴더 복사 실패:** AWS S3 리모트에서 폴더 복사 시 오류가 표시되지 않고 조용히 실패하던 문제를 수정했습니다.
- **Windows에서 제거/재설치 오류:** 완전 제거 후 재설치 시 발생하던 비정상 동작을 해결했습니다.
- **공급자 로딩 플래그 미초기화:** 로딩 상태가 초기화되지 않아 UI 문제가 발생하던 버그를 수정했습니다.

## 종속성 업데이트

- **Flutter SDK 호환성:** Flutter 3.32.0과의 호환성을 위해 완전히 업데이트했습니다.
- **intl 패키지 업데이트:** 향상된 지역화 지원을 위해 0.20.2 버전으로 업데이트했습니다.
