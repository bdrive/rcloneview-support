# RcloneView 0.9 – 릴리스 노트

**출시일:** 2025년 9월<br/>
**지원 플랫폼:** Windows, macOS, Linux

RcloneView 0.9는 리모트 관리, Synology NAS 통합, 테마 커스터마이징, 파일 처리 부문에서 주요 개선 사항을 도입합니다. 이번 릴리스에는 다수의 버그 수정, UX 개선, Flutter 및 Linux 환경 호환성 업데이트가 포함되어 있습니다.

## 다운로드

- **Windows:** [x86_64](https://downloads.bdrive.com/rclone_view/builds/setup_rclone_view-0.9.471.exe)
- **macOS:** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-0.9.467.dmg)
- **Linux:** [AppImage x86_64](https://downloads.bdrive.com/rclone_view/linux/0.9/RcloneView-0.9.431-x86_64.AppImage) · [AppImage arm64](https://downloads.bdrive.com/rclone_view/linux/0.9/RcloneView-0.9.431-aarch64.AppImage) · [Deb x86_64](https://downloads.bdrive.com/rclone_view/linux/0.9/rclone_view-0.9.431+1-linux-x86_64.deb) · [Deb arm64](https://downloads.bdrive.com/rclone_view/linux/0.9/rclone_view-0.9.431+1-linux-aarch64.deb) · [RPM x86_64](https://downloads.bdrive.com/rclone_view/linux/0.9/rclone_view-0.9.431+1-linux-x86_64.rpm) · [RPM arm64](https://downloads.bdrive.com/rclone_view/linux/0.9/rclone_view-0.9.431+1-linux-aarch64.rpm)

## 새로운 기능

- **Synology NAS 자동 감지 및 설정:** Synology NAS 장치를 자동으로 감지하여 기본 설정으로 추가합니다.
- **클라우드 드라이브 자동 추가:** iCloud, Google Drive 등 연결된 드라이브를 자동으로 추가합니다.
- **트레이로 최소화하여 실행:** 시스템 트레이로 최소화된 상태로 시작하여 데스크톱을 더 깔끔하게 유지합니다.
- **시작 시 Synology NAS 대화상자 표시:** Synology NAS가 감지되면 대화상자가 나타납니다.

## UI 및 UX 개선

- **테마 색상 수정:** 설정 화면에서 체크박스 테마가 적용되지 않던 문제를 수정했습니다.
- **일반 설정 화면 재설계:** 더 쉬운 탐색과 명확성을 위해 레이아웃을 개선했습니다.
- **리모트 구성 UX:** 개선된 대화상자와 오류 처리로 더 직관적인 리모트 설정 경험을 제공합니다.
- **비밀번호 표시/숨기기:** rclone 설정의 비밀번호 필드에서 표시 여부를 전환할 수 있습니다.
- **파일 선택기 마이그레이션:** 더 나은 호환성을 위해 `file_picker`를 `file_selector`로 교체했습니다.
- **전역 옵션 유효성 검사:** 잘못되었거나 충돌하는 rclone 옵션을 감지하여 경고합니다.
- **앱 로컬라이제이션 폴백:** 내장 rclone 모드에서 로컬라이제이션 파일이 없을 때 이를 처리합니다.
- **커스텀 필터 중복 방지:** 작업 편집 시 필터가 중복 생성되는 것을 방지합니다.
- **지원 링크 추가:** 도움말 섹션이 이제 [rcloneview.com/support](https://rcloneview.com/support)로 연결됩니다.
- **비교 중 파일 작업 비활성화:** 비교가 실행되는 동안 버튼이 비활성화됩니다.
- **Linux 트레이 아이콘 수정:** 다양한 Linux 테마 및 데스크톱 환경에서 트레이 아이콘이 정상적으로 표시됩니다.

## 보안 및 설정

- **환경 변수를 통한 설정 암호화:** 이제 환경 변수를 사용해 rclone 설정 암호화 키를 지정할 수 있습니다.
- **ESC 키 동작 방지:** ESC 키가 의도치 않게 대화상자를 닫지 않도록 했습니다(예: 비밀번호 입력 시).
- **개선된 언인스톨러(Windows):** 제거 시 앱 프로세스를 올바르게 종료하고 파일을 삭제합니다.
- **macOS 라이선스 유지:** 재부팅 후에도 라이선스가 올바르게 유지됩니다.

## 작업 및 전송 개선

- **대량 삭제 최적화:** 대용량 파일 삭제가 이제 단일 요청으로 더 효율적으로 처리됩니다.
- **리모트 삭제 시 탭 자동 닫기:** 리모트를 삭제하면 관련된 열린 탭도 함께 닫힙니다.
- **작업 더블클릭 동작:** 이제 연결된 리모트를 탐색기 보기에서 올바르게 엽니다.

## 버그 수정

- **[AWS S3] 잘못된 폴더 삭제:** 폴더 하나를 삭제하면 관련 없는 폴더까지 삭제되던 문제를 수정했습니다.
- **[macOS] CLI 창이 표시되지 않음:** 새 리모트 추가 시 터미널 창이 열리지 않던 문제를 수정했습니다.
- **잘못된 전역 옵션:** 잘못된 rclone 플래그로 인한 시작 실패에서 앱이 이제 복구됩니다.
- **"열려 있는 파일이 너무 많음" 알림:** 시스템 파일 핸들 제한을 초과하면 사용자에게 경고합니다.
- **잘못된 파일 개수:** 전송 파일 개수가 이제 올바르게 표시됩니다.
- **터치패드 선택 버그(macOS):** 두 손가락 스크롤로 인해 파일이 선택되는 문제를 방지합니다.
- **개선된 드래그 선택:** 마우스 드래그를 이용한 파일 선택 정확도가 향상되었습니다.
- **언어 설정 유지:** 앱이 이제 저장된 언어 설정으로 올바르게 실행됩니다.
- **내장 rclone 업데이트 멈춤 현상:** 내장 모드에서 rclone 바이너리 업데이트 중 멈추던 문제를 수정했습니다.

## 기술 및 빌드

- **Flutter 3.25 업그레이드:** 더 나은 성능과 호환성을 위해 최신 Flutter 버전으로 업그레이드했습니다.
- **macOS 권한(Entitlement) 수정:** 파일 접근 및 시스템 대화상자에 필요한 누락된 권한을 추가했습니다.
- **광고 코드 제거:** 방해받지 않는 사용 경험을 위해 광고 관련 패키지 및 스크립트를 모두 정리했습니다.
