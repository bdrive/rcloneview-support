# RcloneView 0.8 – 릴리스 노트

**출시일:** 2025년 7월<br/>
**지원 플랫폼:** Windows, macOS, Linux

RcloneView 0.8은 성능 안정성, 동기화 작업 사용성, 오류 처리, 그리고 확장된 클라우드 제공업체 지원에 초점을 맞춘 주요 업데이트입니다. 이번 릴리스에는 사용자 인터페이스 동작과 호환성에 대한 주요 개선 사항도 포함되어 있습니다.

## 신규 기능 및 개선 사항

- **전송 및 완료 패널 통합:** 작업 추적을 간소화하기 위해 하나의 뷰로 병합했습니다.
- **마운트 대화상자 개선:** `Reload` 버튼을 추가하고 사용성을 위해 높이를 조정했습니다.
- **리모트 탭 드래그:** 유연한 UI 관리를 위해 패널 간에 리모트 탭을 드래그할 수 있습니다.
- **라이선스 처리 개선:** 등록 실패 또는 기기 접근 권한 상실 시 더 나은 오류 보고 및 라이선스 이전 옵션을 제공합니다.
- **Dropbox Business 지원:** 별칭 표시 문제를 수정하고 업로드 오류 처리를 개선했습니다.
- **동기화-작업 흐름:** 동기화 작업 저장 후 작업 관리자가 자동으로 열립니다.
- **예약 작업 기록 로깅:** 실패한 예약 작업의 오류 로그가 이제 기록(History)에 표시됩니다.
- **동기화 완료 팝업 비활성화:** 동기화 완료 알림을 끌 수 있는 새 옵션이 추가되었습니다.

## UI 및 UX 개선

- **탐색기 패널 더블클릭 수정:** 작업을 더블클릭할 때 리모트가 열리지 않던 문제를 수정했습니다.
- **작업 아이콘 및 이름 왼쪽 정렬:** 레이아웃 일관성을 개선했습니다.
- **정보 대화상자 다크 모드 수정:** 다크 모드에서의 표시 문제를 해결했습니다.
- **탭 동작 업데이트:** 필드를 탭할 때 전체 텍스트가 선택되는 것을 방지합니다.
- **이름 변경 필드 사전 입력:** 이름 변경 시 현재 파일명을 표시합니다.
- **메인 메뉴 스타일링:** 아이콘과 텍스트의 대비를 개선했습니다.
- **맞춤형 제거 아이콘:** 브랜드가 적용된 제거 아이콘을 추가했습니다.
- **시작 시 로그 파일 정리:** 애플리케이션이 시작 시 이전 로그 파일을 삭제하여 세션마다 더 깔끔한 로그를 보장합니다.

## 버그 수정

- **DB 업그레이드 차단 문제:** 데이터베이스 업그레이드 실패와 관련된 시작 시 충돌을 수정했습니다.
- **비교 뷰 수정:** 파일 수 표시 및 원본/대상 선택 버그를 수정했습니다.
- **작업 가져오기/내보내기 문제:** UX를 개선하고 가져오기 시 작업 이름 중복을 방지했습니다.
- **마운트 드라이브 충돌:** 마운트 시 드라이브 문자 중복을 방지했습니다.
- **Rclone 실행 경로:** 이제 항상 시스템 루트 디렉터리에서 실행됩니다.
- **폴더 삭제 시 후행 슬래시 문제:** 디렉터리가 정확하게 삭제되도록 보장합니다.
- **네트워크 끊김 처리:** 네트워크 연결이 끊길 때 팝업 메시지가 표시됩니다.
- **Windows 리모트 및 폴더 새로고침:** CLI로 추가한 후 리모트가 표시되지 않던 문제와 파일 삭제 후 폴더가 새로고침되지 않던 문제를 수정했습니다.
- **다중 파일 삭제 로깅:** 삭제된 모든 파일이 완료 목록에 기록됩니다.
- **드라이 런(Dry Run) 멈춤 현상 수정:** 많은 파일 결과를 볼 때 앱이 응답하지 않던 문제를 해결했습니다.
- **KDE Plasma 트레이 아이콘 지원:** KDE Plasma 데스크톱 환경에서 실행할 때 트레이 아이콘 표시 문제를 수정했습니다.

## 현지화 및 호환성

- **일본어 현지화:** 누락된 일본어 현지화 파일을 추가했습니다.
- **독일어 번역 수정:** 잘못된 번역 문자열로 인한 컴파일 오류를 해결했습니다.
