# RcloneView 라이선스 종류

RcloneView는 사용 수준에 맞게 두 가지 유형의 라이선스를 제공합니다: **무료(Free)**와 **플러스(Plus)**.  
아래 표는 각 플랜에서 제공되는 기능과 제한 사항을 비교한 것입니다:

## 🔍 기능 비교

| 기능                                | 무료 라이선스 | 플러스 라이선스 |
| ---------------------------------- | ------------ | ------------ |
| 리모트 마운트/마운트 해제               | ✅            | ✅            |
| 파일 탐색기 (탐색/복사/삭제)             | ✅            | ✅            |
| 폴더 비교 및 파일 관리                  | ✅            | ✅            |
| 필터를 사용한 폴더 비교                 | ❌            | ✅            |
| 파일 다운로드/업로드                   | ✅            | ✅            |
| 동기화 및 작업 관리                    | ✅            | ✅            |
| 1:N 동기화                          | ✅            | ✅            |
| 예약 기반 동기화 작업                  | ❌            | ✅            |
| 시작 시 예약 자동 실행                 | ❌            | ✅            |
| 리모트 스토리지 마운트                 | ✅            | ✅            |
| 시작 시 자동 마운트                   | ❌            | ✅            |
| 작업 기록 및 로그                    | ✅            | ✅            |
| 시스템 트레이 통합                    | ✅            | ✅            |
| 다중 RcloneView 창                  | ❌            | ✅            |
| 라이선스 키 필요                     | ❌            | ✅            |

## 플러스 라이선스 활성화 방법

1. [요금제 페이지](https://rcloneview.com/src/pricing.html)를 방문하여 라이선스를 구매합니다.
2. RcloneView에서 **도움말 > 라이선스 활성화**로 이동합니다.
3. **이메일**과 발급받은 **라이선스 키**를 입력합니다.

활성화가 완료되면 프리미엄 기능이 자동으로 해제됩니다.
