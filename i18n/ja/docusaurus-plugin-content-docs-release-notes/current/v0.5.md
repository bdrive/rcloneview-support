# RcloneView 0.5 – リリースノート

RcloneView 0.5のリリースをお知らせします。Compare(比較)およびExplorer(エクスプローラー)の強力な機能強化、Jobスケジュール機能の拡張、Remote(リモート)管理の改善、洗練されたUI/UX、そして重要なバグ修正が含まれています。今回のバージョンは、ワークフローをよりスムーズに、よりスマートに、より安定させることを目指しています。

- **macOS:** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-0.5.271.dmg)

## 主な改善点

### Compareウィンドウの機能強化

- Compareウィンドウ内で直接リモートを選択できるようになりました。
- Compareウィンドウでユーザーが選択した表示設定を保存するようになりました。
- マウスドラッグによるファイル選択に対応しました。
- リモート選択のUXを改善しました。
- Compareウィンドウを閉じると、実行中のタスクを自動的に終了するようになりました。

### ExplorerのUX改善

- ブラウザウィンドウでフォルダをダブルクリックすると、左側のツリービューで自動的にハイライト表示されるようになりました。

### Jobスケジュール機能の追加

- Job Manager(ジョブマネージャー)ウィンドウに、スケジュール済みJobの新しい列を追加しました。
- History(履歴)ウィンドウにもスケジュール済みJobの情報が表示されるようになりました。
- Job Managerにスケジュール済みJob用のアイコンとツールチップを追加しました。
- ユーザーからの要望に基づき、Job設定をエクスポートする機能を追加しました。

### リモート関連の強化

- フォルダを選択してaliasリモートを登録できる機能を追加しました。
- iCloud設定に対応するため、Rcloneを最新バージョンにアップグレードしました。
- 2FAの制限によりiCloud Driveを追加できない問題を解決しました。
- GUIでのリモート作成時にAlibaba OSSバケットが表示されない問題を修正しました。
- `client_id` / `client_secret` フィールドを詳細オプション(Advanced Options)に移動しました。

## UI/UXの改善

- メインメニューバーのレイアウトと構造を改善しました。
- 視認性向上のため、SyncおよびCompareのメニューバーの色を変更しました。
- Windows版でのウィンドウサイズ変更の挙動を改善しました。
- アプリケーション全体のダークモードを見直し、強化しました。
- ルートレベルでディレクトリを作成する際のユーザー案内メッセージを改善しました。

## バグ修正

- ソースと宛先が同一の場合に、完了ログの情報が誤って表示される問題を修正しました。

## その他の機能強化

- ドライラン結果の詳細ビューに、対象ファイル名を追加しました。
