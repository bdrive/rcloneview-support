# RcloneView 0.7 – リリースノート

**リリース日:** 2025年6月<br/>
**対応プラットフォーム:** Windows、macOS、Linux

RcloneView 0.7 では、信頼性、使いやすさ、クラウド互換性を向上させるための各種機能強化、重大なバグ修正、新機能が導入されています。また、パフォーマンス向上と将来の互換性のためにコア依存関係のアップグレードも含まれています。

- **macOS:** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-0.7.323.dmg)

## 新機能

- **公開共有リンクの取得:** 対応するリモートの公開共有リンクをインターフェースから直接生成・コピーできるようになりました。
- **Google Drive の重複排除:** rclone の `dedupe` 機能を使用して、Google Drive 上の重複ファイル名を解決できます。
- **SSL証明書無視オプション:** 新しいリモートを登録する際にSSL検証を無視するオプションを追加しました。
- **Rclone設定パスワードの安全な保存:** パスワードはプラットフォームネイティブAPIを使用して安全に保存されるようになりました。
- **同期完了通知:** 同期が完了した際に「詳細を表示」オプション付きの通知を追加しました。
- **無効なリモート名の警告:** リモート名に無効な文字が含まれている場合にユーザーへ通知されるようになりました。
- **削除されたリモートの処理:** 削除されたリモートにリンクされたタブが、適切なメッセージとともに正しく処理されるようになりました。
- **比較ビュー – 表示ベースのフォルダ検索:** フォルダ検索が現在の表示フィルター設定を考慮するようになりました。

## 改善点

- **ダークモードUIの修正:** ダークモード使用時にマウントダイアログで黒いテキストが見えにくかった問題を修正しました。
- **Flutter 3.32.0向けダイアログテーマの更新:** ダイアログテーマの実装を `DialogThemeData` を使用するように更新しました。

## バグ修正

- **転送ウィンドウの速度表示バグ:** 転送中の速度表示が不正確だった問題を修正しました。
- **設定の参照ボタンが動作しない問題:** 設定パネル内の参照ボタンが反応しなかった問題を修正しました。
- **新規リモートのプロバイダー一覧が表示されない問題:** 新しいリモートを追加する際にプロバイダーの選択肢が表示されなかった問題を解決しました。
- **リモート一覧の再読み込みの問題:** リモート一覧が正しく更新されなかった問題を修正しました。
- **S3でのフォルダ作成エラーが報告されない問題:** S3でフォルダ作成が失敗した際にエラーメッセージが表示されるようになりました。
- **AWS S3でのフォルダコピー失敗:** AWS S3リモートでフォルダをコピーする際に無言で失敗するエラーを修正しました。
- **Windowsでのアンインストール・再インストール時のエラー:** クリーンなアンインストールと再インストール後の異常な動作を解決しました。
- **プロバイダー読み込みフラグがリセットされない問題:** 読み込み状態がリセットされずUIに問題が生じるバグを修正しました。

## 依存関係の更新

- **Flutter SDK互換性:** Flutter 3.32.0との互換性のため完全に更新しました。
- **intlパッケージの更新:** ローカライゼーションサポート向上のためバージョン0.20.2に更新しました。
