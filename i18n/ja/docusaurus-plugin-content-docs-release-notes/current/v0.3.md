# RcloneView 0.3 – リリースノート

フォルダ比較、Linuxサポートの強化、そしてさまざまなUI・パフォーマンスの改善を含むRcloneView v0.3をリリースしました。今回のアップデートには、安定性と使いやすさを向上させる各種バグ修正も含まれています。

## ダウンロード

- **Windows:** [x86_64](https://downloads.bdrive.com/rclone_view/builds/setup_rclone_view-0.3.187.exe)
- **macOS:** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-0.3.190.dmg)
- **Linux:** [AppImage x86_64](https://downloads.bdrive.com/rclone_view/linux/x86-64/RcloneView-0.3.170-x86_64.AppImage) · [AppImage arm64](https://downloads.bdrive.com/rclone_view/linux/arm-64/RcloneView-0.3.170-aarch64.AppImage) · [Deb x86_64](https://downloads.bdrive.com/rclone_view/linux/x86-64/rclone_view-0.3.170+1-linux.deb) · [Deb arm64](https://downloads.bdrive.com/rclone_view/linux/arm-64/rclone_view-0.3.170+1-linux-arm64.deb) · [RPM x86_64](https://downloads.bdrive.com/rclone_view/linux/x86-64/rclone_view-0.3.170+1-linux.rpm) · [RPM arm64](https://downloads.bdrive.com/rclone_view/linux/arm-64/rclone_view-0.3.170+1-linux-arm64.rpm)

## 新機能・改善点

- **フォルダ比較:** フォルダ比較機能をついにサポートしました。フォルダの内容を視覚的に比較し、サブフォルダ内の差分を再帰的に分析できるようになり、ファイル管理がより直感的かつ効率的になりました。
- **Linuxサポートの強化:** 最新のLinuxディストリビューションでの問題を修正しました。
- **UI・UXの改善:** メインバーのUXデザインを更新し、アイコンの色とファイル名テキストの視認性を調整、無料ライセンスユーザー向けのリモートリストアイコンを改善、メニューバーのレイアウトとツールチップを洗練、ウィンドウを閉じるショートカットを修正、macOSでのウィンドウリサイズ表示を調整しました。
- **同期・リモート管理の強化:** 同期ダイアログにステップ完了インジケーターとクイックナビゲーションを追加、SFTPリモート登録時のキーファイルパスの自動生成の不具合を修正、フォルダ同期の比較を改善、リモート登録時にプロバイダーアイコンが表示されない問題を解決しました。
- **その他の改善:** 永久ライセンスと「About」表示の問題を修正、RcloneViewの広告更新機能を改善、ロケーションパネルにRclone関連の設定を表示、タスクバーアイコンの動作でウィンドウが正しく表示されるように修正、トレイ実行通知の不具合を修正、ウィンドウがトレイに閉じるようにし、トレイからの終了操作をサポートしました。

## バグ修正

- リモートを指定せずにジョブを追加した際に、ジョブおよびマウントダイアログがフリーズする問題を解決しました。
- アプリケーション全体でエラーハンドリングを改善しました。
- Windowsでの広告読み込みの問題を修正しました。
