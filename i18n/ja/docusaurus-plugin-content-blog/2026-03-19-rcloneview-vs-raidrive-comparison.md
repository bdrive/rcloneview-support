# RcloneView vs RaiDrive — クラウドマウントと管理機能の比較

> RaiDriveはクラウドを無料でWindowsドライブとしてマウントします。RcloneViewはそれに加えて、同期、スケジュール設定、検証、そして全プラットフォームで70以上のプロバイダーに対応しています。ここでは徹底比較をご紹介します。

RaiDriveは、クラウドストレージをローカルのドライブレターとしてマウントする人気のWindowsツールです。Google Drive、OneDrive、Dropboxなどのクラウドを、ネイティブのWindowsドライブとして表示させることができます。RcloneViewはマウント機能に加え、同期、スケジュール設定、フォルダ比較、暗号化を含む包括的なクラウド管理機能一式を、Windows、macOS、Linuxで提供します。

## 簡易比較

| 機能 | RcloneView | RaiDrive |
|---------|-----------|---------|
| クラウドプロバイダー | 70以上 | 約15 |
| ドライブとしてマウント | 対応 | 対応（主要機能） |
| 2ペインエクスプローラー | 対応 | 非対応 |
| クラウド間転送 | 対応 | 非対応 |
| 同期/コピージョブ | 対応 | 非対応 |
| ジョブスケジュール | 内蔵 | 非対応 |
| フォルダ比較 | 対応 | 非対応 |
| 暗号化 | Cryptリモート | 非対応 |
| S3互換 | 任意のエンドポイント | AWS S3、GCSのみ |
| 対応プラットフォーム | Windows、macOS、Linux | Windowsのみ |
| 価格 | 無料 | 無料（Pro: 約$3/月） |

## RaiDriveが優れている点

### Windowsとの統合

RaiDriveは、Windowsエクスプローラー、PowerShell、コマンドプロンプトとシームレスに統合される、正真正銘のドライブレター（G:、H:など）を作成します。

### シンプルなセットアップ

クラウドアカウントを追加し、ドライブレターを選ぶだけで完了です。RaiDriveはこの単一のワークフローに最適化されています。

### NASプロトコル対応

RaiDriveは、SFTP、FTP、WebDAVもWindowsドライブとしてマウントできます。

## RcloneViewが優れている点

### マウントを超えたすべての機能

マウントは一つの機能に過ぎません。RcloneViewはさらに以下を提供します。

- 任意の2つのプロバイダー間での**ビジュアルなファイルブラウジング**：

- **スケジュールされた同期・バックアップジョブ**：

<img src="/support/images/en/howto/rcloneview-advanced/create-job-schedule.png" alt="Job scheduling" class="img-large img-center" />

- **転送の検証**：

### クロスプラットフォーム対応

RcloneViewはWindows、macOS、Linuxで動作します。RaiDriveはWindows専用です。

### 70以上のプロバイダー

RcloneViewは、任意のS3互換エンドポイント、セルフホスト型クラウド、ニッチなサービスを含む70以上のクラウドプロバイダーに対応しています。

### 制限なしの完全無料

RcloneViewは完全に無料です。RaiDriveの無料プランには制限があり、Pro機能にはサブスクリプションが必要です。

## RcloneViewを始める

1. [rcloneview.com](https://rcloneview.com/src/download.html)から**RcloneViewをダウンロード**します。
2. **クラウドアカウントを追加**します。
3. **マウント、同期、スケジュール設定、比較**を、すべて1つのツールから行います。

マウントは、クラウド管理の始まりに過ぎません。
