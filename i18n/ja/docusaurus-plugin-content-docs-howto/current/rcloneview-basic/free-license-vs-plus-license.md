# RcloneViewライセンスの種類

RcloneViewは、さまざまな利用レベルに対応する2種類のライセンスを提供しています: **Free**と**Plus**です。  
以下の表は、各プランで利用できる機能と制限を比較したものです。

## 🔍 機能比較

| 機能                            | Freeライセンス | Plusライセンス |
| ---------------------------------- | ------------ | ------------ |
| リモートのマウント/アンマウント              | ✅            | ✅            |
| ファイルエクスプローラー(参照/コピー/削除) | ✅            | ✅            |
| フォルダ比較とファイル管理   | ✅            | ✅            |
| フィルター付きフォルダ比較         | ❌            | ✅            |
| ファイルのダウンロード/アップロード               | ✅            | ✅            |
| 同期とジョブ管理              | ✅            | ✅            |
| 1:N同期                | ✅            | ✅            |
| スケジュールベースの同期ジョブ            | ❌            | ✅            |
| 起動時のスケジュール自動開始      | ❌            | ✅            |
| リモートストレージのマウント               | ✅            | ✅            |
| 起動時の自動マウント              | ❌            | ✅            |
| ジョブ履歴とログ                 | ✅            | ✅            |
| システムトレイ統合            | ✅            | ✅            |
| 複数のRcloneViewウィンドウ         | ❌            | ✅            |
| ライセンスキーが必要               | ❌            | ✅            |

## Plusライセンスの有効化方法

1. [料金ページ](https://rcloneview.com/src/pricing.html)にアクセスし、ライセンスを購入します。
2. RcloneViewで、**Help > Activate License**に移動します。
3. **メールアドレス**と提供された**ライセンスキー**を入力します。

有効化すると、プレミアム機能が自動的にアンロックされます。
