# Tipos de licencia de RcloneView

RcloneView ofrece dos tipos de licencia para adaptarse a distintos niveles de uso: **Free** y **Plus**.  
La siguiente tabla compara las funciones y limitaciones disponibles en cada plan:

## 🔍 Comparación de funciones

| Función                            | Licencia Free | Licencia Plus |
| ---------------------------------- | ------------ | ------------ |
| Montar/Desmontar remotos           | ✅            | ✅            |
| Explorador de archivos (Examinar/Copiar/Eliminar) | ✅            | ✅            |
| Comparación de carpetas y gestión de archivos | ✅            | ✅            |
| Comparación de carpetas con filtro | ❌            | ✅            |
| Descarga/Carga de archivos         | ✅            | ✅            |
| Sincronización y gestión de tareas | ✅            | ✅            |
| Sincronización 1:N                 | ✅            | ✅            |
| Tarea de sincronización programada | ❌            | ✅            |
| Inicio automático de programación al arrancar | ❌            | ✅            |
| Montar almacenamiento remoto       | ✅            | ✅            |
| Montaje automático al arrancar     | ❌            | ✅            |
| Historial de tareas y registros    | ✅            | ✅            |
| Integración con la bandeja del sistema | ✅            | ✅            |
| Múltiples ventanas de RcloneView   | ❌            | ✅            |
| Requiere clave de licencia         | ❌            | ✅            |

## Cómo activar la licencia Plus

1. Visita la [página de precios](https://rcloneview.com/src/pricing.html) y adquiere una licencia.
2. En RcloneView, ve a **Help > Activate License**.
3. Introduce tu **correo electrónico** y la **clave de licencia** proporcionada.

Una vez activada, las funciones premium se desbloquearán automáticamente.
