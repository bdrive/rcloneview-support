# RcloneView 0.2 – Notas de la versión

Nos complace lanzar RcloneView v0.2, que incluye soporte para múltiples ventanas, sincronización mejorada, actualizaciones de paquetes para Linux y una interfaz de usuario refinada para un mejor rendimiento y usabilidad. Esta actualización también incluye correcciones de errores críticos para garantizar una experiencia más fluida.

## Descarga

- **Windows:** [x86_64](https://downloads.bdrive.com/rclone_view/builds/setup_rclone_view-0.2.158.exe)
- **macOS:** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-0.2.158.dmg)
- **Linux:** [AppImage x86_64](https://downloads.bdrive.com/rclone_view/linux/RcloneView-x86_64.AppImage) · [AppImage arm64](https://downloads.bdrive.com/rclone_view/linux/RcloneView-aarch64.AppImage)

## Nuevas funciones y mejoras

- **Soporte para distribuciones de Linux:** Se introdujo soporte para el paquete AppImage en Linux, junto con una mejor gestión de ventanas y operaciones multiventana en Linux. Tenga en cuenta que se requiere la instalación de FUSE para usar el AppImage en Linux. Los paquetes RPM y DEB estarán disponibles próximamente.
- **Soporte multiventana:** Se introdujo la gestión de múltiples ventanas en todas las plataformas, incluidas Linux, macOS y Windows.
- **Sincronización 1:N:** Se agregó la funcionalidad de sincronización de múltiples trabajos (1:N) y se refinó su experiencia de usuario.
- **Íconos de proveedores:** Se agregaron íconos de proveedores en el nuevo cuadro de diálogo de remotos y en las listas de remotos para una mayor claridad visual.
- **Actualizaciones de Rclone integrado:** Se implementaron capacidades de actualización interna/integrada de Rclone en Windows y macOS, con un cierre adecuado al salir de la aplicación.
- **Interfaz de usuario y accesibilidad mejoradas:** Incluyendo información sobre herramientas mejorada y cuadros de diálogo emergentes de configuración y ajustes que se pueden mover.
- **Cuadros de diálogo de trabajos y montaje mejorados:** Se evita que la aplicación se congele al agregar trabajos sin un remoto.
- **Se mejoraron las pantallas de cálculo de velocidad de transmisión y tiempo restante.**
- **Actualización a Rclone 1.68.1:** Se garantizó el manejo adecuado de nuevos remotos.

## Corrección de errores

- Se corrigieron discrepancias en los cálculos de tiempo del historial de trabajos.
- Se resolvieron problemas de truncamiento de texto en la barra de menú.
- Se corrigieron problemas de tamaño de fuente en los cuadros de diálogo de mensajes.
- Se eliminaron los bloqueos y se mejoró el manejo de errores en toda la aplicación.
