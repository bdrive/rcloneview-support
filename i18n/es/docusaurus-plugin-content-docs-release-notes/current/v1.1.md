# RcloneView 1.1 – Notas de la versión

**Fecha de lanzamiento:** noviembre de 2025<br/>
**Plataformas:** Windows, macOS, Linux

RcloneView 1.1 introduce nuevas funciones prácticas y mejoras de usabilidad en la navegación de remotos, el acceso a terminal, la gestión de ajustes y el comportamiento de la interfaz. Esta actualización se centra en mejorar la eficiencia del flujo de trabajo y en solucionar problemas clave reportados por los usuarios.

## Nuevas funciones

- **Pestaña de terminal integrada:**
  Ahora hay disponible una nueva pestaña de terminal directamente dentro de RcloneView. Ya no necesitas cambiar a una terminal externa o al símbolo del sistema: ejecuta comandos de shell, comandos de rclone y diagnósticos rápidos directamente desde la aplicación.

- **Bloqueo de la aplicación (protección con contraseña):**
  RcloneView ahora se puede proteger con una contraseña a nivel de aplicación para evitar accesos no autorizados.

- **Compatibilidad con ubicación personalizada de la base de datos:**
  Ahora puedes elegir una ruta de guardado personalizada para el archivo `rclone_view.db`, lo que permite configuraciones de espacio de trabajo flexibles, uso de unidades externas o sincronización selectiva.

- **Gestor de remotos: visualización del tamaño de elementos en modo lista:**
  El modo lista en el Gestor de remotos ahora muestra información detallada del tamaño de cada elemento.

- **Comando "Copiar ruta completa (con remoto)":**
  Añade una forma cómoda de copiar la ruta remota completa, incluyendo el nombre del remoto, al portapapeles.

- **Vista previa del árbol de carpetas de remoto virtual:**
  Al añadir o editar un remoto virtual, ahora se muestra la vista previa del árbol de carpetas para ayudar a los usuarios a elegir rutas fácilmente.

## Mejoras de interfaz y experiencia de usuario

- **Redimensionamiento adaptable de pestañas y ventana:** Las pestañas se redimensionan dinámicamente cuando se estrecha la ventana de la aplicación, evitando que el texto se corte o se recorte.
- **Correcciones en la navegación de carpetas en la nube:** iCloud Drive y carpetas en la nube similares ahora aparecen correctamente en el explorador de archivos.
- **Correcciones en el panel de ajustes:** Las rutas de ejecutables locales de rclone ya no desaparecen inesperadamente.
- **Campo de nombre de carpeta para remotos combinados:** Los remotos de tipo combine ahora utilizan un campo de texto dedicado para la introducción del nombre de carpeta, mejorando la claridad.
- **Ventana de transferencia – Correcciones del modo oscuro:** Se resolvieron incoherencias de color al cambiar de tema.
- **Mejoras en el panel de historial:**
  - Los mensajes de error ya no se truncan.
  - Los errores de tareas programadas ahora aparecen correctamente en el Historial.
- **Mejoras en el diseño del Explorador:** Se mejoraron los anchos de columna en las vistas del Explorador para una mejor legibilidad y alineación.
- **Mejor gestión de tipos de remoto no compatibles:** Los diálogos "Acerca de" para tipos de remoto con API no compatibles ahora muestran mensajes de reserva adecuados.

## Corrección de errores

- **Problema de actualización de la interfaz en la operación Comparar:** Se corrigió un error por el que la interfaz seguía actualizándose después de que las tareas de copia basadas en Comparar ya hubieran finalizado.
- **Corrección del explorador de Sincronizar/Comparar:** Se resolvió un problema por el que la navegación de una carpeta remota fallaba cuando el nodo de disco local estaba preseleccionado.
- **Gestión de rutas en la terminal de Windows:** Se corrigió un problema en Windows por el que las configuraciones de rclone no se leían correctamente dentro del entorno de terminal.
- **Error de validación del nombre de tarea:** Se corrigió una excepción de base de datos que se activaba durante las comprobaciones de unicidad del nombre de tarea.
- **Corrección de la etiqueta de éxito "Sin cambios":** Las vistas de detalle de archivo ahora muestran correctamente "Éxito" cuando no se detectan cambios durante la sincronización/comparación.
- **Regla de precisión de finalización al 99%:** Cuando los resultados son casi, pero no perfectamente, idénticos, el indicador de progreso ahora muestra 99% en lugar de mostrar incorrectamente 100%.
