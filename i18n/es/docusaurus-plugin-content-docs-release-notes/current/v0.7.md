# RcloneView 0.7 – Notas de la versión

**Fecha de lanzamiento:** junio de 2025<br/>
**Plataformas:** Windows, macOS, Linux

RcloneView 0.7 introduce varias mejoras, correcciones de errores críticos y nuevas funciones diseñadas para mejorar la fiabilidad, la usabilidad y la compatibilidad con la nube. Esta versión también incluye actualizaciones de dependencias principales para un mejor rendimiento y compatibilidad futura.

## Nuevas funciones

- **Obtención de enlaces de compartición pública:** Genera y copia enlaces de compartición pública para remotos compatibles directamente desde la interfaz.
- **Deduplicación para Google Drive:** Resuelve nombres de archivo duplicados en Google Drive utilizando la función `dedupe` de rclone.
- **Opción para ignorar el certificado SSL:** Opción para ignorar la verificación SSL al registrar un nuevo remoto.
- **Almacenamiento seguro de la contraseña de configuración de rclone:** Las contraseñas ahora se almacenan de forma segura utilizando las API nativas de la plataforma.
- **Notificaciones de finalización de sincronización:** Se añadió una notificación con la opción "Ver detalles" cuando finaliza una sincronización.
- **Aviso de nombre de remoto no válido:** Se notifica a los usuarios si el nombre de un remoto contiene caracteres no válidos.
- **Gestión de remotos eliminados:** Las pestañas vinculadas a remotos eliminados ahora se gestionan correctamente con un mensaje apropiado.
- **Vista de comparación: búsqueda de carpetas basada en la visualización:** La búsqueda de carpetas ahora respeta la configuración de filtro de visualización actual.

## Mejoras

- **Corrección de la interfaz en modo oscuro:** Se corrigieron problemas de visibilidad de texto negro en el diálogo de Montar al usar el modo oscuro.
- **Actualización del tema de diálogo para Flutter 3.32.0:** Se actualizó la implementación del tema de diálogo para usar `DialogThemeData`.

## Corrección de errores

- **Error en la visualización de velocidad en la ventana de transferencia:** Se corrigió la visualización inexacta de la velocidad durante las transferencias.
- **El botón Examinar de la configuración no funcionaba:** Se corrigió el botón de examinar que no respondía en el panel de configuración.
- **Falta la lista de proveedores al añadir un nuevo remoto:** Se resolvió el problema de las opciones de proveedor faltantes al añadir un nuevo remoto.
- **Problema de recarga de la lista de remotos:** Se corrigió el problema por el cual la lista de remotos no se actualizaba correctamente.
- **No se informaba del error de creación de carpetas en S3:** Ahora se muestran mensajes de error cuando falla la creación de carpetas en S3.
- **Fallo al copiar carpetas en AWS S3:** Se corrigió un error silencioso al copiar carpetas en remotos de AWS S3.
- **Error de desinstalación/reinstalación en Windows:** Se resolvió el comportamiento anómalo tras una desinstalación limpia y reinstalación.
- **La marca de carga del proveedor no se restablecía:** Se corrigió un error por el cual el estado de carga no se restablecía, causando problemas en la interfaz.

## Actualizaciones de dependencias

- **Compatibilidad con Flutter SDK:** Totalmente actualizado para compatibilidad con Flutter 3.32.0.
- **Paquete intl actualizado:** Actualizado a la versión 0.20.2 para mejorar la compatibilidad con la localización.
