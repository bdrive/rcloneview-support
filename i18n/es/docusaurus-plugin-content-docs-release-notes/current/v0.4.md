# RcloneView 0.4 – Notas de la versión

Nos complace anunciar el lanzamiento de RcloneView v0.4, que incluye programación de sincronización, una función mejorada de comparación de carpetas, soporte mejorado para Linux y diversas optimizaciones de interfaz y rendimiento. Esta actualización también resuelve varios errores para mejorar la estabilidad y la usabilidad.

## Nuevas funciones y mejoras

- **Programación de sincronización:** Ahora puedes programar trabajos para que se ejecuten periódicamente. Ten en cuenta que RcloneView debe estar en ejecución para que la programación funcione. La aplicación también admite la ejecución en segundo plano con integración en la bandeja del sistema.
- **Comparación de carpetas:** Hemos mejorado significativamente la comparación de carpetas para lograr mayor velocidad y estabilidad. Ahora puedes comparar carpetas visualmente y analizar recursivamente las diferencias dentro de las subcarpetas, lo que hace que la gestión de archivos sea más intuitiva y eficiente.
- **Mejoras en la gestión de remotos:** Los campos `client_id` y `client_secret` se han movido a las opciones avanzadas.
- **Soporte mejorado para Linux:** Se corrigieron problemas de compatibilidad con distribuciones recientes de Linux.

## Corrección de errores

- Se corrigió un problema con el campo de texto del archivo de registro en Configuración global.
- Se mejoró el manejo de errores en toda la aplicación.
