# Mega S4 白标浏览器（预览版）

本页面是 RcloneView 如何以完全品牌化的 **Mega S4 浏览器**桌面应用程序形式为您的用户提供服务的私有预览。

此白标方案的目标是：

- 让 **Mega S4 品牌在整个产品中处于中心位置**。
- 让用户在安装后**可以轻松连接他们的 Mega S4 账户**。
- 确保 **Mega S4 是每个管理和导航工作流程中的首选**。
- 为 Mega S4 品牌部署提供**优先支持和维护**。

---

## 1. 品牌套件与视觉整合

我们提供一套品牌套件，让 Mega S4 成为整个产品中主要且可见的品牌。所有 RcloneView 品牌标识都将被移除，或仅在法律要求的地方保留（例如内部版本字符串）。

主要要素：

- 应用程序名称设置为 **“Mega S4 浏览器”**（或其他商定的名称）。
- 所有 RcloneView 徽标替换为 **Mega S4 徽标**：
  - 桌面快捷方式和任务栏图标。
  - 安装程序图标。
  - 应用内页头和窗口图标。
- 在适当的情况下调整颜色主题，以匹配 Mega S4 的品牌调色板。

## 2. 安装后 Mega S4 远程连接向导

安装完成后，系统会立即引导用户连接他们的 Mega S4 账户，让他们无需额外的设置步骤即可开始使用该服务。

主要行为：

- 在安装向导结束时，启动应用会打开**“添加 Mega S4 远程”**作为默认的引导流程。
- 该向导已针对 Mega S4 进行了简化和预配置：
  - 提供商类型预先选择为 **Mega S4**。
  - 默认仅显示 Mega S4 特定的选项。
  - 高级选项仍可通过**“显示高级选项”**链接访问。

<img src="/support/images/en/howto/remote-storage-connection-settings/post-install-mega-remote-wizard.png" alt="post install mega remote wizard" class="img-large img-center" />


### 主页快速访问向导

如果用户跳过初始连接或关闭向导：

- 主屏幕右侧面板会显示一个**大型 Mega S4 图块**。
- 点击该图块可随时重新打开**“添加 Mega S4 远程”**向导。

<img src="/support/images/en/howto/remote-storage-connection-settings/on-home-quick-access-wizard-for-mega.png" alt="on home quick access wizard for mega" class="img-large img-center" />

 
这确保了连接 Mega S4 始终是新用户或回访用户最显眼的下一步操作。

## 3. Mega S4 优先的导航与管理

添加 Mega S4 远程后，界面会经过优化，使 Mega S4 在所有关键导航和管理界面中始终保持突出可见。

### 3.1 重新启动时固定的 Mega S4 面板

配置 Mega S4 远程后：

- 在后续启动时，界面默认会打开并**固定 Mega S4 面板**。
- 典型布局：
  - 左侧：本地磁盘或其他来源。
  - 右侧：用户的 **MYMegaS4** 远程。
- 用户可以通过拖放自由重新排列面板，因此 Mega S4 面板可以根据用户偏好的工作流程在左右两侧之间移动。
- 无论 Mega S4 面板当前位于哪一侧，这都能使本地文件夹与 Mega S4 之间的重复同步或复制操作变成一键式操作。

<img src="/support/images/en/howto/remote-storage-connection-settings/mega-panel-pinned-on-re-launch.png" alt="mega panel pinned on re launch" class="img-large img-center" />

### 3.2 “新建远程”与“远程管理器”中的 Mega S4 优先显示

为突出 Mega S4 作为主要存储提供商：

- 在**新建远程**对话框中：
  - Mega S4 显示在**提供商列表顶部**。
  - Mega S4 图块会以视觉方式突出显示，以鼓励用户选择。
- 在**远程管理器**中：
  - Mega S4 远程（例如 `MYMegaS4`）显示在**远程列表顶部**。
  - 在列表视图和卡片视图中，Mega S4 远程都会以视觉方式强调，以便更快访问。

<img src="/support/images/en/howto/remote-storage-connection-settings/maga-first-in-management-dialog.png" alt="maga first in management dialog" class="img-large img-center" />
