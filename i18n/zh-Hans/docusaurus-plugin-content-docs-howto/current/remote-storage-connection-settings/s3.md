# AWS S3 和兼容 S3 的存储

RcloneView 支持 **Amazon S3** 以及多种**兼容 S3 的对象存储提供商**，包括 Wasabi、Backblaze、Cloudflare R2 等主流厂商。

这些远程使用基于 S3 协议的相同配置界面。选择后，您需要提供访问凭据，并可选择性地设置自定义端点 URL。


## 如何添加兼容 S3 的远程

### 第 1 步：打开新建远程配置窗口

- 从顶部菜单的 **`Remote`** 下点击 **`+ New Remote`**。
- 或者，点击资源管理器窗格中的 **`+`** 按钮，然后选择 **`New Remote`**。

### 第 2 步：配置 S3 提供商

#### 在 **Provider** 选项卡中：
1. 搜索 `s3` 或您的提供商名称（例如 `aws`、`wasabi`、`minio`）。
2. 从结果列表中选择 **`S3`**。

#### 在 **Options** 选项卡中：
3. 设置您的**提供商**（例如 `AWS`、`Wasabi`、`Minio` 等）。
4. 输入您的[**访问密钥 ID（Access Key ID）**](/howto/cloud-storage-setting/aws-account-info#step-by-step-get-your-aws-access-key-id-and-secret-access-key)。
5. 输入您的[**私有访问密钥（Secret Access Key）**](/howto/cloud-storage-setting/aws-account-info#step-by-step-get-your-aws-access-key-id-and-secret-access-key)。
6. 为您的存储桶选择正确的**区域（region）**（例如 `ap-northeast-2`）。
   如有需要，可设置自定义**端点（endpoint）**（适用于不使用 AWS 端点格式的提供商）。

### 第 3 步：命名并保存远程

#### 在 **Name** 选项卡中：
7. 为远程指定一个名称（例如 `myWasabi`、`minio-prod`）。

#### 在 **Save** 选项卡中：
8. 点击 **Save** 完成设置。
