# RcloneView 许可证类型

RcloneView 提供两种许可证类型以支持不同程度的使用需求：**免费版（Free）** 和 **Plus 版（Plus）**。  
下表比较了各方案的可用功能与限制：

## 🔍 功能对比

| 功能                            | 免费许可证 | Plus 许可证 |
| ---------------------------------- | ------------ | ------------ |
| 挂载/卸载远程                | ✅            | ✅            |
| 文件浏览器（浏览/复制/删除） | ✅            | ✅            |
| 文件夹比较与文件管理           | ✅            | ✅            |
| 带过滤条件的文件夹比较         | ❌            | ✅            |
| 文件下载/上传               | ✅            | ✅            |
| 同步与任务管理              | ✅            | ✅            |
| 1:N 同步                | ✅            | ✅            |
| 基于计划的同步任务            | ❌            | ✅            |
| 启动时自动运行计划      | ❌            | ✅            |
| 挂载远程存储               | ✅            | ✅            |
| 启动时自动挂载              | ❌            | ✅            |
| 任务历史与日志                 | ✅            | ✅            |
| 系统托盘集成            | ✅            | ✅            |
| 多个 RcloneView 窗口         | ❌            | ✅            |
| 需要许可证密钥               | ❌            | ✅            |

## 如何激活 Plus 许可证

1. 访问[定价页面](https://rcloneview.com/src/pricing.html)并购买许可证。
2. 在 RcloneView 中，前往 **帮助 > 激活许可证**。
3. 输入您的**电子邮箱**和提供的**许可证密钥**。

激活后，高级功能将自动解锁。
