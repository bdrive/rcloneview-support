# RcloneView 0.3 – 发行说明

我们很高兴发布 RcloneView v0.3，此版本包含文件夹对比功能、增强的 Linux 支持，以及多项界面和性能改进。本次更新还包含多项错误修复，以提升稳定性和易用性。

## 新功能与改进

- **文件夹对比：** 我们很自豪地推出文件夹对比支持！现在您可以直观地对比文件夹，并递归分析子文件夹内的差异，让文件管理更加直观高效。
- **增强的 Linux 支持：** 修复了在新版本 Linux 发行版中出现的问题。
- **界面与用户体验改进：** 更新了主工具栏的用户体验设计，调整了图标颜色与文件名文本以提升可见性，改进了免费许可证用户的远程列表图标，优化了菜单栏布局与工具提示，修复了窗口关闭快捷键问题，并调整了 macOS 窗口调整大小时的可见性。
- **同步与远程管理增强：** 在同步对话框中新增了步骤完成指示器和快速导航，修复了 SFTP 远程注册中密钥文件密码自动生成的问题，改进了文件夹同步对比，并解决了远程注册中缺失服务商图标的问题。
- **其他增强：** 修复了终身许可证和关于页面的显示问题，改进了 RcloneView 广告更新功能，在位置面板中显示 Rclone 相关设置，确保任务栏图标行为能正确显示窗口，修复了托盘运行通知，确保窗口关闭至托盘并支持基于托盘的退出。

## 错误修复

- 解决了在未选择远程的情况下添加任务时，任务和挂载对话框出现的卡死问题。
- 改进了整个应用程序的错误处理。
- 修复了 Windows 广告加载问题。
