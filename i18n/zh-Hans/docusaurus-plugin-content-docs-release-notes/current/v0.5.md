# RcloneView 0.5 – 发布说明

我们很高兴地宣布 RcloneView 0.5 正式发布，本次更新为“比较”和“文件浏览器”带来了强大的增强功能，扩展了“任务计划”能力，改进了远程管理，优化了 UI/UX，并修复了一些关键问题。本版本旨在让您的工作流程更流畅、更智能、更稳定。

## 主要改进

### 比较窗口功能增强

- 支持在“比较”窗口内直接选择远程。
- 保存用户在“比较”窗口中选择的显示设置。
- 支持鼠标拖拽选择文件。
- 改进了远程选择的用户体验。
- 关闭“比较”窗口时自动终止正在运行的任务。

### 文件浏览器用户体验增强

- 在浏览器窗口中双击文件夹后，现在会自动在左侧树形视图中高亮显示该文件夹。

### 任务计划功能新增

- 在“任务管理器”窗口中为计划任务新增了列。
- “历史记录”窗口中现在也会显示计划任务信息。
- 在“任务管理器”中为计划任务添加了图标和工具提示。
- 新增导出任务设置的功能（根据用户需求添加）。

### 远程相关增强

- 新增通过选择文件夹注册别名远程的功能。
- 将 Rclone 升级到最新版本，以支持 iCloud 配置。
- 修复了因双重验证（2FA）限制导致无法添加 iCloud Drive 的问题。
- 修复了在 GUI 中创建远程时阿里云 OSS 存储桶不显示的问题。
- 将 `client_id` / `client_secret` 字段移至高级选项中。

## UI/UX 改进

- 改进了主菜单栏的布局和结构。
- 更改了“同步”和“比较”菜单栏的颜色，以提升可视性。
- 改进了 Windows 版本的窗口大小调整行为。
- 审查并增强了应用程序中的深色模式。
- 改进了在根目录创建文件夹时的用户提示信息。

## 问题修复

- 修复了源和目标相同时“已完成”日志中信息不正确的问题。

## 其他功能增强

- 在预演（dry-run）结果的详情视图中新增了目标文件名。
