# RcloneView 1.4 – 发行说明

**发布日期：** 2026 年 3 月<br/>
**支持平台：** Windows、macOS、Linux

RcloneView 1.4 引入了 iCloud 照片集成、用于远程访问的网页版前端、增强的批处理任务功能，以及覆盖所有平台的众多稳定性与易用性改进。

## 亮点功能

- **iCloud 照片支持（测试版，只读）：**
  直接在 RcloneView 中浏览和下载 iCloud 照片图库中的照片。
  查看相册、按日期搜索并下载原图——无需离开应用。
  该测试版功能仅支持只读访问，目前暂不支持上传、删除和重命名。

- **网页版前端（测试版）：**
  通过浏览器随时随地访问 RcloneView。内置网页服务器提供功能完整的
  双面板文件浏览器和实时传输监控。
  支持密码保护访问，并可选择允许远程连接。

- **增强的批处理任务：**
  批处理任务已大幅升级，新增分步执行、单步试运行（dry run）、
  步骤间变量依赖、风险等级提示、模板搜索以及完整的执行历史记录。
  借助试运行预览和清晰的风险提示，放心构建复杂的多步骤工作流。

## 新增功能

- **筛选预设的保存与加载：**
  保存和加载筛选预设，可在不同任务和会话中重复使用。

- **Windows 挂载选项：**
  Windows 平台现已支持挂载操作的文件和目录权限设置。

## 改进

- **支持 macOS 12：**
  恢复了对 macOS 12（Monterey）的兼容性。

- **Dropbox 隐藏文件排除：**
  新增选项，可在同步到 Dropbox 时自动排除隐藏文件（例如 .DS_Store）。

- **Linux 托盘图标兼容性：**
  改进了对 Linux Cinnamon 及其他支持原生托盘的桌面环境的系统托盘处理。

- **Windows 托盘图标颜色：**
  Windows 平台现已支持调整托盘图标颜色。

- **数据库路径处理：**
  改进了数据库路径配置，新增旧版路径支持，使升级更加顺畅。

- **移除 Proton Drive：**
  由于上游变更，Proton Drive 已从远程存储提供商列表中移除。

## 问题修复

- **自定义参数未生效：**
  修复了任务执行期间自定义参数被忽略的问题。

- **移动任务的试运行：**
  修复了移动（Move）类型任务的“试运行”按钮无法启用的问题。

- **终端最大化状态：**
  修复了终端面板最大化时的两个问题，包括界面元素状态错误。

- **数据库初始化：**
  消除了重复调用 sqfliteFfiInit() 的问题，该问题可能导致间歇性的“数据库已关闭”错误。

- **安全存储中的设备 ID：**
  修复了安全存储中设备 ID 的处理问题，并解决了 Synology 自动检测禁用状态未被保存的错误。

- **调度器输入延迟：**
  修复了输入调度参数时出现的界面卡顿问题。
