# RcloneView 0.4 – 发行说明

我们很高兴地宣布 RcloneView v0.4 版本发布,此版本包含同步计划任务、改进的文件夹比较功能、增强的 Linux 支持,以及各种 UI 和性能优化。此更新还修复了多个错误,以提升稳定性和可用性。

## 新功能与改进

- **同步计划任务:** 现在您可以设置作业定期运行。请注意,RcloneView 必须处于运行状态,计划任务才能生效。该应用还支持后台运行,并集成了系统托盘功能。
- **文件夹比较:** 我们大幅改进了文件夹比较功能,提升了速度和稳定性。您现在可以直观地比较文件夹,并递归分析子文件夹内的差异,使文件管理更加直观高效。
- **远程管理增强:** `client_id` 和 `client_secret` 字段已移至高级选项中。
- **增强的 Linux 支持:** 修复了与近期 Linux 发行版的兼容性问题。

## 错误修复

- 修复了全局设置中日志文件文本字段的一个问题。
- 增强了整个应用程序的错误处理机制。
