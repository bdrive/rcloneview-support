# RcloneView 0.2 – 发行说明

我们很高兴发布 RcloneView v0.2，此版本带来了多窗口支持、增强的同步功能、Linux 软件包更新以及经过优化的界面，以提升性能与易用性。本次更新还包含多项关键错误修复，带来更流畅的使用体验。

## 下载

- **Windows：** [x86_64](https://downloads.bdrive.com/rclone_view/builds/setup_rclone_view-0.2.158.exe)
- **macOS：** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-0.2.158.dmg)
- **Linux：** [AppImage x86_64](https://downloads.bdrive.com/rclone_view/linux/RcloneView-x86_64.AppImage) · [AppImage arm64](https://downloads.bdrive.com/rclone_view/linux/RcloneView-aarch64.AppImage)

## 新特性与改进

- **Linux 发行版支持：** 引入了 Linux AppImage 软件包支持，并改进了 Linux 平台上的窗口管理和多窗口操作。请注意，在 Linux 上使用 AppImage 需要安装 FUSE。RPM 和 DEB 软件包即将推出。
- **多窗口支持：** 在 Linux、macOS 和 Windows 各平台上引入了多窗口管理功能。
- **1:N 同步：** 新增多任务（1:N）同步功能，并优化了其用户体验。
- **提供商图标：** 在新建远程对话框和远程列表中添加了提供商图标，以提升视觉清晰度。
- **内嵌 Rclone 更新：** 在 Windows 和 macOS 上实现了内部/内嵌的 Rclone 更新能力，并在应用退出时正确关闭相关进程。
- **改进的界面与无障碍支持：** 包括改进的工具提示，以及可移动的设置和配置弹窗对话框。
- **增强的任务与挂载对话框：** 防止在未选择远程的情况下添加任务时出现卡死。
- **改进了传输速度和剩余时间计算的显示。**
- **升级至 Rclone 1.68.1：** 确保对新远程的正确处理。

## 错误修复

- 修正了任务历史记录时间计算中的差异。
- 解决了菜单栏中文本截断的问题。
- 修复了消息对话框中字体大小的问题。
- 消除了卡死问题，并改进了整个应用的错误处理。
