# RcloneView 1.1 – 发行说明

**发布日期：** 2025 年 11 月<br/>
**支持平台：** Windows、macOS、Linux

RcloneView 1.1 在远程浏览、终端访问、设置管理和界面行为等方面引入了实用的新功能和易用性改进。此次更新专注于提升工作流效率，并解决用户反馈的关键问题。

- **macOS：** [通用版](https://downloads.bdrive.com/rclone_view/builds/RcloneView-1.1.533.dmg)

## 新功能

- **内置终端标签页：**
  RcloneView 现已直接内置全新的终端标签页。您无需再切换到外部终端或命令提示符——可直接在应用内运行 shell 命令、rclone 命令以及快速诊断。

- **应用锁定（密码保护）：**
  RcloneView 现在可以设置应用级密码，以防止未经授权的访问。

- **自定义数据库位置支持：**
  您现在可以为 `rclone_view.db` 文件选择自定义保存路径，从而实现灵活的工作区设置、外部驱动器使用或选择性同步。

- **远程管理器：列表模式下的项目大小显示：**
  远程管理器的列表模式现在会显示每个项目的详细大小信息。

- **"复制完整路径（含远程）"命令：**
  新增了一种便捷方式，可将包含远程名称在内的完整远程路径复制到剪贴板。

- **虚拟远程文件夹树预览：**
  在添加或编辑虚拟远程时，现在会显示文件夹树预览，方便用户轻松选择路径。

## 界面与体验改进

- **响应式标签页与窗口缩放：** 当应用窗口变窄时，标签页会动态调整大小，避免文本被裁剪或截断。
- **云文件夹浏览修复：** iCloud Drive 及类似云文件夹现在可以正确显示在文件浏览器中。
- **设置面板修复：** 本地 rclone 可执行文件路径不再意外消失。
- **合并远程文件夹名称字段：** Combine 类型的远程现在使用专用文本字段输入文件夹名称，提升清晰度。
- **传输窗口——深色模式修复：** 解决了切换主题时出现的颜色不一致问题。
- **历史记录面板改进：**
  - 错误消息不再被截断。
  - 计划任务的错误现在会正确显示在历史记录中。
- **资源管理器布局增强：** 改进了资源管理器视图中的列宽，以获得更好的可读性和对齐效果。
- **对不支持的远程类型的更好处理：** 对于 API 不受支持的远程类型，关于对话框现在会显示恰当的回退消息。

## 缺陷修复

- **比较操作界面刷新问题：** 修复了基于比较的复制任务已经完成后界面仍持续刷新的错误。
- **同步/比较浏览器修复：** 解决了在预先选择本地磁盘节点时浏览远程文件夹失败的问题。
- **Windows 终端路径处理：** 修复了 Windows 下终端环境中 rclone 配置未被正确读取的问题。
- **任务名称验证错误：** 修正了任务名称唯一性检查过程中触发的数据库异常。
- **"无更改"成功标签修复：** 当同步/比较过程中未检测到任何更改时，文件详情视图现在会正确显示"成功"。
- **99% 完成度准确规则：** 当结果几乎（但并非完全）一致时，进度指示器现在会显示 99%，而不是错误地显示 100%。
