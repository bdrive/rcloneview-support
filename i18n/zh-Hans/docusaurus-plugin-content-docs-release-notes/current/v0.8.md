# RcloneView 0.8 – 发行说明

**发布日期：** 2025年7月<br/>
**支持平台：** Windows、macOS、Linux

RcloneView 0.8 是一次重大更新，重点关注性能稳定性、同步任务可用性、错误处理以及扩展的云存储提供商支持。此版本还包含对用户界面行为和兼容性的关键改进。

- **macOS：** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-0.8.368.dmg)

## 新功能与增强

- **统一传输与已完成面板：** 合并为单一视图，简化任务跟踪。
- **挂载对话框增强：** 新增 `Reload` 按钮并调整了高度以提升可用性。
- **远程标签拖拽：** 可在面板之间拖拽远程标签，实现灵活的界面管理。
- **许可证处理改进：** 在注册失败或设备访问丢失时，提供更好的错误报告和许可证转移选项。
- **Dropbox Business 支持：** 修复了别名可见性问题，并改进了上传错误处理。
- **同步到任务流程：** 保存同步任务后自动打开任务管理器。
- **计划任务历史日志：** 计划任务失败的错误日志现已显示在历史记录中。
- **禁用同步完成弹窗：** 新增选项，可关闭同步完成通知。

## 界面与用户体验改进

- **资源管理器面板双击修复：** 修复了双击任务时远程未打开的问题。
- **任务图标与名称左对齐：** 改进了布局一致性。
- **关于对话框深色模式修复：** 解决了深色模式下的可见性问题。
- **点按行为更新：** 防止点按字段时选中整段文本。
- **重命名字段预填充：** 重命名时显示当前文件名。
- **主菜单样式：** 增强了图标和文字的对比度。
- **自定义卸载图标：** 添加了品牌化的卸载图标。
- **启动时清理日志文件：** 应用现在会在启动时删除上一次的日志文件，确保每次会话的日志更干净。

## 错误修复

- **数据库升级阻塞：** 修复了因数据库升级失败导致的启动崩溃问题。
- **比较视图修复：** 修正了文件计数显示以及源/目标选择的错误。
- **任务导入/导出问题：** 改进了用户体验，并防止导入时出现重复的任务名称。
- **挂载驱动器冲突：** 防止挂载时出现驱动器盘符重复。
- **Rclone 执行路径：** 现在始终从系统根目录运行。
- **文件夹删除时的尾部斜杠：** 确保目录删除的准确性。
- **网络断开处理：** 网络断开时现在会显示弹窗提示。
- **Windows 远程与文件夹刷新：** 修复了通过 CLI 添加后远程不显示，以及删除文件后文件夹不刷新的问题。
- **多文件删除日志记录：** 所有已删除的文件现在都会记录在已完成列表中。
- **Dry Run 冻结修复：** 解决了查看大量文件结果时应用无响应的问题。
- **KDE Plasma 托盘图标支持：** 修复了在 KDE Plasma 桌面环境下运行时托盘图标显示的问题。

## 本地化与兼容性

- **日语本地化：** 补充了缺失的日语本地化文件。
- **德语翻译修复：** 解决了因翻译字符串错误导致的编译错误。
