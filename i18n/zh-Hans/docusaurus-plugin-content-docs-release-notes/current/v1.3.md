# RcloneView 1.3 – 发行说明

**发布日期：** 2026年2月<br/>
**支持平台：** Windows、macOS、Linux

RcloneView 1.3 新增了任务管理功能，并扩展了与 Slack 和 Discord 等常用通讯工具的集成。此外还对邮件通知、界面行为和任务执行跟踪进行了改进。

## 新功能

- **批量任务支持（测试版）：**
  现在您可以将多个任务分组为一个批次并一起运行。
  此测试版功能允许您定义一系列按顺序执行的任务，从而更方便地管理相关任务。
  未来的更新中计划提供更多控制选项和错误处理机制。

- **新增“移动”任务类型：**
  现已支持新的“移动”任务类型，可将文件从源位置移动到目标位置进行传输。

- **重试失败的任务：**
  现在您可以重试失败的任务，而无需手动重新创建它们。

- **Slack 集成：**
  RcloneView 现已支持通过 Slack 接收通知，并可在 Slack 频道中控制任务。

- **Discord 集成：**
  现在您可以通过 Discord 接收任务通知。

- **远程控制设置指南位置调整：**
  Telegram 和 Slack 远程控制设置指南的链接现已移至集成部分顶部，更便于查找。

- **支持印度尼西亚语：**
  界面现已支持 Bahasa Indonesia（印度尼西亚语）。

## 改进

- **邮件通知增强：**
  - 改进了邮件设置面板的布局和清晰度
  - 任务状态邮件现已包含详细信息
  - 修复了邮件配置界面中偶发的界面崩溃问题
  - 免费试用结束后，邮件通知仍可继续使用

- **远程图标更新：** 新增了多个图标并更新了旧图标，方便更轻松地识别远程。

- **文件列表加载稳定性：** 防止在文件加载缓慢时切换页面导致显示过期内容的情况。

- **“无更改”结果处理：** 无论任务完成状态如何，详情视图中的“无更改”状态现在都能被正确保留。

## 错误修复

- **OneDrive 文件删除：** 解决了导致 OneDrive 远程无法删除文件的问题。
- **传输状态显示错误：** 修复了任务已完成但文件仍显示为传输中的问题。
- **本地磁盘哈希比较：** 修复了在本地存储上使用哈希校验比较文件时出现的卡死问题。
- **任务历史记录不准确：** 被强制终止的任务现在会正确地从历史记录的“运行中”状态中移除。
- **传输取消提示框错位：** 修正了传输视图中提示框的位置问题。
- **重新注册同名远程：** 防止在添加与之前同名的远程时复用过期的文件列表。
- **试运行列截断：** 试运行详情面板中的最后一列现在会自动扩展以适应内容。
- **外部 rclone 与 Google Drive 令牌错误：** 修复了在无头模式下使用外部 rclone 配合 Google Drive 令牌时出现的连接问题。
