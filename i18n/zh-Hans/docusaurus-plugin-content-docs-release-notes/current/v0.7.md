# RcloneView 0.7 – 发行说明

**发布日期：** 2025 年 6 月<br/>
**支持平台：** Windows、macOS、Linux

RcloneView 0.7 引入了多项增强功能、关键错误修复和新特性，旨在提升可靠性、易用性和云兼容性。此版本还升级了核心依赖项，以获得更好的性能和未来兼容性。

## 新增功能

- **公开分享链接获取：** 可直接在界面中为受支持的远程生成并复制公开分享链接。
- **Google Drive 去重：** 使用 rclone 的 `dedupe` 功能解决 Google Drive 上的重复文件名问题。
- **忽略 SSL 证书选项：** 注册新远程时可选择忽略 SSL 验证。
- **Rclone 配置密码的安全存储：** 密码现在使用平台原生 API 进行安全存储。
- **同步完成通知：** 同步完成后新增带有“查看详情”选项的通知。
- **非法远程名称警告：** 当远程名称包含无效字符时会通知用户。
- **已删除远程的处理：** 链接到已删除远程的标签页现在会以适当的提示妥善处理。
- **比较视图 —— 基于显示设置的文件夹搜索：** 文件夹搜索现在会遵循当前的显示筛选设置。

## 改进

- **深色模式界面修复：** 修复了深色模式下挂载对话框中黑色文字不可见的问题。
- **适配 Flutter 3.32.0 的对话框主题更新：** 更新了对话框主题实现，改用 `DialogThemeData`。

## 错误修复

- **传输窗口中的速度显示错误：** 修正了传输过程中速度显示不准确的问题。
- **设置中“浏览”按钮无响应：** 修复了设置面板中浏览按钮无响应的问题。
- **新建远程时提供商列表缺失：** 解决了添加新远程时提供商选项缺失的问题。
- **远程列表刷新问题：** 修复了远程列表无法正常刷新的问题。
- **S3 文件夹创建错误未提示：** 现在当 S3 上文件夹创建失败时会显示错误消息。
- **AWS S3 文件夹复制失败：** 修复了在 AWS S3 远程中复制文件夹时的静默错误。
- **Windows 上卸载/重新安装错误：** 解决了完全卸载后重新安装出现异常行为的问题。
- **提供商加载标志未重置：** 修复了加载状态未重置导致的界面问题。

## 依赖更新

- **Flutter SDK 兼容性：** 已完全更新以兼容 Flutter 3.32.0。
- **intl 软件包更新：** 更新至 0.20.2 版本，以改善本地化支持。
