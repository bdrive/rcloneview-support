# RcloneView 0.6 – 发行说明

**发布日期：** 2025年4月<br/>
**平台：** Windows、macOS、Linux

我们很高兴宣布 RcloneView 0.6 正式发布，本次更新为用户体验、托盘功能、筛选能力以及整体稳定性带来了显著改进。此版本基于用户反馈和持续的开发工作,包含了大量增强功能、错误修复和新特性。

## 新功能

- **启动时自动挂载：** RcloneView 现已支持在应用启动时自动挂载远程。挂载设置会持久保存并在内部数据库中进行管理。
- **从托盘快速挂载：** 现在可以直接从托盘菜单挂载远程,简化了常用远程的操作流程。
- **从托盘菜单打开远程：** 新增了从托盘快速打开远程目录的功能,提升了可访问性和便利性。
- **筛选支持正则表达式：** 高级用户现在可以在筛选视图中使用正则表达式,充分利用 rclone 原生的筛选能力。
- **对比视图增强 - 支持删除：** 现在可以直接在对比视图中删除文件,扩展了应用内的管理能力。

## 用户体验与功能改进

- **macOS 托盘图标自适应：** 托盘图标会根据 macOS 的浅色和深色背景动态调整,以提升可见性。
- **资源管理器增强：** 用户现在可以在资源管理器中手动输入文件路径,直接导航至目标位置。
- **本地化时间显示：** 文件时间戳现在会按照本地系统时区进行显示。
- **历史记录视图界面修复：** 改进了开始时间字段的可见性,并修复了本地图标渲染的问题。

## 其他

- **移除广告：** 应用现已完全无广告,提供更简洁、更专业的使用体验。
