# 云存储空间不足？使用 RcloneView 清理并释放所有账户的空间

> 云存储空间快用完了？了解如何通过找出臃肿的文件夹、删除重复文件并优化云账户,来重新获得数 GB 的空间。

当你的云存储达到上限时,这不仅仅是不便——它会阻止上传、拖慢工作效率,并迫使你进行昂贵的升级。RcloneView 帮助你快速找出正在占用空间的内容,并在一个地方清理所有账户。

## 找出占用空间的元凶

RcloneView 的文件浏览器和排序功能可让你即时查看哪些文件和文件夹占用了最多空间。按大小、修改日期或文件数量排序,即可发现你可能已经忘记的臃肿文件夹。

![Cloud storage file listing](/images/en/blog/new-remote.png)

常见的元凶包括旧的项目备份、重复下载的文件和归档的邮件附件。清晰地了解你的存储状况后,你就能就删除哪些内容做出明智的决定。

## 识别并删除重复文件

重复文件积累得很快——不小心上传的文件、备份副本和版本冗余都在悄悄占用空间。RcloneView 的比较工具可帮助你在各个云账户中找出重复文件。

![Comparing files and folders](/images/en/howto/rcloneview-basic/compare-display-select.png)

识别出来后,你就可以安全地删除重复文件,在不丢失重要数据的情况下回收大量存储空间。

## 批量删除旧文件

使用 RcloneView 的筛选和选择工具,按年龄、扩展名或命名模式定位文件。筛选出早于特定日期的文件,然后一次性批量删除。

![Job history and execution](/images/en/howto/rcloneview-basic/job-history.png)

这种方法对于你不再需要的临时文件、下载内容和归档内容尤其有用。

## 快速上手

1. 从 [rcloneview.com](https://rcloneview.com/src/download.html) **下载 RcloneView**。
2. 连接你的云账户,并导航到占用空间最多的文件夹。
3. 使用排序和筛选功能找出需要删除的文件,然后批量删除。
4. 实时监控存储空间的回收情况。

立即回收你的云存储空间,摆脱昂贵升级的困扰。
