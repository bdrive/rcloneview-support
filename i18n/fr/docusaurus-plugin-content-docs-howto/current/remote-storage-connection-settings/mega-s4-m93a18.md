# Explorateur en marque blanche Mega S4 (Aperçu)

Cette page est un aperçu privé de la manière dont RcloneView peut être fourni en tant qu'application de bureau **Mega S4 Explorer** entièrement personnalisée pour vos utilisateurs.

L'objectif de cette proposition en marque blanche est de :

- Mettre la **marque Mega S4 au premier plan** dans l'ensemble du produit.
- Rendre **la connexion au compte Mega S4 immédiate et sans effort** dès l'installation.
- Garantir que **Mega S4 soit le premier choix** dans chaque flux de gestion et de navigation.
- Fournir un **support et une maintenance prioritaires** spécifiquement pour les déploiements de marque Mega S4.

---

## 1. Kit de marque et intégration visuelle

Nous fournissons un kit de marque dans lequel Mega S4 est la marque principale et visible dans tout le produit. Toute la marque RcloneView est supprimée ou conservée uniquement lorsque légalement requis (par exemple, les chaînes de version internes).

Éléments clés :

- Nom de l'application défini sur **« Mega S4 Explorer »** (ou tout autre libellé convenu).
- Tous les logos RcloneView remplacés par le **logo Mega S4** :
  - Raccourcis de bureau et icônes de la barre des tâches.
  - Icônes de l'installateur.
  - En-tête de l'application et icône de fenêtre.
- Accents de couleur ajustés pour correspondre à la palette de marque de Mega S4 le cas échéant.

## 2. Assistant de distant Mega S4 post-installation

Immédiatement après l'installation, les utilisateurs sont guidés pour connecter leur compte Mega S4 afin de pouvoir commencer à utiliser le service sans étapes de configuration supplémentaires.

Comportements principaux :

- À la fin de l'assistant d'installation, le lancement de l'application ouvre **« Ajouter un distant Mega S4 »** comme flux d'intégration par défaut.
- L'assistant est simplifié et préconfiguré pour Mega S4 :
  - Le type de fournisseur est présélectionné sur **Mega S4**.
  - Seules les options spécifiques à Mega S4 sont affichées par défaut.
  - Les options avancées restent disponibles derrière un lien **« Afficher les options avancées »**.

<img src="/support/images/en/howto/remote-storage-connection-settings/post-install-mega-remote-wizard.png" alt="post install mega remote wizard" class="img-large img-center" />


### Assistant d'accès rapide sur l'écran d'accueil

Si l'utilisateur ignore la connexion initiale ou ferme l'assistant :

- Le panneau de droite de l'écran d'accueil affiche une **grande tuile Mega S4**.
- Cliquer sur la tuile rouvre l'assistant **« Ajouter un distant Mega S4 »** à tout moment.

<img src="/support/images/en/howto/remote-storage-connection-settings/on-home-quick-access-wizard-for-mega.png" alt="on home quick access wizard for mega" class="img-large img-center" />

 
Cela garantit que la connexion à Mega S4 reste toujours l'action suivante la plus visible pour les utilisateurs nouveaux ou récurrents.

## 3. Navigation et gestion privilégiant Mega S4

Une fois qu'un distant Mega S4 est ajouté, l'interface est optimisée pour garder Mega S4 visible de manière proéminente sur tous les écrans clés de navigation et de gestion.

### 3.1 Panneau Mega S4 épinglé au relancement

Une fois qu'un distant Mega S4 a été configuré :

- Lors des lancements suivants, l'interface s'ouvre avec un **panneau Mega S4 épinglé** par défaut.
- Disposition typique :
  - Côté gauche : disque local ou une autre source.
  - Côté droit : le distant **MYMegaS4** de l'utilisateur.
- Les utilisateurs peuvent librement réorganiser les panneaux par glisser-déposer, de sorte que le panneau Mega S4 puisse être déplacé entre le côté gauche et le côté droit selon leur flux de travail préféré.
- Cela rend les opérations répétées de synchronisation ou de copie entre les dossiers locaux et Mega S4 accessibles en un clic, quel que soit le côté sur lequel se trouve actuellement le panneau Mega S4.

<img src="/support/images/en/howto/remote-storage-connection-settings/mega-panel-pinned-on-re-launch.png" alt="mega panel pinned on re launch" class="img-large img-center" />

### 3.2 Mega S4 en premier dans « Nouveau distant » et le « Gestionnaire de distants »

Pour mettre en avant Mega S4 comme fournisseur de stockage principal :

- Dans la boîte de dialogue **Nouveau distant** :
  - Mega S4 apparaît en **haut de la liste des fournisseurs**.
  - La tuile Mega S4 est visuellement mise en évidence pour encourager sa sélection.
- Dans le **Gestionnaire de distants** :
  - Le distant Mega S4 (par exemple, `MYMegaS4`) est affiché en **haut de la liste des distants**.
  - Dans les vues liste et carte, le distant Mega S4 est visuellement mis en avant pour un accès plus rapide.

<img src="/support/images/en/howto/remote-storage-connection-settings/maga-first-in-management-dialog.png" alt="maga first in management dialog" class="img-large img-center" />

## 4. Support et maintenance prioritaires pour Mega S4

Pour le déploiement en marque blanche Mega S4, nous fournissons une filière de support et de maintenance dédiée.

Services inclus :

- **Documentation dédiée**  
  - Pages de manuel distinctes spécifiquement destinées aux utilisateurs de **Mega S4 Explorer**.  
  - Guides étape par étape pour la connexion, la synchronisation et le dépannage avec Mega S4.

- **Traitement prioritaire des incidents**  
  - Les problèmes des utilisateurs Mega S4 sont prioritaires dans la file d'attente du support.  
  - **Réponse d'urgence** pour les incidents critiques affectant les utilisateurs Mega S4 (par exemple, échecs de connexion, problèmes d'accès aux données).

- **Mises à jour continues du produit**  
  - Mises à niveau régulières du client de bureau incluses dans le cadre du contrat de maintenance.  
  - Possibilité de déployer de nouvelles fonctionnalités RcloneView sous la marque Mega S4 après approbation conjointe.

## 5. Prochaines étapes

Si vous souhaitez avancer avec ce produit en marque blanche :

1. **Alignement de la marque**
   - Confirmer les directives d'utilisation du logo Mega S4 et les couleurs de la marque.
   - Décider du nom final du produit (par exemple, *Mega S4 Explorer*).
2. **Définition de l'expérience**
   - Valider le flux d'intégration et les comportements privilégiant Mega S4 décrits ci-dessus.
   - Ajuster les paramètres par défaut (par exemple, mode de synchronisation par défaut, chemin de montage par défaut).
3. **Version pilote**
   - Nous livrons une version pilote privée et une documentation spécifique à Mega S4 pour les tests internes.

Cette page et son URL sont destinées uniquement à Mega S4 et aux parties prenantes partenaires et n'apparaîtront pas dans la navigation ni la recherche publiques. Elle peut être partagée en toute sécurité en tant que lien direct lors des évaluations et des discussions pilotes.
