# Comment utiliser un coupon de réduction

:::info Politique relative aux coupons de réduction
Un coupon de réduction est émis par utilisateur (adresse e-mail) et peut être appliqué quelle que soit la quantité achetée. Il ne peut être utilisé qu'une seule fois.
:::

Suivez les étapes ci-dessous pour appliquer un coupon de réduction lors de l'achat d'une licence Plus ou Lifetime.

1. Enregistrez ou copiez le code du coupon de réduction depuis l'e-mail que vous avez reçu.
2. Rendez-vous sur rcloneview.com et ouvrez [la page d'achat Plus](https://rcloneview.com/src/pricing), puis cliquez sur **Buy Now** pour la licence Plus ou Lifetime.

<img src="/support/images/en/howto/FAQ/click-buy-now-buttons.png" alt="click buy now buttons" class="img-medium img-center" />

3. Saisissez l'adresse e-mail à laquelle vous souhaitez recevoir votre code de licence, puis cliquez sur **Continue**.
<img src="/support/images/en/howto/FAQ/enter-email-address.png" alt="enter email address" class="img-medium img-center" />

4. Cliquez sur `**Enter Coupon Code**`.

<img src="/support/images/en/howto/FAQ/click-enter-coupon-code.png" alt="click enter coupon code" class="img-medium img-center" />

5. Collez le code de coupon de réduction que vous avez enregistré, cliquez sur **Apply**, vérifiez que le total réduit s'affiche, puis finalisez le paiement.

<img src="/support/images/en/howto/FAQ/enter-your-coupon-code.png" alt="enter your coupon code" class="img-medium img-center" />
