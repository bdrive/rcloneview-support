# Types de licences RcloneView

RcloneView propose deux types de licences pour répondre à différents niveaux d'utilisation : **Free** et **Plus**.  
Le tableau suivant compare les fonctionnalités et limitations disponibles pour chaque offre :

## 🔍 Comparaison des fonctionnalités

| Fonctionnalité                              | Licence Free | Licence Plus |
| -------------------------------------------- | ------------ | ------------ |
| Monter/Démonter des distants                | ✅            | ✅            |
| Explorateur de fichiers (Parcourir/Copier/Supprimer) | ✅   | ✅            |
| Comparaison de dossiers et gestion des fichiers | ✅        | ✅            |
| Comparaison de dossiers avec filtre         | ❌            | ✅            |
| Téléchargement/Envoi de fichiers            | ✅            | ✅            |
| Synchronisation et gestion des tâches       | ✅            | ✅            |
| Synchronisation 1:N                          | ✅            | ✅            |
| Tâche de synchronisation planifiée          | ❌            | ✅            |
| Démarrage automatique de la planification au démarrage | ❌ | ✅         |
| Monter un stockage distant                  | ✅            | ✅            |
| Montage automatique au démarrage            | ❌            | ✅            |
| Historique et journaux des tâches           | ✅            | ✅            |
| Intégration à la barre système              | ✅            | ✅            |
| Plusieurs fenêtres RcloneView                | ❌            | ✅            |
| Clé de licence requise                      | ❌            | ✅            |

## Comment activer la licence Plus

1. Visitez la [page des tarifs](https://rcloneview.com/src/pricing.html) et achetez une licence.
2. Dans RcloneView, allez dans **Aide > Activer la licence**.
3. Entrez votre **e-mail** et la **clé de licence** fournie.

Une fois activées, les fonctionnalités premium seront déverrouillées automatiquement.
