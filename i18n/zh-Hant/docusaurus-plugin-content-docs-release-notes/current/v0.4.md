# RcloneView 0.4 – 版本說明

我們很高興宣布推出 RcloneView v0.4，此版本包含同步排程、改進的資料夾比較功能、增強的 Linux 支援，以及各種 UI 與效能最佳化。此更新也修復了多個錯誤，以提升穩定性與可用性。

## 新功能與改進

- **同步排程：** 您現在可以排定工作定期執行。請注意，RcloneView 必須保持執行中排程才能運作。應用程式也支援搭配系統匣整合的背景執行。
- **資料夾比較：** 我們大幅改進了資料夾比較功能，提升速度與穩定性。您現在可以視覺化比較資料夾，並遞迴分析子資料夾中的差異，讓檔案管理更直覺、更有效率。
- **遠端管理增強：** `client_id` 與 `client_secret` 欄位已移至進階選項中。
- **增強的 Linux 支援：** 修復了與近期 Linux 發行版的相容性問題。

## 錯誤修復

- 修復了全域設定中日誌檔文字欄位的問題。
- 增強了整個應用程式的錯誤處理。
