# RcloneView 0.3 – 版本說明

我們很高興發布 RcloneView v0.3，此版本推出了資料夾比較功能、強化了 Linux 支援，並帶來多項使用者介面與效能改進。此更新同時包含多項錯誤修正，以提升穩定性與易用性。

## 新功能與改進

- **資料夾比較：** 我們很自豪地推出資料夾比較支援！現在您可以以視覺化方式比較資料夾，並遞迴分析子資料夾內的差異，讓檔案管理更直覺、更有效率。
- **強化 Linux 支援：** 修正了近期 Linux 發行版的相關問題。
- **使用者介面與體驗改進：** 更新了主選單列的使用者體驗設計、調整圖示顏色與檔名文字以提升可視性、改善免費授權使用者的遠端清單圖示、優化選單列版面與工具提示、修正視窗關閉快速鍵，並調整 macOS 視窗縮放的可視性。
- **同步與遠端管理強化：** 在同步對話框中新增步驟完成指示與快速導覽、修正 SFTP 遠端註冊時金鑰檔案密碼自動產生的問題、改善資料夾同步比較功能，並解決遠端註冊時提供商圖示遺失的問題。
- **其他改進：** 修正終身授權與「關於」顯示問題、改善 RcloneView 廣告更新功能、在位置面板中顯示與 Rclone 相關的設定、確保工作列圖示行為能正確顯示視窗、修正系統匣執行通知、確保視窗可關閉至系統匣並支援以系統匣方式結束程式。

## 錯誤修正

- 解決在未指定遠端的情況下新增工作時，工作與掛載對話框發生凍結的問題。
- 改進了整個應用程式的錯誤處理機制。
- 修正 Windows 廣告載入問題。
