# RcloneView 0.5 – 版本說明

我們很高興宣布推出 RcloneView 0.5，此版本為「比較」與「檔案總管」帶來強大的功能強化、擴充了工作排程功能、改進了遠端管理、優化了 UI/UX，並修復了重要的錯誤。此版本旨在讓您的工作流程更順暢、更智慧、更穩定。

## 主要改進

### 比較視窗功能強化

- 可直接在比較視窗中選取遠端。
- 儲存使用者在比較視窗中選擇的顯示設定。
- 支援以滑鼠拖曳方式選取檔案。
- 改進遠端選取的使用者體驗。
- 關閉比較視窗時，自動終止任何執行中的工作。

### 檔案總管 UX 強化

- 在瀏覽視窗中雙擊資料夾時，現在會自動在左側樹狀檢視中反白顯示該資料夾。

### 工作排程功能新增

- 在工作管理員視窗中，為排程工作新增了欄位。
- 排程工作的資訊現在也會顯示在歷史紀錄視窗中。
- 在工作管理員中為排程工作新增了圖示與工具提示。
- 新增匯出工作設定的功能（依使用者要求開發）。

### 遠端相關強化

- 新增可透過選取資料夾來註冊別名遠端的功能。
- 將 Rclone 升級至最新版本，以支援 iCloud 設定。
- 解決因雙重驗證（2FA）限制而無法新增 iCloud Drive 的問題。
- 修復在 GUI 建立遠端時，Alibaba OSS 儲存桶未出現的問題。
- 將 `client_id` / `client_secret` 欄位移至進階選項中。

## UI/UX 改進

- 改進主選單列的版面配置與結構。
- 變更同步與比較的選單列顏色，以提升可視性。
- 改進 Windows 版本的視窗縮放行為。
- 檢視並強化了整個應用程式的深色模式。
- 改進在根目錄層級建立目錄時的使用者提示訊息。

## 錯誤修復

- 修復來源與目的地相同時，導致「已完成」記錄中資訊錯誤的問題。

## 其他功能強化

- 在乾跑（dry-run）結果的詳細資料檢視中，新增目標檔案名稱。
