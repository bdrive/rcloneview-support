# RcloneView 1.1 – 發行說明

**發行日期：** 2025 年 11 月<br/>
**平台：** Windows、macOS、Linux

RcloneView 1.1 在遠端瀏覽、終端機存取、設定管理和介面行為方面引入了實用的新功能與可用性增強。此次更新著重於提升工作流程效率，並解決使用者回報的重要問題。

- **macOS：** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-1.1.533.dmg)

## 新功能

- **內建終端機分頁：**
  RcloneView 現已直接內建全新的終端機分頁。您不再需要在外部終端機或命令提示字元之間切換——可直接在應用程式中執行 shell 命令、rclone 命令及快速診斷。

- **應用程式鎖定（密碼保護）：**
  RcloneView 現在可以使用應用程式層級密碼進行保護，防止未經授權的存取。

- **自訂資料庫位置支援：**
  您現在可以為 `rclone_view.db` 檔案選擇自訂儲存路徑，方便彈性的工作區設定、外接硬碟使用或選擇性同步。

- **遠端管理員：清單模式下顯示項目大小：**
  遠端管理員的清單模式現在會顯示每個項目的詳細大小資訊。

- **「複製完整路徑（含遠端）」命令：**
  新增一個方便的方式，可將完整的遠端路徑（包含遠端名稱）複製到剪貼簿。

- **虛擬遠端資料夾樹狀預覽：**
  在新增或編輯虛擬遠端時，現在會顯示資料夾樹狀預覽，方便使用者選取路徑。

## 使用者介面與體驗改進

- **響應式分頁與視窗大小調整：** 當應用程式視窗變窄時，分頁會動態調整大小，避免文字被裁切或截斷。
- **雲端資料夾瀏覽修正：** iCloud Drive 及類似的雲端資料夾現在可以正確顯示在檔案瀏覽器中。
- **設定面板修正：** 本機 rclone 執行檔路徑不再無故消失。
- **合併遠端資料夾名稱欄位：** 合併類型遠端現在使用專用文字欄位輸入資料夾名稱，提升清晰度。
- **傳輸視窗——深色模式修正：** 解決了切換主題時的色彩不一致問題。
- **歷史紀錄面板改進：**
  - 錯誤訊息不再被截斷。
  - 排程工作的錯誤現在會正確顯示在歷史紀錄中。
- **檔案總管版面增強：** 改進了檔案總管檢視中的欄寬，提升可讀性與對齊效果。
- **改進對不支援遠端類型的處理：** 對於 API 不受支援的遠端類型，「關於」對話方塊現在會顯示適當的備援訊息。

## 錯誤修正

- **比較操作介面重新整理問題：** 修正了在以比較為基礎的複製工作已完成後，介面仍持續重新整理的錯誤。
- **同步／比較瀏覽器修正：** 解決了在預先選取本機磁碟節點時，瀏覽遠端資料夾會失敗的問題。
- **Windows 終端機路徑處理：** 修正了 Windows 上終端機環境中無法正確讀取 rclone 設定的問題。
- **工作名稱驗證錯誤：** 修正了在檢查工作名稱唯一性時觸發的資料庫例外狀況。
- **「無變更」成功標籤修正：** 當同步／比較過程中未偵測到變更時，檔案詳細資訊檢視現在會正確顯示「成功」。
- **99% 完成準確度規則：** 當結果幾乎（但非完全）相同時，進度指示器現在會顯示 99%，而非錯誤地顯示 100%。
