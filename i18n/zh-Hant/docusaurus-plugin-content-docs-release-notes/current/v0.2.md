# RcloneView 0.2 – 版本說明

我們很高興發布 RcloneView v0.2,此版本新增多視窗支援、強化的同步功能、Linux 套件更新,以及經過改良的介面,帶來更好的效能與可用性。此更新也包含重要的錯誤修正,讓使用體驗更加順暢。

## 下載

- **Windows:** [x86_64](https://downloads.bdrive.com/rclone_view/builds/setup_rclone_view-0.2.158.exe)
- **macOS:** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-0.2.158.dmg)
- **Linux:** [AppImage x86_64](https://downloads.bdrive.com/rclone_view/linux/RcloneView-x86_64.AppImage) · [AppImage arm64](https://downloads.bdrive.com/rclone_view/linux/RcloneView-aarch64.AppImage)

## 新功能與改進

- **Linux 發行版支援:** 新增 Linux AppImage 套件支援,並改善 Linux 上的視窗管理與多視窗操作。請注意,在 Linux 上使用 AppImage 需要安裝 FUSE。RPM 與 DEB 套件將於稍後推出。
- **多視窗支援:** 在 Linux、macOS 與 Windows 等平台上,新增多視窗管理功能。
- **1:N 同步:** 新增多工作(1:N)同步功能,並改善其使用體驗。
- **提供者圖示:** 在新增遠端對話框與遠端清單中新增提供者圖示,提升視覺辨識度。
- **內嵌 Rclone 更新:** 在 Windows 與 macOS 上實作內部/內嵌 Rclone 更新功能,並確保應用程式結束時正常關閉。
- **改良的介面與無障礙功能:** 包含改良的工具提示,以及可移動的設定與配置彈出對話框。
- **強化工作與掛載對話框:** 避免在未指定遠端時新增工作導致當機。
- **改善傳輸速度與剩餘時間計算的顯示。**
- **升級至 Rclone 1.68.1:** 確保正確處理新的遠端。

## 錯誤修正

- 修正工作歷程記錄時間計算的誤差。
- 解決選單列文字截斷的問題。
- 修正訊息對話框中的字型大小問題。
- 消除當機問題,並改善整個應用程式的錯誤處理。
