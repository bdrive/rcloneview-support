# RcloneView 0.7 – 版本說明

**發布日期：** 2025年6月<br/>
**支援平台：** Windows、macOS、Linux

RcloneView 0.7 帶來多項增強功能、重要錯誤修正以及新功能，旨在提升可靠性、易用性與雲端相容性。此版本也升級了核心依賴套件，以獲得更好的效能與未來相容性。

## 新功能

- **公開分享連結取得：** 可直接在介面中為支援的遠端產生並複製公開分享連結。
- **Google Drive 去重複：** 使用 rclone 的 `dedupe` 功能解決 Google Drive 上的重複檔名問題。
- **忽略 SSL 憑證選項：** 註冊新遠端時可選擇忽略 SSL 驗證。
- **Rclone 設定密碼安全儲存：** 密碼現在使用平台原生 API 安全儲存。
- **同步完成通知：** 同步完成時新增附有「檢視詳情」選項的通知。
- **無效遠端名稱警告：** 若遠端名稱包含無效字元，將通知使用者。
- **已刪除遠端的處理：** 連結至已刪除遠端的分頁現在會妥善顯示對應訊息。
- **比較檢視 – 依顯示篩選的資料夾搜尋：** 資料夾搜尋現在會遵循目前的顯示篩選設定。

## 改進

- **深色模式介面修正：** 修正深色模式下掛載對話框中黑色文字顯示不清的問題。
- **Flutter 3.32.0 對話框主題更新：** 更新對話框主題實作，改用 `DialogThemeData`。

## 錯誤修正

- **傳輸視窗速度顯示錯誤：** 修正傳輸過程中速度顯示不準確的問題。
- **設定瀏覽按鈕無效：** 修正設定面板中瀏覽按鈕無回應的問題。
- **新增遠端供應商清單遺失：** 解決新增遠端時供應商選項遺失的問題。
- **遠端清單重新載入問題：** 修正遠端清單無法正確重新整理的問題。
- **S3 資料夾建立錯誤未回報：** 現在 S3 建立資料夾失敗時會顯示錯誤訊息。
- **AWS S3 資料夾複製失敗：** 修正在 AWS S3 遠端複製資料夾時無聲失敗的問題。
- **Windows 解除安裝／重新安裝錯誤：** 解決完全解除安裝後重新安裝發生異常行為的問題。
- **供應商載入旗標未重置：** 修正載入狀態未重置導致介面異常的問題。

## 依賴套件更新

- **Flutter SDK 相容性：** 全面更新以相容於 Flutter 3.32.0。
- **intl 套件更新：** 更新至 0.20.2 版本，改善在地化支援。
