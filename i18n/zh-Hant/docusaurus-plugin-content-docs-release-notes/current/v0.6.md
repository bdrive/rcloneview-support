# RcloneView 0.6 – 發行說明

**發行日期：** 2025 年 4 月<br/>
**支援平台：** Windows、macOS、Linux

我們很高興宣布 RcloneView 0.6 正式發布，本次更新為使用者體驗、系統匣功能、篩選能力以及整體穩定性帶來重大改進。此版本反映了根據使用者回饋與持續開發所累積的大量強化、錯誤修正與新功能。

## 新功能

- **啟動時自動掛載：** RcloneView 現在支援在應用程式啟動時自動掛載遠端。掛載設定會持續儲存並在內部資料庫中管理。
- **從系統匣快速掛載：** 現在可以直接從系統匣選單掛載遠端，簡化常用遠端的操作流程。
- **從系統匣選單開啟遠端：** 新增可從系統匣快速開啟遠端目錄的功能，提升便利性與可存取性。
- **篩選功能支援正規表示式：** 進階使用者現在可以在篩選檢視中使用正規表示式，充分發揮 rclone 原生篩選能力。
- **比較檢視強化 - 支援刪除：** 現在可以直接在比較檢視中刪除檔案，擴充應用程式內的管理能力。

## 使用者體驗與功能改進

- **macOS 系統匣圖示自適應：** 系統匣圖示會依 macOS 淺色與深色背景動態調整，提升可視性。
- **檔案總管強化：** 使用者現在可以在檔案總管中手動輸入檔案路徑以直接導覽至該位置。
- **在地化時間顯示：** 檔案時間戳記現在會依系統本地時區顯示。
- **歷史紀錄檢視介面修正：** 改善「開始時間」欄位的可視性，並修正本機圖示顯示的問題。

## 其他

- **移除廣告：** 應用程式現已完全無廣告，提供更簡潔、專業的使用體驗。
