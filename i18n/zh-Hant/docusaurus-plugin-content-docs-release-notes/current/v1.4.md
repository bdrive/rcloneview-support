# RcloneView 1.4 – 版本說明

**發布日期：** 2026 年 3 月<br/>
**支援平台：** Windows、macOS、Linux

RcloneView 1.4 導入 iCloud 照片整合、用於遠端存取的網頁前端、強化的批次工作（Batch Job）功能，並在所有平台上帶來眾多穩定性與易用性改進。

## 重點功能

- **iCloud 照片支援（Beta，唯讀）：**
  直接在 RcloneView 中瀏覽並下載您的 iCloud 照片圖庫內容。
  檢視相簿、依日期搜尋，並下載原始檔案——完全不需離開應用程式。
  此測試版功能僅支援唯讀存取，目前尚無法上傳、刪除或重新命名。

- **網頁前端（Beta）：**
  透過瀏覽器隨時隨地存取 RcloneView。內建的網頁伺服器提供功能完整的
  雙面板檔案總管以及即時傳輸監控。
  支援密碼保護存取，並可選擇是否允許遠端連線。

- **強化的批次工作：**
  批次工作已大幅升級，新增逐步執行、每步驟的試跑（Dry Run）、
  步驟間的變數相依性、風險等級指示、範本搜尋，以及完整的執行歷史紀錄追蹤。
  透過試跑預覽與清楚的風險可見性，安心建立複雜的多步驟工作流程。

## 新功能

- **篩選預設集儲存與載入：**
  儲存並載入篩選預設集，供不同工作與工作階段重複使用。

- **Windows 上的掛載選項：**
  Windows 平台現已支援掛載操作的檔案與目錄權限設定。

## 改進項目

- **macOS 12 支援：**
  恢復對 macOS 12（Monterey）的相容性。

- **Dropbox 隱藏檔排除：**
  新增選項，可在同步至 Dropbox 時自動排除隱藏檔（例如 .DS_Store）。

- **Linux 系統匣圖示相容性：**
  改善 Linux Cinnamon 及其他具原生系統匣支援桌面環境的系統匣處理方式。

- **Windows 系統匣圖示顏色：**
  Windows 上的系統匣圖示顏色現已可調整。

- **資料庫路徑處理：**
  改進資料庫路徑設定，並支援舊版路徑，讓升級過程更順暢。

- **移除 Proton Drive：**
  由於上游變更，Proton Drive 已從遠端提供者清單中移除。

## 錯誤修正

- **自訂旗標未套用：**
  修正工作執行期間自訂旗標（Flags）遭忽略的問題。

- **移動工作的試跑：**
  修正移動（Move）工作類型的試跑按鈕未啟用的問題。

- **終端機最大化狀態：**
  修正終端機面板最大化時的兩個問題，包括 UI 元件狀態錯誤。

- **資料庫初始化：**
  消除重複呼叫 sqfliteFfiInit() 所導致的間歇性「database closed」錯誤。

- **安全儲存裝置 ID：**
  修正安全儲存中的裝置 ID 處理問題，並解決 Synology 自動偵測停用狀態未被儲存的錯誤。

- **排程輸入延遲：**
  修正輸入排程參數時發生的 UI 延遲問題。
