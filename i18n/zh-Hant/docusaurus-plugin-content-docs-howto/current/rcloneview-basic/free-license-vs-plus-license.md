# RcloneView 授權類型

RcloneView 提供兩種授權類型，以支援不同使用需求：**免費版**與 **Plus 版**。
下表比較了各方案的可用功能與限制：

## 🔍 功能比較

| 功能                                | 免費版授權 | Plus 版授權 |
| ---------------------------------- | ------------ | ------------ |
| 掛載／卸載遠端                       | ✅            | ✅            |
| 檔案總管（瀏覽／複製／刪除）           | ✅            | ✅            |
| 資料夾比較與檔案管理                  | ✅            | ✅            |
| 具篩選功能的資料夾比較                | ❌            | ✅            |
| 檔案下載／上傳                       | ✅            | ✅            |
| 同步與工作管理                       | ✅            | ✅            |
| 1:N 同步                            | ✅            | ✅            |
| 排程式同步工作                       | ❌            | ✅            |
| 啟動時自動開始排程                    | ❌            | ✅            |
| 掛載遠端儲存空間                      | ✅            | ✅            |
| 啟動時自動掛載                        | ❌            | ✅            |
| 工作歷史與紀錄                       | ✅            | ✅            |
| 系統匣整合                          | ✅            | ✅            |
| 多個 RcloneView 視窗                | ❌            | ✅            |
| 需要授權金鑰                        | ❌            | ✅            |

## 如何啟用 Plus 授權

1. 造訪[定價頁面](https://rcloneview.com/src/pricing.html)並購買授權。
2. 在 RcloneView 中，前往 **Help > Activate License**。
3. 輸入您的**電子郵件**與提供的**授權金鑰**。

啟用後，進階功能將會自動解鎖。
