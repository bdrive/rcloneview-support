# AWS S3 與 S3 相容儲存

RcloneView 支援 **Amazon S3** 以及多種**相容 S3 協定的物件儲存供應商**，包括 Wasabi、Backblaze、Cloudflare R2 等主要廠商。

這些遠端共用相同的設定介面，皆基於 S3 協定。選擇後，您需要提供存取憑證，並可選擇性地設定自訂端點 URL。


## 如何新增相容 S3 的遠端

### 步驟 1：開啟新增遠端設定視窗

- 從上方選單的 **`Remote`** 底下點選 **`+ New Remote`**。
- 或者，在 Explorer 面板中點選 **`+`** 按鈕，然後選擇 **`New Remote`**。

### 步驟 2：設定 S3 供應商

#### 在**Provider**（供應商）分頁：
1. 搜尋 `s3` 或您的供應商名稱（例如 `aws`、`wasabi`、`minio`）。
2. 從結果清單中選擇 **`S3`**。

#### 在**Options**（選項）分頁：
3. 設定您的**供應商**（例如 `AWS`、`Wasabi`、`Minio` 等）。
4. 輸入您的[**Access Key ID**](/howto/cloud-storage-setting/aws-account-info#step-by-step-get-your-aws-access-key-id-and-secret-access-key)。
5. 輸入您的[**Secret Access Key**](/howto/cloud-storage-setting/aws-account-info#step-by-step-get-your-aws-access-key-id-and-secret-access-key)。
6. 為您的儲存空間 bucket 選擇正確的**地區**（例如 `ap-northeast-2`）。
   如有需要，可設定自訂**端點**（適用於不使用 AWS 端點格式的供應商）。

### 步驟 3：命名並儲存遠端

#### 在**Name**（名稱）分頁：
7. 為遠端指定名稱（例如 `myWasabi`、`minio-prod`）。

#### 在**Save**（儲存）分頁：
8. 點選 **Save** 完成設定。

1. 從 **Remote** 選單開啟 **Remote Manager**。
2. 確認新遠端已出現在清單中。
