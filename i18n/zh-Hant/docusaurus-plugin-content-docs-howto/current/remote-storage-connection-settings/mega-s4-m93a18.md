# Mega S4 白牌 Explorer（預覽版）

本頁面私人預覽了 RcloneView 如何以完全品牌化的 **Mega S4 Explorer** 桌面應用程式形式提供給您的使用者。

此白牌提案的目標為:

- 讓 **Mega S4 品牌在整個產品中處於最顯著的位置**。
- 讓使用者**在安裝後能立即輕鬆連接他們的 Mega S4 帳號**。
- 確保 **Mega S4 在每個管理與導覽流程中都是首選**。
- 為 Mega S4 品牌部署提供**專屬的優先支援與維護**。

---

## 1. 品牌套件與視覺整合

我們提供一套品牌套件,讓 Mega S4 在整個產品中作為主要且可見的品牌。所有 RcloneView 品牌標識都會被移除,或僅在法律要求的情況下保留(例如內部版本字串)。

主要元素:

- 應用程式名稱設定為 **「Mega S4 Explorer」**(或其他協議的名稱)。
- 所有 RcloneView 標誌皆替換為 **Mega S4 標誌**:
  - 桌面捷徑與工作列圖示。
  - 安裝程式圖示。
  - 應用程式內標題列與視窗圖示。
- 在適當情況下調整配色以符合 Mega S4 的品牌色調。

## 2. 安裝後的 Mega S4 遠端精靈

安裝完成後,系統會立即引導使用者連接他們的 Mega S4 帳號,讓他們無需額外設定步驟即可開始使用該服務。

主要行為:

- 在安裝精靈結束時,啟動應用程式會將 **「新增 Mega S4 遠端」** 開啟為預設的新手導覽流程。
- 該精靈已針對 Mega S4 進行簡化與預先設定:
  - 提供者類型會預先選取為 **Mega S4**。
  - 預設僅顯示 Mega S4 專屬選項。
  - 進階選項仍可透過 **「顯示進階選項」** 連結取得。

<img src="/support/images/en/howto/remote-storage-connection-settings/post-install-mega-remote-wizard.png" alt="post install mega remote wizard" class="img-large img-center" />


### 首頁快速存取精靈

如果使用者略過初次連接或關閉了精靈:

- 首頁右側面板會顯示一個**大型 Mega S4 圖磚**。
- 點擊該圖磚可隨時重新開啟 **「新增 Mega S4 遠端」** 精靈。

<img src="/support/images/en/howto/remote-storage-connection-settings/on-home-quick-access-wizard-for-mega.png" alt="on home quick access wizard for mega" class="img-large img-center" />

 
這確保了對新使用者或回訪使用者而言,連接 Mega S4 始終是最顯眼的下一步操作。

---

## 3. 以 Mega S4 為優先的導覽與管理

新增 Mega S4 遠端後,UI 會經過最佳化,讓 Mega S4 在所有主要導覽與管理畫面中皆保持顯眼可見。

### 3.1 重新啟動時固定 Mega S4 面板

在設定好 Mega S4 遠端之後:

- 在後續啟動時,UI 會預設以**固定的 Mega S4 面板**開啟。
- 典型版面配置:
  - 左側:本機磁碟或其他來源。
  - 右側:使用者的 **MYMegaS4** 遠端。
- 使用者可透過拖放自由重新排列面板,因此 Mega S4 面板可依照個人偏好的工作流程在左右兩側之間移動。
- 無論 Mega S4 面板目前位於哪一側,這都讓本機資料夾與 Mega S4 之間的重複同步或複製操作變成一鍵即可完成。

<img src="/support/images/en/howto/remote-storage-connection-settings/mega-panel-pinned-on-re-launch.png" alt="mega panel pinned on re launch" class="img-large img-center" />

### 3.2 在「新增遠端」與「遠端管理員」中優先顯示 Mega S4

為了凸顯 Mega S4 作為主要儲存提供者的地位:

- 在**新增遠端**對話框中:
  - Mega S4 會出現在**提供者清單的最上方**。
  - Mega S4 圖磚會以視覺方式突出顯示,以鼓勵使用者選取。
- 在**遠端管理員**中:
  - Mega S4 遠端(例如 `MYMegaS4`)會顯示在**遠端清單的最上方**。
  - 在清單檢視與卡片檢視中,Mega S4 遠端皆會以視覺方式強調,以便更快速地存取。

<img src="/support/images/en/howto/remote-storage-connection-settings/maga-first-in-management-dialog.png" alt="maga first in management dialog" class="img-large img-center" />

## 4. Mega S4 的優先支援與維護

針對 Mega S4 白牌部署,我們提供專屬的支援與維護方案。

包含的服務:

- **專屬文件**  
  - 專門為 **Mega S4 Explorer** 使用者準備的獨立說明頁面。  
  - 針對 Mega S4 連接、同步與疑難排解的逐步指南。

- **優先事件處理**  
  - Mega S4 使用者的問題會在支援佇列中優先處理。  
  - 針對影響 Mega S4 使用者的重大事件(例如連線失敗、資料存取問題)提供**緊急應變**。

- **持續的產品更新**  
  - 維護合約中包含定期的桌面客戶端升級。  
  - 在雙方共同核准後,可在 Mega S4 品牌下推出新的 RcloneView 功能。
