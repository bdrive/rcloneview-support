# RcloneView 0.7 – Catatan Rilis

**Tanggal Rilis:** Juni 2025<br/>
**Platform:** Windows, macOS, Linux

RcloneView 0.7 menghadirkan berbagai peningkatan, perbaikan bug penting, dan fitur baru yang dirancang untuk meningkatkan keandalan, kemudahan penggunaan, dan kompatibilitas cloud. Versi ini juga mencakup pembaruan dependensi inti untuk performa yang lebih baik dan kompatibilitas di masa depan.

## Unduhan

- **Windows:** [x86_64](https://downloads.bdrive.com/rclone_view/builds/setup_rclone_view-0.7.323.exe)
- **macOS:** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-0.7.323.dmg)
- **Linux:** [AppImage x86_64](https://downloads.bdrive.com/rclone_view/linux/0.7/RcloneView-0.7.310-x86_64.AppImage) · [AppImage arm64](https://downloads.bdrive.com/rclone_view/linux/0.7/RcloneView-0.7.310-aarch64.AppImage) · [Deb x86_64](https://downloads.bdrive.com/rclone_view/linux/0.7/rclone_view-0.7.310+1-linux-x86_64.deb) · [Deb arm64](https://downloads.bdrive.com/rclone_view/linux/0.7/rclone_view-0.7.310+1-linux-aarch64.deb) · [RPM x86_64](https://downloads.bdrive.com/rclone_view/linux/0.7/rclone_view-0.7.310+1-linux-x86_64.rpm) · [RPM arm64](https://downloads.bdrive.com/rclone_view/linux/0.7/rclone_view-0.7.310+1-linux-aarch64.rpm)

## Fitur Baru

- **Pengambilan Tautan Berbagi Publik:** Buat dan salin tautan berbagi publik untuk remote yang didukung langsung dari antarmuka.
- **Dedupe untuk Google Drive:** Selesaikan nama file duplikat di Google Drive menggunakan fitur `dedupe` dari rclone.
- **Opsi Abaikan Sertifikat SSL:** Opsi untuk mengabaikan verifikasi SSL saat mendaftarkan remote baru.
- **Penyimpanan Aman untuk Kata Sandi Konfigurasi Rclone:** Kata sandi kini disimpan secara aman menggunakan API native platform.
- **Notifikasi Penyelesaian Sinkronisasi:** Menambahkan notifikasi dengan opsi "View Details" saat sinkronisasi selesai.
- **Peringatan Nama Remote Tidak Valid:** Pengguna diberi tahu jika nama remote mengandung karakter yang tidak valid.
- **Penanganan Remote yang Dihapus:** Tab yang tertaut ke remote yang telah dihapus kini ditangani dengan baik disertai pesan yang sesuai.
- **Compare View – Pencarian Folder Berdasarkan Tampilan:** Pencarian folder kini mengikuti pengaturan filter tampilan yang sedang aktif.

## Peningkatan

- **Perbaikan UI Mode Gelap:** Memperbaiki masalah visibilitas teks hitam pada dialog Mount saat menggunakan mode gelap.
- **Pembaruan Tema Dialog untuk Flutter 3.32.0:** Memperbarui implementasi tema dialog agar menggunakan `DialogThemeData`.

## Perbaikan Bug

- **Bug Tampilan Kecepatan pada Jendela Transfer:** Memperbaiki tampilan kecepatan yang tidak akurat selama transfer.
- **Tombol Browse pada Pengaturan Tidak Berfungsi:** Memperbaiki tombol browse yang tidak responsif pada panel pengaturan.
- **Daftar Provider Remote Baru Hilang:** Menyelesaikan masalah opsi provider yang hilang saat menambahkan remote baru.
- **Masalah Reload Daftar Remote:** Memperbaiki masalah di mana daftar remote gagal dimuat ulang dengan benar.
- **Kesalahan Pembuatan Folder di S3 Tidak Dilaporkan:** Pesan kesalahan kini ditampilkan saat pembuatan folder gagal di S3.
- **Kegagalan Penyalinan Folder AWS S3:** Memperbaiki kesalahan diam-diam saat menyalin folder pada remote AWS S3.
- **Kesalahan Uninstall/Reinstall di Windows:** Menyelesaikan perilaku tidak normal setelah uninstall bersih dan pemasangan ulang.
- **Flag Pemuatan Provider Tidak Direset:** Memperbaiki bug di mana status pemuatan tidak direset, sehingga menyebabkan masalah UI.

## Pembaruan Dependensi

- **Kompatibilitas Flutter SDK:** Sepenuhnya diperbarui agar kompatibel dengan Flutter 3.32.0.
- **Paket intl Diperbarui:** Diperbarui ke versi 0.20.2 untuk dukungan lokalisasi yang lebih baik.
