# RcloneView 0.8 – Catatan Rilis

**Tanggal Rilis:** Juli 2025<br/>
**Platform:** Windows, macOS, Linux

RcloneView 0.8 adalah pembaruan besar yang berfokus pada stabilitas performa, kemudahan penggunaan job sinkronisasi, penanganan error, dan dukungan penyedia cloud yang lebih luas. Rilis ini juga mencakup peningkatan penting pada perilaku antarmuka pengguna dan kompatibilitas.

- **macOS:** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-0.8.368.dmg)

## Fitur Baru & Peningkatan

- **Panel Transfer & Selesai Terpadu:** Digabungkan menjadi satu tampilan untuk pelacakan job yang lebih efisien.
- **Peningkatan Dialog Mount:** Menambahkan tombol `Reload` dan menyesuaikan tinggi dialog agar lebih mudah digunakan.
- **Drag Tab Remote:** Seret tab remote antar panel untuk pengelolaan UI yang lebih fleksibel.
- **Peningkatan Penanganan Lisensi:** Pelaporan error dan opsi transfer lisensi yang lebih baik saat registrasi gagal atau akses perangkat hilang.
- **Dukungan Dropbox Business:** Memperbaiki masalah visibilitas alias dan meningkatkan penanganan error unggah.
- **Alur Sync-to-Job:** Secara otomatis membuka Job Manager setelah menyimpan tugas sinkronisasi.
- **Pencatatan Riwayat Penjadwalan:** Log error untuk job terjadwal yang gagal kini muncul di Riwayat.
- **Nonaktifkan Popup Penyelesaian Sinkronisasi:** Opsi baru untuk mematikan notifikasi penyelesaian sinkronisasi.

## Peningkatan UI & UX

- **Perbaikan Klik Ganda Panel Explorer:** Memperbaiki remote yang tidak terbuka saat job diklik dua kali.
- **Ikon & Nama Job Rata Kiri:** Meningkatkan konsistensi tata letak.
- **Perbaikan Dialog About di Mode Gelap:** Mengatasi masalah visibilitas pada mode gelap.
- **Pembaruan Perilaku Tap:** Mencegah seluruh teks terpilih saat mengetuk kolom.
- **Kolom Rename Terisi Otomatis:** Menampilkan nama file saat ini selama proses rename.
- **Penataan Menu Utama:** Meningkatkan kontras untuk ikon dan teks.
- **Ikon Uninstall Kustom:** Menambahkan ikon uninstall bermerek.
- **Pembersihan File Log saat Startup:** Aplikasi kini menghapus file log sebelumnya saat memulai, memastikan log lebih bersih untuk setiap sesi.

## Perbaikan Bug

- **Penghambat Upgrade DB:** Memperbaiki crash saat startup yang terkait dengan kegagalan upgrade database.
- **Perbaikan Tampilan Compare:** Memperbaiki tampilan jumlah file dan bug pemilihan sumber/tujuan.
- **Masalah Import/Export Job:** Meningkatkan UX dan mencegah nama job duplikat saat import.
- **Konflik Drive Mount:** Mencegah duplikasi huruf drive saat mount.
- **Jalur Eksekusi Rclone:** Kini selalu berjalan dari direktori root sistem.
- **Trailing Slash pada Penghapusan Folder:** Memastikan penghapusan direktori yang akurat.
- **Penanganan Kehilangan Jaringan:** Pesan popup kini ditampilkan saat terjadi pemutusan jaringan.
- **Refresh Remote & Folder di Windows:** Memperbaiki remote yang tidak muncul setelah ditambahkan via CLI dan folder yang tidak me-refresh setelah penghapusan file.
- **Pencatatan Log Hapus Multi-File:** Semua file yang dihapus kini tercatat dalam daftar selesai.
- **Perbaikan Freeze Dry Run:** Mengatasi aplikasi yang tidak responsif saat menampilkan banyak hasil file.
- **Dukungan Ikon Tray KDE Plasma:** Memperbaiki masalah tampilan ikon tray saat berjalan pada lingkungan desktop KDE Plasma.

## Lokalisasi & Kompatibilitas

- **Lokalisasi Bahasa Jepang:** Menambahkan file lokalisasi bahasa Jepang yang sebelumnya hilang.
- **Perbaikan Terjemahan Bahasa Jerman:** Mengatasi error kompilasi akibat string terjemahan yang tidak tepat.
