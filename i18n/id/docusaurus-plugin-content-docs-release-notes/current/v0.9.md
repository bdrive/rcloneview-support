# RcloneView 0.9 – Catatan Rilis

**Tanggal Rilis:** September 2025<br/>
**Platform:** Windows, macOS, Linux

RcloneView 0.9 menghadirkan peningkatan besar dalam manajemen remote, integrasi Synology NAS, kustomisasi tema, dan penanganan file. Rilis ini mencakup banyak perbaikan bug, peningkatan UX, dan pembaruan kompatibilitas untuk lingkungan Flutter dan Linux.

## Unduh

- **Windows:** [x86_64](https://downloads.bdrive.com/rclone_view/builds/setup_rclone_view-0.9.471.exe)
- **macOS:** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-0.9.467.dmg)
- **Linux:** [AppImage x86_64](https://downloads.bdrive.com/rclone_view/linux/0.9/RcloneView-0.9.431-x86_64.AppImage) · [AppImage arm64](https://downloads.bdrive.com/rclone_view/linux/0.9/RcloneView-0.9.431-aarch64.AppImage) · [Deb x86_64](https://downloads.bdrive.com/rclone_view/linux/0.9/rclone_view-0.9.431+1-linux-x86_64.deb) · [Deb arm64](https://downloads.bdrive.com/rclone_view/linux/0.9/rclone_view-0.9.431+1-linux-aarch64.deb) · [RPM x86_64](https://downloads.bdrive.com/rclone_view/linux/0.9/rclone_view-0.9.431+1-linux-x86_64.rpm) · [RPM arm64](https://downloads.bdrive.com/rclone_view/linux/0.9/rclone_view-0.9.431+1-linux-aarch64.rpm)

## Fitur Baru

- **Deteksi dan Penyiapan Otomatis Synology NAS:** Secara otomatis mendeteksi dan menambahkan perangkat Synology NAS dengan pengaturan default.
- **Penambahan Otomatis Cloud Drive:** Menambahkan otomatis drive tertaut seperti iCloud dan Google Drive.
- **Peluncuran Minimize ke Tray:** Aplikasi dimulai dalam keadaan minimize ke system tray agar desktop tidak berantakan.
- **Tampilkan Dialog Synology NAS saat Startup:** Dialog muncul saat Synology NAS terdeteksi.

## Peningkatan UI & UX

- **Perbaikan Warna Tema:** Memperbaiki penerapan tema checkbox yang hilang di pengaturan.
- **Desain Ulang Pengaturan Umum:** Tata letak disempurnakan agar navigasi lebih mudah dan jelas.
- **UX Konfigurasi Remote:** Pengalaman penyiapan remote yang lebih intuitif dengan dialog dan penanganan error yang lebih baik.
- **Tampilkan/Sembunyikan Kata Sandi:** Toggle visibilitas pada kolom kata sandi konfigurasi rclone.
- **Migrasi File Selector:** Mengganti `file_picker` dengan `file_selector` untuk kompatibilitas yang lebih baik.
- **Validasi Opsi Global:** Mendeteksi dan memberi peringatan untuk opsi rclone yang tidak valid atau bertentangan.
- **Fallback Lokalisasi Aplikasi:** Menangani file lokalisasi yang hilang dalam mode rclone tertanam (embedded).
- **Mencegah Duplikasi Filter Kustom:** Menghindari filter duplikat saat mengedit job.
- **Tautan Dukungan Ditambahkan:** Bagian Bantuan kini menautkan ke [rcloneview.com/support](https://rcloneview.com/support).
- **Nonaktifkan Aksi File Selama Perbandingan:** Tombol dinonaktifkan selama proses perbandingan berjalan.
- **Perbaikan Ikon Tray Linux:** Ikon tray kini terlihat di berbagai tema dan desktop environment Linux.

## Keamanan & Pengaturan

- **Enkripsi Konfigurasi via Variabel Lingkungan:** Kunci enkripsi konfigurasi rclone kini dapat diatur menggunakan variabel lingkungan.
- **Pencegahan Tombol ESC:** Tombol ESC tidak lagi menutup dialog secara tidak sengaja (misalnya input kata sandi).
- **Peningkatan Uninstaller (Windows):** Menghentikan proses aplikasi dan menghapus file dengan benar selama uninstall.
- **Persistensi Lisensi macOS:** Lisensi kini tersimpan dengan benar setelah restart.

## Peningkatan Job & Transfer

- **Optimasi Penghapusan Massal:** Penghapusan file besar kini berjalan lebih efisien dalam satu permintaan.
- **Tutup Tab Otomatis saat Penghapusan Remote:** Menghapus remote juga menutup tab terkait yang terbuka.
- **Perilaku Klik Ganda Job:** Kini membuka remote terkait dengan benar di tampilan Explorer.

## Perbaikan Bug

- **[AWS S3] Penghapusan Folder yang Salah:** Memperbaiki masalah di mana menghapus satu folder juga menghapus folder yang tidak terkait.
- **[macOS] Jendela CLI Tidak Muncul:** Memperbaiki jendela terminal yang tidak terbuka saat menambahkan remote baru.
- **Opsi Global Tidak Valid:** Aplikasi kini dapat pulih dari kegagalan startup akibat flag rclone yang tidak valid.
- **Peringatan "Too Many Open Files":** Memperingatkan pengguna saat batas file handle sistem terlampaui.
- **Jumlah File Tidak Akurat:** Jumlah file transfer kini ditampilkan dengan benar.
- **Bug Seleksi Touchpad (macOS):** Mencegah scrolling dua jari memilih file secara tidak sengaja.
- **Peningkatan Drag-to-Select:** Pemilihan file yang lebih akurat menggunakan drag mouse.
- **Persistensi Bahasa:** Aplikasi kini diluncurkan dalam bahasa yang tersimpan dengan benar.
- **Pembekuan saat Pembaruan rclone Tertanam:** Memperbaiki pembekuan (freeze) saat pembaruan biner rclone dalam mode embedded.

## Teknis & Build

- **Upgrade Flutter 3.25:** Ditingkatkan ke versi Flutter terbaru untuk performa dan kompatibilitas yang lebih baik.
- **Perbaikan Entitlement macOS:** Menambahkan izin yang hilang yang diperlukan untuk akses file dan dialog sistem.
- **Kode Iklan Dihapus:** Membersihkan semua paket dan skrip terkait iklan untuk pengalaman bebas gangguan.
