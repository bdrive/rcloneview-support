# RcloneView 0.4 – Catatan Rilis

Kami dengan bangga mengumumkan rilis RcloneView v0.4, yang mencakup Penjadwalan Sinkronisasi, fitur Perbandingan Folder yang lebih baik, dukungan Linux yang ditingkatkan, serta berbagai optimasi UI dan performa. Pembaruan ini juga memperbaiki berbagai bug untuk meningkatkan stabilitas dan kegunaan.

- **macOS:** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-0.4.219.dmg)
- **Linux:** [AppImage x86_64](https://downloads.bdrive.com/rclone_view/linux/0.4/RcloneView-0.4.190-x86_64.AppImage) · [AppImage arm64](https://downloads.bdrive.com/rclone_view/linux/0.4/RcloneView-0.4.190-aarch64.AppImage) · [Deb x86_64](https://downloads.bdrive.com/rclone_view/linux/0.4/rclone_view-0.4.190+1-linux-x86_64.deb) · [Deb arm64](https://downloads.bdrive.com/rclone_view/linux/0.4/rclone_view-0.4.190+1-linux-aarch64.deb) · [RPM x86_64](https://downloads.bdrive.com/rclone_view/linux/0.4/rclone_view-0.4.190+1-linux-x86_64.rpm) · [RPM arm64](https://downloads.bdrive.com/rclone_view/linux/0.4/rclone_view-0.4.190+1-linux-aarch64.rpm)

## Fitur Baru & Peningkatan

- **Penjadwalan Sinkronisasi:** Kini Anda dapat menjadwalkan pekerjaan agar berjalan secara berkala. Perlu diperhatikan bahwa RcloneView harus tetap berjalan agar penjadwalan dapat berfungsi. Aplikasi ini juga mendukung eksekusi di latar belakang dengan integrasi system tray.
- **Perbandingan Folder:** Kami telah meningkatkan secara signifikan perbandingan folder untuk kecepatan dan stabilitas yang lebih baik. Kini Anda dapat membandingkan folder secara visual dan menganalisis perbedaan di dalam subfolder secara rekursif, sehingga manajemen file menjadi lebih intuitif dan efisien.
- **Peningkatan Manajemen Remote:** Kolom `client_id` dan `client_secret` telah dipindahkan ke opsi lanjutan.
- **Dukungan Linux yang Ditingkatkan:** Memperbaiki masalah kompatibilitas dengan distribusi Linux terbaru.

## Perbaikan Bug

- Memperbaiki masalah pada kolom teks file log di Global Settings.
- Meningkatkan penanganan error di seluruh aplikasi.
