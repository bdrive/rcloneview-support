# RcloneView 0.6 – Catatan Rilis

**Tanggal Rilis:** April 2025<br/>
**Platform:** Windows, macOS, Linux

Kami dengan bangga mengumumkan rilis RcloneView 0.6, yang menghadirkan peningkatan signifikan pada pengalaman pengguna, fungsi tray, kemampuan filtering, dan stabilitas secara keseluruhan. Versi ini mencerminkan berbagai peningkatan, perbaikan bug, dan fitur baru berdasarkan masukan pengguna dan upaya pengembangan yang berkelanjutan.

## Fitur Baru

- **Auto-Mount saat Startup:** RcloneView kini mendukung mounting remote otomatis saat aplikasi diluncurkan. Pengaturan mount disimpan secara persisten dan dikelola dalam database internal.
- **Quick Mount dari Tray:** Anda kini dapat melakukan mount remote langsung dari menu tray, mempercepat proses untuk remote yang sering diakses.
- **Buka Remote dari Menu Tray:** Ditambahkan fungsi untuk membuka direktori remote secara cepat dari tray, meningkatkan aksesibilitas dan kenyamanan.
- **Dukungan Regex untuk Filtering:** Pengguna tingkat lanjut kini dapat menggunakan regular expression pada tampilan filter, memanfaatkan sepenuhnya kemampuan filter native rclone.
- **Peningkatan Compare View - Dukungan Hapus:** File kini dapat dihapus langsung dari tampilan Compare, memperluas kemampuan manajemen dalam aplikasi.

## Peningkatan UX & Fungsional

- **Adaptasi Ikon Tray untuk macOS:** Ikon tray kini menyesuaikan secara dinamis terhadap latar belakang macOS terang dan gelap untuk visibilitas yang lebih baik.
- **Peningkatan Explorer:** Pengguna kini dapat memasukkan path file secara manual di explorer untuk langsung menuju lokasi tersebut.
- **Tampilan Waktu Terlokalisasi:** Timestamp file kini ditampilkan sesuai dengan zona waktu sistem lokal.
- **Perbaikan UI Tampilan History:** Peningkatan visibilitas kolom Start Time dan perbaikan masalah pada rendering ikon lokal.

## Lain-lain

- **Iklan Dihapus:** Aplikasi kini sepenuhnya bebas iklan, menawarkan pengalaman yang lebih bersih dan profesional.
