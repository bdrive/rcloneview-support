# RcloneView 0.2 – Catatan Rilis

Kami dengan senang hati merilis RcloneView v0.2, yang menghadirkan dukungan multi-window, sinkronisasi yang ditingkatkan, pembaruan paket Linux, dan UI yang disempurnakan untuk performa dan kegunaan yang lebih baik. Pembaruan ini juga mencakup perbaikan bug penting untuk memastikan pengalaman yang lebih lancar.

## Unduh

- **Windows:** [x86_64](https://downloads.bdrive.com/rclone_view/builds/setup_rclone_view-0.2.158.exe)
- **macOS:** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-0.2.158.dmg)
- **Linux:** [AppImage x86_64](https://downloads.bdrive.com/rclone_view/linux/RcloneView-x86_64.AppImage) · [AppImage arm64](https://downloads.bdrive.com/rclone_view/linux/RcloneView-aarch64.AppImage)

## Fitur & Peningkatan Baru

- **Dukungan Distribusi Linux:** Memperkenalkan dukungan paket AppImage Linux, beserta manajemen jendela dan operasi multi-window yang lebih baik di Linux. Perlu diketahui bahwa instalasi FUSE diperlukan untuk menggunakan AppImage di Linux. Paket RPM dan DEB akan menyusul segera.
- **Dukungan Multi-Window:** Memperkenalkan manajemen banyak jendela di berbagai platform, termasuk Linux, macOS, dan Windows.
- **Sinkronisasi 1:N:** Menambahkan fungsi sinkronisasi multi-job (1:N) dan menyempurnakan UX-nya.
- **Ikon Provider:** Menambahkan ikon provider pada dialog remote baru dan daftar remote untuk kejelasan visual yang lebih baik.
- **Pembaruan Rclone Tersemat:** Mengimplementasikan kemampuan pembaruan Rclone internal/tersemat di Windows dan macOS, dengan proses shutdown yang tepat saat aplikasi keluar.
- **Peningkatan UI & Aksesibilitas:** Termasuk tooltip yang lebih baik serta dialog popup pengaturan dan konfigurasi yang dapat dipindahkan.
- **Peningkatan dialog job dan mount:** Mencegah freeze saat menambahkan job tanpa remote.
- **Peningkatan tampilan kecepatan transmisi dan perhitungan waktu tersisa.**
- **Diperbarui ke Rclone 1.68.1:** Memastikan penanganan yang tepat untuk remote baru.

## Perbaikan Bug

- Memperbaiki ketidaksesuaian dalam perhitungan waktu riwayat job.
- Mengatasi masalah pemotongan teks pada menu bar.
- Memperbaiki masalah ukuran font pada dialog pesan.
- Menghilangkan freeze dan meningkatkan penanganan error di seluruh aplikasi.
