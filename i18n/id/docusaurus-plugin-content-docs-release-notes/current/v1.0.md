# RcloneView 1.0 – Catatan Rilis

**Tanggal Rilis:** Desember 2025<br/>
**Platform:** Windows, macOS, Linux (termasuk Raspberry Pi OS)

RcloneView 1.0 menandai tonggak besar dengan stabilitas yang lebih baik, dukungan platform yang lebih luas, manajemen remote yang lebih baik, dan antarmuka yang lebih bersih dan informatif. Rilis ini mencakup fitur baru, peningkatan UX, dan perbaikan bug penting berdasarkan masukan pengguna dan penggunaan di dunia nyata.

## Fitur Baru

- **Dukungan Jenis Job Copy:** Menambahkan jenis job "Copy" baru di Job Manager, sehingga lebih mudah mengonfigurasi transfer file satu arah antar remote.
- **Deteksi Otomatis Network Drive (Windows):** RcloneView kini secara otomatis mendeteksi dan menampilkan mapped network drive (misalnya, SMB share) tanpa perlu interaksi pengguna. Drive ini muncul langsung di tab **Local**, sehingga siap digunakan untuk sinkronisasi, perbandingan, dan mount.
- **Get Size di Menu Konteks:** Opsi "Get Size" baru kini tersedia di menu konteks untuk menghitung ukuran folder yang dipilih dengan cepat.
- **Panel Fitur Mendatang:** Panel baru di jendela log bagian bawah menampilkan daftar fitur yang direncanakan dan akan datang, membantu pengguna tetap mendapat informasi tentang pembaruan mendatang.

## Peningkatan UI & UX

- **Desain Ulang Jendela Transfer:** Merombak total UI job Transfer dengan tata letak yang lebih bersih, menampilkan hingga 100 entri untuk performa dan keterbacaan yang lebih baik.
- **Penyempurnaan Tampilan Status Job:** Meningkatkan indikator status job agar pemantauan tugas yang aktif dan selesai lebih akurat dan intuitif.
- **Perilaku Explorer Tree:** Tampilan tree kini otomatis memperluas folder root saat pertama kali dijalankan untuk navigasi yang lebih mudah. Juga memperbaiki masalah overlay nama folder di panel layout.
- **Peningkatan UI Mount (Windows/macOS):** Memperbaiki masalah tata letak saat memilih folder lokal di Windows, dan menambahkan refresh otomatis tab Local Disk saat mount/unmount. Juga memperbaiki perilaku pembaruan mount path dan masalah `cmount` multi-remote di macOS.
- **Perbaikan Pemilihan & Tampilan Remote:** Meningkatkan dialog pemilihan remote agar menampilkan cloud drive dengan benar selama operasi Sync dan Compare.

## Perbaikan Bug

- **Perbaikan Deteksi Otomatis Synology NAS:** Memperbaiki deteksi yang tidak akurat dan memastikan alamat default tersimpan saat menambahkan remote NAS.
- **Penetapan Ulang Remote Default saat Restart:** Memperbaiki masalah di mana me-restart aplikasi mereset koneksi remote default setelah menggunakan jendela tambahan.
- **Masalah Tampilan Tab Jobs:** Memperbaiki informasi file dan kecepatan yang hilang di tab Jobs.
- **Perbaikan Perilaku AWS S3:** Menambahkan `--no-check-bucket` sebagai default untuk mencegah error sinkronisasi.
- **Kompatibilitas Raspberry Pi OS 12:** RcloneView kini dapat dijalankan dengan baik di Raspberry Pi OS 12.
