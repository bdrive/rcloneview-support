# RcloneView 1.3 – Catatan Rilis

**Tanggal Rilis:** Februari 2026<br/>
**Platform:** Windows, macOS, Linux

RcloneView 1.3 menambahkan fitur manajemen job baru dan memperluas integrasi dengan alat komunikasi populer seperti Slack dan Discord. Rilis ini juga menghadirkan penyempurnaan pada notifikasi email, perilaku UI, dan pelacakan eksekusi job.

- **macOS:** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-1.3.554.dmg)

## Fitur Baru

- **Dukungan Batch Job (Beta):**
  Anda kini dapat mengelompokkan beberapa job ke dalam satu batch dan menjalankannya bersamaan.
  Fitur beta ini memungkinkan Anda menentukan urutan job yang dijalankan secara berurutan, sehingga lebih mudah mengelola tugas-tugas yang saling berkaitan.
  Kontrol tambahan dan penanganan error direncanakan pada pembaruan mendatang.

- **Penambahan Tipe Job Move:**
  Tipe job "Move" baru kini tersedia, memungkinkan Anda mentransfer file dengan memindahkannya dari sumber ke tujuan.

- **Coba Ulang Job yang Gagal:**
  Anda kini dapat mencoba ulang job yang gagal tanpa perlu membuatnya kembali secara manual.

- **Integrasi Slack:**
  RcloneView kini mendukung notifikasi Slack dan kontrol job dari channel Slack.

- **Integrasi Discord:**
  Anda kini dapat menerima notifikasi job melalui Discord.

- **Penempatan Panduan Setup Remote Control:**
  Tautan panduan setup remote-control untuk Telegram dan Slack kini lebih mudah ditemukan di bagian atas bagian integrasi.

- **Dukungan Bahasa Indonesia:**
  Antarmuka kini tersedia dalam Bahasa Indonesia.

## Peningkatan

- **Penyempurnaan Notifikasi Email:**
  - Tata letak dan kejelasan panel pengaturan email ditingkatkan
  - Informasi detail kini disertakan dalam email status job
  - Memperbaiki crash UI langka pada layar konfigurasi email
  - Notifikasi email kini tetap dapat diakses setelah masa uji coba gratis berakhir

- **Ikon Remote Diperbarui:** Menambahkan ikon baru dan memperbarui ikon lama untuk memudahkan identifikasi remote.

- **Stabilitas Pemuatan Daftar File:** Mencegah kasus di mana berpindah halaman saat pemuatan file lambat berlangsung dapat menampilkan konten yang sudah usang.

- **Penanganan Hasil "No Changes":** Memastikan status "No Changes" tetap dipertahankan pada tampilan detail terlepas dari status penyelesaian job.

## Perbaikan Bug

- **Penghapusan File OneDrive:** Memperbaiki masalah yang mencegah penghapusan file pada remote OneDrive.
- **Bug Tampilan Status Transfer:** Memperbaiki masalah di mana file tampak masih dalam proses transfer meskipun job telah selesai.
- **Compare with Hash pada Disk Lokal:** Memperbaiki kondisi hang saat membandingkan file menggunakan pemeriksaan hash pada penyimpanan lokal.
- **Ketidakakuratan Riwayat Job:** Job yang dihentikan paksa kini dihapus dengan benar dari status "running" di History.
- **Ketidaksejajaran Tooltip Batal Transfer:** Memperbaiki posisi tooltip pada tampilan Transfer.
- **Pendaftaran Ulang Remote dengan Nama Sama:** Mencegah penggunaan ulang daftar file usang saat remote ditambahkan dengan nama yang sama seperti sebelumnya.
- **Pemotongan Kolom Dry Run:** Kolom terakhir pada panel detail Dry Run kini otomatis melebar untuk menyesuaikan isinya.
- **Error Token External rclone + Google Drive:** Memperbaiki masalah koneksi saat menggunakan external rclone dengan token Google Drive mode headless.
