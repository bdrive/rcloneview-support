# RcloneView 1.1 – Catatan Rilis

**Tanggal Rilis:** November 2025<br/>
**Platform:** Windows, macOS, Linux

RcloneView 1.1 menghadirkan fitur baru yang praktis dan peningkatan kegunaan pada penjelajahan remote, akses terminal, manajemen pengaturan, dan perilaku antarmuka. Pembaruan ini berfokus pada peningkatan efisiensi alur kerja dan mengatasi masalah utama yang dilaporkan pengguna.

- **macOS:** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-1.1.533.dmg)

## Fitur Baru

- **Tab Terminal Bawaan:**
  Tab Terminal yang benar-benar baru kini tersedia langsung di dalam RcloneView. Anda tidak perlu lagi beralih ke terminal eksternal atau command prompt — jalankan perintah shell, perintah rclone, dan diagnostik cepat langsung dari aplikasi.

- **App Lock (Perlindungan Kata Sandi):**
  RcloneView kini dapat dilindungi dengan kata sandi tingkat aplikasi untuk mencegah akses yang tidak sah.

- **Dukungan Lokasi Database Kustom:**
  Anda kini dapat memilih jalur penyimpanan khusus untuk file `rclone_view.db`, memungkinkan pengaturan ruang kerja yang fleksibel, penggunaan drive eksternal, atau sinkronisasi selektif.

- **Remote Manager: Tampilan Ukuran Item di Mode List:**
  Mode list pada Remote Manager kini menampilkan informasi ukuran yang detail untuk setiap item.

- **Perintah "Copy Full Path (with Remote)":**
  Menambahkan cara praktis untuk menyalin jalur remote lengkap — termasuk nama remote — ke clipboard.

- **Pratinjau Struktur Folder Virtual Remote:**
  Saat menambahkan atau mengedit virtual remote, pratinjau struktur folder kini ditampilkan untuk membantu pengguna memilih jalur dengan mudah.

## Peningkatan UI & UX

- **Pengubahan Ukuran Tab & Jendela yang Responsif:** Tab secara dinamis menyesuaikan ukurannya saat jendela aplikasi dipersempit, mencegah teks terpotong.
- **Perbaikan Penjelajahan Folder Cloud:** iCloud Drive dan folder cloud serupa kini muncul dengan benar di file browser.
- **Perbaikan Panel Pengaturan:** Jalur eksekutabel rclone lokal tidak lagi hilang secara tiba-tiba.
- **Kolom Nama Folder untuk Combine Remote:** Remote bertipe combine kini menggunakan kolom teks khusus untuk memasukkan nama folder demi kejelasan yang lebih baik.
- **Jendela Transfer – Perbaikan Dark Mode:** Mengatasi ketidakkonsistenan warna saat berpindah tema.
- **Peningkatan Panel History:**
  - Pesan error tidak lagi terpotong.
  - Error pada job terjadwal kini muncul dengan benar di History.
- **Peningkatan Tata Letak Explorer:** Lebar kolom ditingkatkan di seluruh tampilan Explorer untuk keterbacaan dan perataan yang lebih baik.
- **Penanganan yang Lebih Baik untuk Jenis Remote yang Tidak Didukung:** Dialog About untuk jenis remote dengan API yang tidak didukung kini menampilkan pesan fallback yang sesuai.

## Perbaikan Bug

- **Masalah Refresh UI pada Operasi Compare:** Memperbaiki bug di mana UI terus melakukan refresh meskipun tugas penyalinan berbasis Compare sudah selesai.
- **Perbaikan Browser Sync/Compare:** Mengatasi masalah di mana penjelajahan folder remote gagal saat node disk lokal sudah terpilih sebelumnya.
- **Penanganan Jalur Terminal di Windows:** Memperbaiki masalah di Windows di mana konfigurasi rclone tidak terbaca dengan benar di dalam lingkungan terminal.
- **Error Validasi Nama Job:** Memperbaiki exception database yang muncul saat pemeriksaan keunikan nama job.
- **Perbaikan Label Sukses "No Changes":** Tampilan detail file kini menampilkan "Success" dengan benar saat tidak ada perubahan yang terdeteksi selama sinkronisasi/compare.
- **Aturan Akurasi Penyelesaian 99%:** Ketika hasil hampir — tetapi tidak sepenuhnya — identik, indikator progres kini menampilkan 99% alih-alih 100% yang salah.
