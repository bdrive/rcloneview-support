# RcloneView 0.3 – Catatan Rilis

Kami dengan bangga merilis RcloneView v0.3, menghadirkan Perbandingan Folder, dukungan Linux yang lebih baik, serta berbagai peningkatan UI dan performa. Pembaruan ini juga mencakup berbagai perbaikan bug untuk meningkatkan stabilitas dan kemudahan penggunaan.

- **macOS:** [Universal](https://downloads.bdrive.com/rclone_view/builds/RcloneView-0.3.190.dmg)
- **Linux:** [AppImage x86_64](https://downloads.bdrive.com/rclone_view/linux/x86-64/RcloneView-0.3.170-x86_64.AppImage) · [AppImage arm64](https://downloads.bdrive.com/rclone_view/linux/arm-64/RcloneView-0.3.170-aarch64.AppImage) · [Deb x86_64](https://downloads.bdrive.com/rclone_view/linux/x86-64/rclone_view-0.3.170+1-linux.deb) · [Deb arm64](https://downloads.bdrive.com/rclone_view/linux/arm-64/rclone_view-0.3.170+1-linux-arm64.deb) · [RPM x86_64](https://downloads.bdrive.com/rclone_view/linux/x86-64/rclone_view-0.3.170+1-linux.rpm) · [RPM arm64](https://downloads.bdrive.com/rclone_view/linux/arm-64/rclone_view-0.3.170+1-linux-arm64.rpm)

## Fitur Baru & Peningkatan

- **Perbandingan Folder:** Kami dengan bangga memperkenalkan dukungan perbandingan folder! Kini Anda dapat membandingkan folder secara visual dan menganalisis perbedaan di dalam subfolder secara rekursif, membuat manajemen file lebih intuitif dan efisien.
- **Dukungan Linux yang Lebih Baik:** Memperbaiki masalah pada distribusi Linux terbaru.
- **Peningkatan UI & UX:** Memperbarui desain UX bilah utama, menyesuaikan warna ikon dan teks nama file agar lebih jelas terlihat, memperbaiki ikon daftar remote untuk pengguna lisensi gratis, menyempurnakan tata letak bilah menu dan tooltip, memperbaiki pintasan penutupan jendela, dan menyesuaikan visibilitas pengubahan ukuran jendela di macOS.
- **Peningkatan Sinkronisasi & Manajemen Remote:** Menambahkan indikator penyelesaian langkah dan navigasi cepat pada dialog Sinkronisasi, memperbaiki pembuatan otomatis passphrase file kunci pada pendaftaran remote SFTP, meningkatkan perbandingan sinkronisasi folder, dan memperbaiki ikon penyedia yang hilang pada pendaftaran remote.
- **Peningkatan Lain-lain:** Memperbaiki masalah tampilan Lisensi Seumur Hidup dan Tentang, meningkatkan fungsi pembaruan iklan RcloneView, menampilkan pengaturan terkait Rclone pada panel lokasi, memastikan perilaku ikon taskbar menampilkan jendela dengan benar, memperbaiki notifikasi eksekusi tray, memastikan jendela ditutup ke tray dan mendukung keluar melalui tray.

## Perbaikan Bug

- Memperbaiki masalah macet pada dialog job dan mount saat menambahkan job tanpa remote.
- Meningkatkan penanganan error di seluruh aplikasi.
- Memperbaiki masalah pemuatan iklan di Windows.
