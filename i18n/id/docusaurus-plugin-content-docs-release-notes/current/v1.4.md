# RcloneView 1.4 – Catatan Rilis

**Tanggal Rilis:** Maret 2026<br/>
**Platform:** Windows, macOS, Linux

RcloneView 1.4 menghadirkan integrasi iCloud Photos, frontend berbasis web untuk akses jarak jauh, peningkatan kemampuan Batch Job, serta berbagai perbaikan stabilitas dan kegunaan di semua platform.

## Sorotan

- **Dukungan iCloud Photos (Beta, Read-Only):**
  Jelajahi dan unduh foto dari iCloud Photo Library Anda langsung di dalam RcloneView.
  Lihat album, cari berdasarkan tanggal, dan unduh file asli — semuanya tanpa perlu keluar dari aplikasi.
  Fitur beta ini hanya mendukung akses baca (read-only) — unggah, hapus, dan ganti nama belum tersedia saat ini.

- **Frontend Web (Beta):**
  Akses RcloneView dari mana saja melalui browser Anda. Web server bawaan menyediakan
  penjelajah file dual-panel yang lengkap serta pemantauan transfer secara real-time.
  Termasuk akses yang dilindungi kata sandi dan opsi untuk mengizinkan koneksi jarak jauh.

- **Batch Job yang Disempurnakan:**
  Batch Job telah ditingkatkan secara signifikan dengan eksekusi langkah demi langkah, dry run per langkah,
  ketergantungan variabel antar langkah, indikator tingkat risiko, pencarian template, dan pelacakan riwayat eksekusi secara penuh.
  Bangun alur kerja multi-langkah yang kompleks dengan percaya diri menggunakan pratinjau dry run dan visibilitas risiko yang jelas.

## Fitur Baru

- **Simpan & Muat Preset Filter:**
  Simpan dan muat preset filter untuk digunakan kembali di berbagai job dan sesi.

- **Opsi Mount di Windows:**
  Pengaturan izin file dan direktori kini tersedia di Windows untuk operasi mount.

## Peningkatan

- **Dukungan macOS 12:**
  Kompatibilitas dengan macOS 12 (Monterey) dipulihkan kembali.

- **Pengecualian Dot-File Dropbox:**
  Opsi untuk secara otomatis mengecualikan dot-file (misalnya, .DS_Store) saat melakukan sinkronisasi ke Dropbox.

- **Kompatibilitas Ikon Tray Linux:**
  Penanganan system tray yang ditingkatkan untuk Linux Cinnamon dan desktop lain dengan dukungan tray native.

- **Warna Ikon Tray Windows:**
  Warna ikon tray kini dapat disesuaikan di Windows.

- **Penanganan Path Database:**
  Konfigurasi path database yang ditingkatkan dengan dukungan path lama (legacy) untuk proses upgrade yang lebih lancar.

- **Proton Drive Dihapus:**
  Proton Drive telah dihapus dari daftar penyedia remote karena adanya perubahan pada pihak upstream.

## Perbaikan Bug

- **Custom Flags Tidak Diterapkan:**
  Memperbaiki masalah di mana custom flags diabaikan selama eksekusi job.

- **Dry Run pada Job Move:**
  Memperbaiki tombol Dry Run yang tidak aktif untuk tipe job Move.

- **Status Terminal Maksimal:**
  Memperbaiki dua masalah saat panel terminal dalam keadaan maksimal — termasuk error status elemen UI.

- **Inisialisasi Database:**
  Menghilangkan pemanggilan sqfliteFfiInit() yang duplikat, yang dapat menyebabkan error "database closed" secara sesekali.

- **ID Perangkat Secure Storage:**
  Memperbaiki penanganan ID perangkat pada secure storage, dan mengatasi bug di mana status nonaktif dari deteksi otomatis Synology tidak tersimpan.

- **Lag Input Scheduler:**
  Memperbaiki lag UI saat memasukkan parameter penjadwalan.
