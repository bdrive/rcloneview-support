# Jenis Lisensi RcloneView

RcloneView menawarkan dua jenis lisensi untuk mendukung berbagai tingkat penggunaan: **Free** dan **Plus**.  
Tabel berikut membandingkan fitur dan batasan yang tersedia pada masing-masing paket:

## 🔍 Perbandingan Fitur

| Fitur                               | Lisensi Free | Lisensi Plus |
| ---------------------------------- | ------------ | ------------ |
| Mount/Unmount Remote               | ✅            | ✅            |
| File Explorer (Jelajahi/Salin/Hapus) | ✅            | ✅            |
| Perbandingan Folder & Manajemen File | ✅            | ✅            |
| Perbandingan Folder dengan Filter  | ❌            | ✅            |
| Download/Upload File               | ✅            | ✅            |
| Sinkronisasi & Manajemen Job       | ✅            | ✅            |
| Sinkronisasi 1:N                   | ✅            | ✅            |
| Job Sinkronisasi Berjadwal         | ❌            | ✅            |
| Auto Start Jadwal saat Startup     | ❌            | ✅            |
| Mount Penyimpanan Remote           | ✅            | ✅            |
| Auto Mount saat Startup            | ❌            | ✅            |
| Riwayat Job & Log                  | ✅            | ✅            |
| Integrasi System Tray              | ✅            | ✅            |
| Multiple Jendela RcloneView        | ❌            | ✅            |
| Membutuhkan License Key            | ❌            | ✅            |

## Cara Mengaktifkan Lisensi Plus

1. Kunjungi [Halaman Harga](https://rcloneview.com/src/pricing.html) dan beli lisensi.
2. Di RcloneView, buka **Help > Activate License**.
3. Masukkan **email** Anda dan **license key** yang diberikan.

Setelah diaktifkan, fitur premium akan terbuka secara otomatis.
