# AWS S3 dan yang Kompatibel dengan S3

RcloneView mendukung **Amazon S3** dan berbagai **penyedia object storage yang kompatibel dengan S3**, termasuk vendor-vendor besar seperti Wasabi, Backblaze, Cloudflare R2, dan lainnya.

Remote-remote ini menggunakan antarmuka konfigurasi yang sama berdasarkan protokol S3. Setelah dipilih, Anda perlu memasukkan kredensial akses Anda dan, jika diperlukan, URL endpoint khusus.


## Cara Menambahkan Remote yang Kompatibel dengan S3

### Langkah 1: Membuka Jendela Konfigurasi Remote Baru

- Klik **`+ New Remote`** dari menu atas di bawah **`Remote`**.
- Alternatifnya, klik tombol **`+`** pada panel Explorer lalu pilih **`New Remote`**.

### Langkah 2: Mengonfigurasi Penyedia S3

#### Pada tab **Provider**:
1. Cari `s3` atau nama penyedia Anda (misalnya, `aws`, `wasabi`, `minio`).
2. Pilih **`S3`** dari daftar hasil.

#### Pada tab **Options**:
3. Atur **provider** Anda (misalnya, `AWS`, `Wasabi`, `Minio`, dll.).
4. Masukkan [**Access Key ID**](/howto/cloud-storage-setting/aws-account-info#step-by-step-get-your-aws-access-key-id-and-secret-access-key) Anda.
5. Masukkan [**Secret Access Key**](/howto/cloud-storage-setting/aws-account-info#step-by-step-get-your-aws-access-key-id-and-secret-access-key) Anda.
6. Pilih **region** yang benar (misalnya, `ap-northeast-2`) untuk bucket penyimpanan Anda.
   Jika diperlukan, atur **endpoint** khusus (untuk penyedia yang tidak menggunakan format endpoint AWS).

### Langkah 3: Menamai dan Menyimpan Remote

#### Pada tab **Name**:
7. Berikan nama untuk remote (misalnya, `myWasabi`, `minio-prod`).

#### Pada tab **Save**:
8. Klik **Save** untuk menyelesaikan pengaturan.
