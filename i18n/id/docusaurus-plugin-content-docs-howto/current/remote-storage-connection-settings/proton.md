# Proton Drive

## Cara Menambahkan Proton Drive Menggunakan RcloneView (Windows)

### Langkah 1: Buka Remote Manager

- Klik **`+ New Remote`** di pojok kanan atas **Remote Manager**.
- Alternatifnya, klik tombol **`+`** di panel Explorer dan pilih **`New Remote`** untuk memulai konfigurasi remote.

### Langkah 2: Pilih Proton Drive sebagai Storage Provider

1. Di bilah **Search Provider**, ketik `proton`.
2. Pilih **Proton Drive** dari daftar.

Anda kemudian akan diarahkan ke layar konfigurasi untuk Proton Drive.

<img src="/support/images/en/howto/remote-storage-connection-settings/select-proton-drive-remote.png" alt="select proton drive remote" class="img-medium img-center" />

### Langkah 3: Konfigurasikan Remote Proton Drive Anda

Isi kolom yang diperlukan pada formulir konfigurasi:

- **Remote name**: Nama yang mudah dikenali untuk remote Anda (misalnya, `MyProtonDrive`)
- **username**: Alamat email Proton Anda
- **password**: Kata sandi akun Proton Anda
- **2fa** (opsional): Kode 2FA Anda saat ini (hanya jika 2FA diaktifkan)

Setelah memasukkan semua informasi yang diperlukan, klik **`Add Remote`** untuk menyelesaikan pengaturan.

<img src="/support/images/en/howto/remote-storage-connection-settings/remote-configure-proton.png" alt="remote configure proton" class="img-medium img-center" />

### Langkah 4: Konfirmasi Remote yang Ditambahkan

Setelah ditambahkan, remote Proton Drive Anda (misalnya, `MyProtonDrive`) akan muncul di daftar **Remote Manager**.

Sekarang Anda dapat:
- Klik **`Open`** untuk menjelajahi isi Proton Drive Anda.
- Menggunakannya dalam transfer, mount, atau tugas terjadwal.
- Mengedit atau menghapus konfigurasi remote kapan saja.

<img src="/support/images/en/howto/remote-storage-connection-settings/remote-manager-proton-view.png" alt="remote manager proton view" class="img-medium img-center" />


✅ **Selesai!** **Proton Drive** Anda sekarang berhasil terhubung ke **RcloneView**.
