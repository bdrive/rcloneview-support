# Penyimpanan Cloud Penuh? Bersihkan dan Bebaskan Ruang di Semua Akun Anda dengan RcloneView

> Kehabisan ruang penyimpanan cloud? Temukan cara mendapatkan kembali gigabyte ruang dengan mengidentifikasi folder yang membengkak, menghapus duplikat, dan mengoptimalkan akun cloud Anda.

Saat penyimpanan cloud Anda mencapai batasnya, ini bukan sekadar ketidaknyamanan—ini menghambat unggahan, memperlambat produktivitas, dan memaksa Anda melakukan upgrade yang mahal. RcloneView membantu Anda dengan cepat mengidentifikasi apa yang menghabiskan ruang dan membersihkannya di semua akun Anda dalam satu tempat.

## Temukan Pemakan Ruang Terbesar

Fitur file browser dan pengurutan RcloneView memungkinkan Anda langsung melihat file dan folder mana yang menghabiskan ruang paling banyak. Urutkan berdasarkan ukuran, tanggal modifikasi, atau jumlah file untuk menemukan folder yang membengkak yang mungkin sudah Anda lupakan.

![Cloud storage file listing](/images/en/blog/new-remote.png)

Penyebab umum meliputi pencadangan proyek lama, unduhan duplikat, dan lampiran email yang diarsipkan. Dengan tampilan yang jelas atas kondisi penyimpanan Anda, Anda dapat membuat keputusan yang tepat tentang apa yang harus dihapus.

## Identifikasi File Duplikat dan Hapus

Duplikat menumpuk dengan cepat—file yang tidak sengaja diunggah, salinan pencadangan, dan pembengkakan versi diam-diam menghabiskan ruang. Alat perbandingan RcloneView membantu Anda menemukan duplikat di seluruh akun cloud Anda.

![Comparing files and folders](/images/en/howto/rcloneview-basic/compare-display-select.png)

Setelah teridentifikasi, Anda dapat dengan aman menghapus duplikat dan mendapatkan kembali ruang penyimpanan yang signifikan tanpa kehilangan data penting.

## Hapus File Lama Secara Massal

Gunakan alat pemfilteran dan pemilihan RcloneView untuk menargetkan file berdasarkan usia, ekstensi, atau pola penamaan. Filter file yang lebih lama dari tanggal tertentu, lalu hapus secara massal dalam satu operasi.

![Job history and execution](/images/en/howto/rcloneview-basic/job-history.png)

Pendekatan ini sangat berguna untuk file sementara, unduhan, dan konten yang diarsipkan yang sudah tidak Anda butuhkan lagi.

## Memulai

1. **Unduh RcloneView** dari [rcloneview.com](https://rcloneview.com/src/download.html).
2. Hubungkan ke akun cloud Anda dan navigasikan ke folder yang menghabiskan ruang paling banyak.
3. Gunakan pengurutan dan pemfilteran untuk mengidentifikasi file yang akan dihapus, lalu hapus secara massal.
4. Pantau pemulihan ruang penyimpanan Anda secara real-time.

Dapatkan kembali penyimpanan cloud Anda hari ini dan hilangkan kebutuhan akan upgrade yang mahal.
